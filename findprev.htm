findprev

    

RTC Function

Finds the preceding record in a database table.

Declaration:

int findprev (dbase db)

Group:

Database functions

Parameters:

db

Database table

Return value:

Function result

 

The function findprev searches the database table specified by the parameter db for a record preceding the current one. As search criteria the index field is used, i.e. the field that has been declared first. The function result returns whether a precedent record was found.

0

Record found

1

Record not found

 

Example:

/* Print first weighings sorted by key number in descending order */

FirstWeight.No = IdentNo;

while (findprev (FirstWeight) == 0)

{

  /* Output record */

}

 

Note:

The function findprev is a method of the declaration word dbase.
