widget_timeout

    

RTC Function

Assigns a timeout for operator input to an object.

Declaration:

void widget_timeout (int id, float t)

Group:

Touch screen programming

Parameters:

id
t

Logical reference number of the object
Timeout value in seconds

Return value:

-

 

The instruction widget_timeout assigns a maximum time t for operator inputs to the object of type WIDGET_TYPE_TEXT_INPUT specified by the parameter id. If no operator input is recognized within the given time, the object automatically terminates the input mode and returns the event EVENT_TYPE_TEXT_ABORTED.

 

Example:

// Password entry wit timeout

void CreatePasswordEntry (int id, string label_prompt)

{

  createwidget (id, WIDGET_TYPE_TEXT_INPUT);

    widget_format  (id, INPUT_FORMAT_NUMERIC_PASSWORD, 8, 0);

    widget_label   (id, label_prompt);

    widget_timeout (id, 5.0);

}
