The Programming Language RTC     String Functions

Composing and Evaluating Strings

    

The function length specifies the actual length of a string in number of characters. Alternatively, the length can also be determined in pixels using the instruction textwidth, taking into account the chosen typeface and font.

The function sub returns a substring of a specified length. Alternatively, the function copy can be used that in one go also writes the substring to a certain position of another string. The position of a certain character within a string can be determined using the function pos.

For the evaluation of a string, the function cssub may be helpful which returns the nth part string. The individual substrings are separated by a given character.

Where a string is to be built up out of different data types the function concat is used. This function also provides the necessary conversion of the individual data types into the data type string. If a numeric value needs to be given a certain format, the instruction adj is used. The function itoh is used to convert an integer number into a string using hexadecimal writing.

In situations where a string should only consist of space characters, the function space is used.
