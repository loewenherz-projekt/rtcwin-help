The Programming Language RTC

Introduction

  

As the character C already suggests, the Programming Language RTC is based on the programming language C. However, only this part of C is implemented that is necessary for implementing applications for a weighing terminal.

The developers of RTC not only did not have the ambition to implement a full C, they willingly forewent all the tricky details that some C programmers might love, others, however, simply drive mad.

Amongst others, RTC differs from a standard C in the following aspects:

·No embedding of libraries. The full extent of the programming language is instantly and completely available.
·Parameter check on function calls in respect of completeness and data type.
·No pointer and address operations.

On the other hand, the amount of RTC instructions is completed with some powerful functions that in some ways remind of the modules of the RTG program generator and which they even get ahead of.
