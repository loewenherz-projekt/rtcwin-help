readparameter

    

RTC Function

Reads a Service Mode parameter.

Declaration:

string readparameter (string group, string par)

Group:

System

Parameters:

group
par

Parameter group
Parameter name

Return value:

Result string

 

The function readparameter reads the parameter par from the Service Mode group group and returns the current setting as result string.
