setdisplayoption

    

RTC Function

Sets specific properties of the display.

Declaration:

void setdisplayoption (int opt, any val)

Group:

Touch screen programming

Parameters:

opt
val

Display option to be set
Value the option is set to

Return value:

-

 

The instruction setdisplayoption sets specific properties of the standard display. The parameter opt determines the property to be modified with the value specified by the parameter val as follows:

Property

Alias

Description

0

DISPLAY_OPTION_NONE

No option

1

DISPLAY_OPTION_SHOW_WIDGET

Activates the weight display (val=0).

2

DISPLAY_OPTION_HIDE_WIDGET

Deactivates the weight display (val=0). In the upper right corner of the screen, a red rectangle containing an exclamation point and an out-crossed "kg" sign is shown.

3

DISPLAY_OPTION_STATUS_WIDGETS

Activates the following options for the header objects specified by val. Objects not contained in the parameter val are deactivated.

 

1

STATUS_WIDGET_USER_TEXT

Headline with application-specific text (see next option)

 

2

STATUS_WIDGET_WEIGHING_LABEL

On-screen electronic marking plate

 

4

STATUS_WIDGET_POWER

Battery status indication

 

8

STATUS_WIDGET_DATE_TIME

Current time

4

DISPLAY_OPTION_STATUS_USER_TEXT

The parameter val contains an application-specific text to be shown in the headline. The text is shown in double angle brackets, e.g. «TEXT».

5

DISPLAY_OPTION_KEYBOARD_MODE

The parameter val defines the layout of the on-screen keypad as follows:

 

0

KEYBOARD_MODE_ALPHANUMERIC

Alphanumeric keyboard

 

1

KEYBOARD_MODE_NUMERIC

Numeric keyboard

6

DISPLAY_OPTION_SCALE_BUTTONS

The parameter val enables the four scale function keys as follows. Inactive keys are invisible.

 

1

SCALE_BUTTON_ZERO

Zero key

 

2

SCALE_BUTTON_SELECT

Selection key for the secondary weight display

 

4

SCALE_BUTTON_PRESET_TARE

Preset tare key (PT)

 

8

SCALE_BUTTON_TARE

Tare key (T)

7

DISPLAY_OPTION_SCALE_LINE_FUNCTIONS

The parameter val enables the information to be shown in the secondary weight display as follows. Inactive functions cannot be chosen by the toggle key.

 

1

SCALE_LINE_TARE_WEIGHT

Tare weight

 

2

SCALE_LINE_GROSS_WEIGHT

Gross weight

 

4

SCALE_LINE_BARGRAPH

Bargraph

 

8

SCALE_LINE_APPROVALDATA

Approved weight storage (alibi memory)

8

DISPLAY_OPTION_KEYBOARD_BEEP_FREQUENCY

The parameter val defines the frequency of the beep as feedback of the keyboard. If val=0, the beeper is deactivated.

9

DISPLAY_OPTION_APPLICATION_BEEP_FREQUENCY

The parameter val defines the frequency of the beep when activated by the application. If val=0, the beeper is deactivated.

10

-

11

DISPLAY_OPTION_DISPLAY_MODE

The parameter val controls the weight display as follows:

 

0

DISPLAY_MODE_WEIGHT

The display contains the weight with current units.

 

1

DISPLAY_MODE_PIECES

The display shows the current number of pieces with unit "pieces."

 

Example:

// Activate electronic marking plate and time in head line

setdisplayoption (DISPLAY_OPTION_STATUS_WIDGET, STATUS_WIDGET_WEIGHING_LABEL | 

                                                STATUS_WIDGET_DATE_TIME);

 

 

Note:

Applicable preprocessor constants (e.g. DISPLAY_OPTION_SHOW_WIDGET) are available in the file "qtmonitor.h."
