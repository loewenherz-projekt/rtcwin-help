The Programming Language RTC     XML Interface

General

    

RTC provides the option to handle up to 8 XML files (instances) at the same time. The individual instances are accessed by their reference number 0 to 7.

The content of an XML instance can be loaded from a file or a string using the instructions xmlloadfile or xmlloadstring, respectively, or can be read from a serial channel using xmlpread. Conversely, the content of an XML instance can be written to a file or string using xmlsavefile or xmlsavestring or written to a serial channel using xmlpwrite.

If the built-in WEB interface is used, XML data is transferred via the message handler and can be transferred to or from the XML instance using the functions xmlfrommsg or xmltomsg, respectively.

The function xmlclear can be used to clear the content of an XML instance. The function xmlpclear can be used to clear both the XML instance and the receive buffer of the related serial interface at the same time.
