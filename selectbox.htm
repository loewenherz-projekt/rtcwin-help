selectbox

    

RTC Function

Menu for function selection.

Declaration:

void selectbox (string title)
{
 prompt   procname( )
 prompt   edit(db_name)
 ...
}

Group:

Display and keyboard

Parameters:

title
prompt
procname
db_name

Headline
Prompt
Executable function
Name of database table

Return value:

-

 

The instruction selectbox generates a complete dialog for a function selection according to the example below. As parameter the instruction is given a text for the title bar after which – enclosed by a pair of curved brackets – the options are listed. Each option again consists of a text to be displayed and either a function without parameters or the statement edit. The function selectbox displays text in black proportional font at a size of 13 pixels.

 

Example:

void SupervisorMode (void)

{

  if (codebox ("Please enter password", UserPassword) == ok)

    selectbox ("Settings")

    {

      "General"       GeneralSettings();

      "Customers"     edit (Client);

      "Products"      edit (Article);

      "Remarks"       EnterRemarks();

      "Configuration" Configuration();

    }

}

 

 

Notes:

The instruction selectbox provides multi-language operator prompting; the use of the function lsub is not necessary.

The instruction selectbox does not save the original content of the application screen or the previously selected font attributes. This has to be considered in the design of the program.

For a selection with more than five options, the instruction selectbox creates a two-column display. This should be avoided for better usability. Use nested calls of selectbox (submenus) instead.
