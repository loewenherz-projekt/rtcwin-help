inputselection

    

RTC Function

Entry of a variable via selection from a list of options.

Declaration:

char inputselection (int x, int y, string list, any v)

Group:

Display and keyboard

Parameters:

x
y
list
v

Horizontal coordinate x in pixels
Vertical coordinate y in pixels
String list with options
Variable of type integer to be entered

Return value:

Code of terminating function key

 

The instruction inputselection creates an input field for the variable specified by v at the position specified by the parameters x and y on the default display and enables data entry for the variable through the terminal keyboard or the connected PC keyboard. The function uses font 28 (RTC Mono) with the currently selected font size. The writing color is black; the background is changed to blue.

The parameter list passes a string containing a list of comma separated options, from which the operator can choose using the horizontal cursor keys. The chosen option from the list is shown in the input field of the variable.

The entry of the variable is terminated when the enter key, a function key or any other special key is pressed. The position number of the chosen option within the provided string is written into the target variable with 0 referring to the first element of the list. The key code of the key pressed for termination is returned as function result (see also inputbox).

If in advance the variable has been assigned an additional help text via the instruction setinfo, the F5 key can be pressed during data entry to call up that help text. Accordingly, the F5 key is labeled 'Info' and thus no more available for termination of the entry.

 

Example:

switch (i)

{

  case 0: showtext (10, 22, "Order No.");

          ch = input (100, 22, OrderNo);

          break;

  case 1: showtext (10, 40, "Dispatch");

          ch = inputselection (100, 40, "Ex work,Freight paid,Free carrier",

                 ShippingMode);

          break;

}  

 

 

Note:

The instruction inputselection provides multi-language operator prompting for the list of options; the use of the function lsub is not necessary.
