sub

    

RTC Function

Separates a substring from a string.

Declaration:

string sub (string s, int p, int n)

Group:

String functions

Parameters:

s
p
n

Source string
Start position of substring
Length of substring

Return value:

Result string

 

The function sub returns a portion of the source string s with the length n, beginning at the start position p.Counting of p starts from 0 (first character). If p is larger than the length of s, sub returns an empty string. If s is shorter than p + n, only the remaining characters contained in s right of the position p are returned.

 

Example:

idx = sub (ReceiveBuffer, 0, 2);
