The Programming Language RTC     The Scale Interface

Variables

    

The internal scale function interface provides the following variables:

Variable

Description

decimal gross%8.dp

Gross weight as number

decimal tare%8.dp

Tare weight as number

decimal net%8.dp

Net weight as number

decimal nethires%10.dp+2

Net weight in a 100-fold resolution

string fgross[x]

Gross weight as readily formatted string

string ftare[x]

Tare weight as readily formatted string

string fnet[x]

Net weight as readily formatted string

decimal capacity8.dp

Maximum capacity (weighing range) of the scale

decimal division%8.dp

Increment size of the scale (scale interval)

int dp

Number of decimals

string unit[2]

Weight unit

bool issettled

Status variable for "scale settled"

bool iszero

Status variable for "scale in zero range"

int range

Current weighing range of a multiple-range or multi-interval scale

int taremode

Tare status

decimal alibino%4.0 or %6.0

Number against that the weighing is stored in the approved weight storage

bool sealed

Status variable for "calibration data sealed"

bool ntep

Status variable for "calibration according NTEP regulations"

The variables are updated by the functions getweight and getapprovedweight (alibino only by getapprovedweight) and thus always refer to the most recently read scale.
