xmlsetnodevalue

    

RTC Function

Sets the value of a node in an XML instance.

Declaration:

bool xmlsetnodevalue (int id, string node, string value)

Group:

XML interface

Parameters:

id
node
value

Logical reference number of the XML instance
Path and name of the node
Value of the node

Return value:

Function result

 

The function xmlsetnodevalue sets the value of a node from an XML instance specified by the parameter id to the provided value. The node is specified by the parameter node which contains the full path and name of the node. If the node does not exist, it is created.

If the value can be set successfully, xmlsetnodevalue returns true as function result; otherwise the result is false.

 

Example:

void WriteErrorLogFile(string errmsg)

{

  string node[4];

 

  ErrorCnt++;                        // increase error counter...

  node = adj(ErrorCnt,4,0);          // node name = ErrorCounter

  xmlloadfile(XML_ERROR_LOG, "shared/errorlog.xml");

  xmlsetnodevalue (XML_ERROR_LOG, concat("error/",node), 

                  concat(node, ": ", date, " ", time, ": ", errmsg));

  xmlsavefile(XML_ERROR_LOG, "shared/errorlog.xml");

}
