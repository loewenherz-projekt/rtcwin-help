The Programming Language RTC     Output to the Display

General

    

The programming language RTC provides a wide range of functions for the output to the screen of the weighing terminal. They are very powerful and help the programmer to create dialogs most efficiently in a relatively short period of time.

On the other hand, also very basic functions are contained to allow the creation of very individual display functions, if the need arises.

As a basic principle, only the lower part of the screen is accessible for the application programmer; the upper part is reserved for the weight display. Accordingly, the x-y coordinate 0-0 addresses the left and topmost pixel just below the weight display. The x-coordinate specifies the horizontal position from left to right whereas the vertical position is described by the y-coordinate from top to down.

For applications that are not subject of Weights & Measures the weight display may be switched off using the instruction weightdisplay (false). In this case, the origin of the coordinates indeed is the topmost left pixel of the screen. In the upper right corner of the screen a red rectangle containing an exclamation point and an out-crossed "kg" sign is shown that cannot be overwritten. Thus, the application is inhibited from faking an approved weight display.
