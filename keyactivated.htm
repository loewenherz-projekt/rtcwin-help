keyactivated

    

RTC Function

Checks whether a key-press invoked by the function activatekey is already read.

Declaration:

bool keyactivated (void)

Group:

Display and keyboard

Parameters:

-

 

Return value:

Function result

 

The function keyactivated checks whether a key-press induced under program control by the instruction activatekey – e.g. out of a background task – has already been read and evaluated, e.g. via the function readkey. If this is the case, keyactivated returns true as function result; otherwise the result is false.

 

Example:

if (cmd = CM_CANCEL) // Terminate operator input...

{

  activatekey (fct6);

  while (!keyactivated()) delay(0.1);

}
