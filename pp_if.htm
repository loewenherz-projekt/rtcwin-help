#if

    

RTC Function

Checks a preprocessor condition.

Declaration:

#if  expression

Group:

Preprocessor

Parameters:

expression

Algebraic expression

 

With the #if statement an algebraic expression can be evaluated and checked. If the result of the expression is true, the subsequent program lines up to the #endif statement are included in the process of compilation. If the expression returns false, the subsequent program lines are discounted.

Between the #if and #endif statement an optional #else may be included. If the algebraic expression is true, the lines between #if and #else are considered for compilation; otherwise the part between #else and #endif is used instead. If the #else has to be followed another #if, the two statements may be combined to an #elif.

 

Example:

  StartFilling();

#if DEBUG_LEVEL == 2

  gettime();

  writestring (Port, concat (time,": Filling started.\r\n"));

#elif DEBUG_LEVEL > 0

  StartTime = systemtimer ();

#endif

 

The #if statement my also be used to check the existence of a preprocessor constant using the operator defined.

 

Example:

void FastFeed (decimal target)

{

  while (net < target)  

  {  

    bout (O_FAST, true);

    getweight (indicatedscale());

#if defined TESTMODE

    showtext (10, 40, adj (net - target, 8, dp));

#endif

  }

  bout (O_FAST, false);

}

 

Note:

#if statements may be nested. However, only one preprocessor statement is accepted per line.
