The Programming Language RTC     Variables and Constants

Data Types

    

In RTC, the following data types are implemented:

 

int

32-bit integer number

float

Floating-point number, 32 bit (6 digits precision)

double

Floating-point number with double precision, 64 bit (16 digits precision)

decimal

Formatted number with fixed decimal point position

bool

Boolean value

string

String of characters

char

Individual character

 

For assignments and mathematical operations, the following agreements have to be observed:

 int

1234      -1

0x1f5e   0x1F5E
 

Decimal notation with negative sign (if applicable).

Hexadecimal notation with preceding 0x

 float

1.234    -12.34    1.

Decimal notation with decimal point and negative sign (if applicable).

The types float and int may be assigned each other. In that case, a possible decimal part will be truncated.
 

 double

as float

 decimal

as float
 

 bool

true   false

Use of predefined constants true and false.

The types bool and int may be assigned each other. In that case, false is interpreted as 0 and true as 1. Contrariwise, every integer number unequal 0 complies to true.
 

 char

'A'

'\n'

ok  cup  fct1

Indication of the character in single quotes.

Indication of a special character (see below).

Use of a predefined key code (see below).

The types char and int may be assigned each other. In that case, the character is represented by its 8-bit ASCII code.
 

  string

"abc"

"\002ABCD\n"

Indication of a string of characters in double quotes.

Indication of a string including special characters (see below).

 

For the data type char, the following special keys are defined:

ok

The enter key of the weighing terminal or a connected PC keyboard

cup

Cursor-up key of the weighing terminal or the connected PC keyboard

cdown

Cursor-down key of the weighing terminal or the connected PC keyboard

cright

Cursor-right key of the weighing terminal or the connected PC keyboard

cleft

Cursor-left key of the weighing terminal or the connected PC keyboard

info

Info key of the weighing terminal or TAB key of the connected PC keyboard

scale

Scale select key

set0

Zero setting key

sel

Selection key for the secondary weight display

ptar

Preset tare key

tar

Tare key for automatic tare and clear-tare functions

fct1 ... fct6

The function keys F1 to F6 of the weighing terminal or the connected PC keyboard

 

For the data types char and string, the following special characters can be entered by a preceding backslash character '\':

\b  

Backspace character

\t  

Tab character

\n  

Line feed character (LF)

\f  

Form feed

\r  

Carriage return character (CR)

\"  

Double quote

\\  

The backslash character itself

\'  

Single quote

\ddd  

Any character with ASCII code ddd (in decimal notation, three digits)
