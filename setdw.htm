setdw

    

RTC Function

Writes a weight into a data word.

Declaration:

void setdw (int dw, float value, int format)

Group:

Fieldbus

Parameters:

dw
value
format

Number of data word
Value
Data word format

Return value:

-

 

The function setdw writes the content specified by value into the data word addressed by dw. The value is converted according to format as follows:

0

value is multiplied by 10 as many times as the number of trailing decimals of the most recently read scale. The result is written into the given data word as 16-bit integer value. Example: The weight value 12.50 is multiplied by 100 and written into the given data word as 1250.

1

value is written into the given and the next data word as 32-bit floating-point number in IEEE format.

2

value is multiplied by 10 as many times as the number of trailing decimals of the most recently read scale. The result is written into the given (lower-bit part) and the next (higher-bit part) data word as 32-bit integer value.

 

Example:

setdw (DW_NET, net, 0);    // Write net weight to data word DW_NET.
setdw (DW_TOTAL, net, 2);  // Write total weight to data words DW_TOTAL and DW_TOTAL + 1
