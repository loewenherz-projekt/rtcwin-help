msgdst

    

RTC Function

Returns the destination of a message.

Declaration:

int msgdst (void)

Group:

Message handler

Parameters:

-

 

Return value:

Message destination

 

The function msgdst returns the destination of the most recently read message. Beforehand the message has to be read from the message queue using the function msgread. msgdst returns the destination of the message as follows:

 

Destination

Description

0

Undefined

3

Application (interpreter)

8

Web interface

9

JSON server (Java script object notification)

Other destinations exist for internal communication.
