sftpclose

    

RTC Function

Closes the connection to an SFTP server.

Declaration:

bool sftpclose (int sno)

Group:

File handling

Parameters:

sno

Logical connection number

Return value:

Function result

 

The function sftpclose closes the connection to an SFTP server. The connection is specified by the parameter sno. The parameter sno is the logical connection number assigned with the execution of the function sftpopen.

If the connection can be closed successfully, sftpclose returns true as function result. If the connection was already closed, the result is false.
