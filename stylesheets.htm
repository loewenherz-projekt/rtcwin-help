Usage of Style Sheets

    

To specify the look of a page, a window or an object more in detail, the style sheet syntax of QT can be used. Applicable commands are pagestyle, windowstyle and widget_style.

A style sheet command consist of a property name followed by a colon and the associated value. If more than one properties are to be set, they have to be separated by semicolons.

Example:

windowstyle (WINDOW_ID, "background: #5a7cf9;"\

                          "border-style: solid;"\ 

                          "border-width: 2px;"\

                          "border-color: white gray gray white;");

 

The available properties are:

Property

Description

background

Sets the background color. The color is specified by a hex number representing the RGB values as follows: #rrggbb. Alternatively, the following color names may be used:

black (#000000), blue (#0000ff), gray (#808080), green (#008000), orange (#ffa500), red (#ff0000), silver (#c0c0c0), white (#ffffff), and yellow (#ffff00).

border-style

Sets the style of the border to none, solid, inset, outset, dotted, double.

border-color

Sets the color of the border as described above. If more than one color (up to four) is given, they specify the top, right, bottom and left edges of a box, respectively. If the left color is not specified, it is taken to be the same as the right color. If the bottom color is not specified, it is taken to be the same as the top color. If the right color is not specified, it is taken to be the same as the top color.

border-width

Width of the border in pixels. The syntax is "npx" with n number of pixels. Up to 4 values can be given specifying the top, right, bottom, and left edges of a box, respectively. If the left width is not specified, it is taken to be the same as the right width. If the bottom width is not specified, it is taken to be the same as the top width. If the right width is not specified, it is also taken to be the same as the top width.

margin

Specifies the margin of the object in pixels. Up to 4 values can be specified that are handled as described above.

Further attributes can be found in the "Qt Style Sheets Reference" in the internet.
