The Programming Language RTC     Functions and Procedures

Function Parameters

    

Variables can be passed to a function as value (call by value) or as reference (call by reference).

Parameter passed as value:

The content (value) of a variable is handed over to the function as parameter. Within the function, the parameter serves as local variable and may even be modified. The original variable, however, remains unaffected.

Example:

int GetSum (int value1, int value2)

{

 int result =  value1 + value2;

 return (result);

}

 

void main (void)

{

 int variable1 = 1;

 int variable1 = 2;

 int result = GetSum (variable1, variable2);

}

Parameter passed as reference:

A pointer to the variable (memory address) is handed over to the function as parameter. Thus, the function directly accesses the original variable and may even modify its content.

To pass a variable as reference, simply add a preceding asterisk '*' to the name of the parameter in the declaration of the function. Inside the function, the parameter again is accessed by its name, but all changes directly affect the original variable. When calling a function, the name is used without a preceding asterisk '*'.

Example:

void GetSum (int value1 , int value2, int *result)

{

 result =  value1 + value2;

}

 

void main (void)

{

 int variable1 = 1;

 int variable2 = 2;

 int result = 0;

 GetSum (variable1, variable2, result);

}
