fontsize

    

RTC Function

Selection of font size.

Declaration:

void fontsize (int pix)

Group

Display and keyboard

Parameters:

pix

Font size in pixels

Return value:

-

 

The instruction fontsize specifies the character size for subsequent write operations. The parameter pix specifies the text size in pixels. The font itself as well as possibly selected attributes remain unchanged.

The bitmap fonts "RTC" and "RTC Mono" are only available in certain sizes. If the size specified by the parameter pix is not available, the next smaller font is selected instead.

For "RTC" (24) and "RTC Mono" (28) available font sizes are:

   11, 12, 13, 15, 17, 19, 21, 24, 27, 30, 34 and 38

 

Example:

int RTC_Font = 24;

int Bold     = 1;

 

/* Write headline */

clearscreen();

font (RTC_Font + Bold);

fontsize (17);

showtext (20, 10, "Article file");
