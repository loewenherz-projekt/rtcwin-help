widget_position

    

RTC Function

Positions an object on a drawing area.

Declaration:

void widget_position (int id, int xpos, int ypos)

Group:

Touch screen programming

Parameters:

id
xpos
ypos

Logical reference number of the object
Horizontal coordinate in pixels
Vertical coordinate in pixels

Return value:

-

 

The instruction widget_position positions the object specified by the parameter id on the selected drawing area. Beforehand, the object has to be placed on the drawing area using the instruction widget_move_to_sketch.

 

Example:

/* Sketch object is a drawing area for the standard

RTC write and drawing instructions with the defined

minimum size and background color. 

*/

createwidget (ID_SKETCH, WIDGET_TYPE_SKETCH); 

  widget_dimensions (ID_SKETCH, SKETCH_WIDTH, SKETCH_HEIGHT);

  background (CL_WHITE);

 

createwidget (ID_START_BUTTON, WIDGET_TYPE_BUTTON);

  widget_dimensions     (ID_START_BUTTON, START_BUTTON_WIDTH, START_BUTTON_HEIGHT);

  widget_move_to_sketch (ID_START_BUTTON, ID_SKETCH);  // Move start button to sketch object 

  widget_position       (ID_START_BUTTON, 20, 160);    // at position x=20 and y=160

  widget_fontsize       (ID_START_BUTTON, 24);

  widget_font           (ID_START_BUTTON, FT_RTC_BOLD);

  widget_text           (ID_START_BUTTON, lsub (TX_START
