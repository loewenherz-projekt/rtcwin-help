gross, net, tare, nethires

    

RTC Variable

Weight variables.

Declaration:

loc decimal   gross %8.dp
loc decimal   net %8.dp
loc decimal   tare %8.dp
loc decimal   nethires %10.dp+2

Group:

Scale functions

 

The variables gross, net and tare are predefined variables of type decimal. They contain the corresponding weights as numeric values and are updated whenever the functions getweight or getapprovedweight are executed. The variables have a length of 8 digits. The number of trailing decimals corresponds with the most recently accessed scale. The variable nethires contains the net weight in a 100-times-higher resolution. It has a length of 10 digits and 2 decimals more than the read scale.

The variables gross, net and tare are task local variables, i.e. they may be used in multiple tasks, however assign different physical memory areas for each task.

For formatting, the variables dp, division, unit and taremode are available. For approved printout the readily formatted string variables fgross, fnet and ftare have to be used.

 

Example:

infobox ("Load scale");

for (;;)

  if ((getweight (indicatedscale ()) == 0) && (gross > MinLoad))

    break;
