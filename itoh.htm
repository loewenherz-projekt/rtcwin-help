itoh

    

RTC Function

Generates a string with the hexadecimal representation of an integer value.

Declaration:

string itoh (int v)

Group:

String functions

Parameters:

v

Integer value

Return value:

Result string

 

The function itoh generates a string with the hexadecimal representation of the provided integer value v.The result string has a length of 8 characters and is – if applicable – filled up with leading zeroes.

 

Example:

SendBuffer = concat (Data, itoh (crc16 (Data, length (Data))));

writestring (Host, SendBuffer);
