preadstring

    

RTC Function

Reads a string from a communication channel.

Declaration:

int preadstring (int port, string *s, int len, char ec, float t)

Group:

Communication

Parameters:

port
s
len
ec
t

Logical port number
String variable for characters received
Number of characters to receive
End character
Timeout value

Return value:

Status

 

The function preadstring reads a string from the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen). The function is terminated if either the number of characters specified by the parameter len is read or the end character specified by the parameter ec is received or if the maximum receive time t (in seconds) has elapsed (timeout). If a string can be received successfully, it is written into the provided variable s. In that case, preadstring returns 0 as function result; otherwise the corresponding error code is returned instead.

 

Example:

/* Read data, end character = ETX, timeout = 5 sec. */

if (preadstring (port, ReceiveBuffer, 255, '\003', 5) != 0)

  ShowErrorMsg ("Error data transmission");

 

Note:

The function preadstring only reads up to the specified end character, i.e. it is not convenient for receiving binary strings. For this purpose, the function pread should be used.
