getdw

    

RTC Function

Reads a weight from a data word.

Declaration:

void getdw (int dw, float *var, int format)

Group:

Fieldbus

Parameters:

dw
var
format

Number of data word
Target variable
Data word format

Return value:

-

 

The function getdw reads the data word addressed by dw and writes the content in the target variable provided as parameter var. The value read from the data word is converted according to format as follows:

0

The data word content is divided by 10 as many times as the number of trailing decimals of the most recently read scale. The result is written into the provided variable. For example: the data word content 1250 is converted into 12.50 for a scale display with 2 trailing decimals.

1

The data word addressed by dw is interpreted in combination with the next data word as a 32-bit IEEE floating-point number. The result is written into the provided target variable.

2

The data word addressed by dw is interpreted in combination with the next data word as a 32-bit integer number. dw contains the lower-bit and the next data word the higher-bit part. The content of this double word is divided by 10 as many times as the number of trailing decimals of the most recently read scale. The result is written into the provided variable.

 

Example:

getdw (DW_TARGET, Target, 2);  // Read target quantity from data words DW_TARGET and DW_TARGET + 1
getdw (DW_PREACT, Preact, 0);  // Read preact from data word DW_PREACT
