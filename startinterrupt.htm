startinterrupt

    

RTC Function

Installs a function to be called cyclically by the operating system.

Declaration:

int startinterrupt (procname p, int ct, int prio)

Group:

Multitasking

Parameters:

p
ct
prio

Name of function without parameters
Interval in milliseconds
Priority

Return value:

-

 

The instruction startinterrupt installs the given parameter-free function specified by the parameter p to be called cyclically by the operating system. Thereafter, the function p is executed in the background simultaneously with the main task, but in contrast to a start using starttask, it is called and processed periodically in fix intervals. The parameter ct specifies the interval in milliseconds between two calls of the function p. The parameter prio assigns the function a priority over all other processes; this value should not exceed 1.

 

Example:

void Control_Task (void)

{

int result = getweight (indicatedscale ());

 

  if (result > 0)

    InterruptFilling ();

}

 

 

void main (void)

{

int result;

  GeneralInit ();

  result = startinterrupt (Control_Task, 50, 0);

  /* ... */

}

 

Note:

The execution of the function run as an independent task by startinterrupt may under no circumstances exceed the specified interval; otherwise this function will be executed exclusively and other functions can no longer be accessed. The execution should not take more than ¼ of the interval.
