waitkey

    

RTC Function

Reads a character from the keyboard with timeout.

Declaration:

char waitkey (float t)

Group:

Display and keyboard

Parameters:

t

Timeout value

Return value:

Key code

 

The function waitkey reads a character from the terminal keyboard or the connected PC keyboard. If the keyboard buffer contains any characters not yet read, readkey returns the corresponding key code without delay. Otherwise program execution waits until the operator presses a key. If no key is pressed within the time specified by the parameter t (in seconds), the function waitkey is terminated and returns 0 as function result.

 

Example:

/* Show message for 3 seconds */

infobox ("Weight registered.");

ch = waitkey (3);

/* ... */
