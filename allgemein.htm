The Programming Language RTC     Preprocessor

Defining Constants

    

Via the preprocessor statement #define constants may be defined. Thus – for example – digital inputs and outputs can be assigned meaningful names or language dependent texts can be grouped together and kept in the header of the program for easy maintenance. Syntax is as follows:

#define  Name  Value

Before program compilation is carried out, the preprocessor replaces Name with the assigned value, wherever it may appear.

 

Example:

/* External inputs/outputs */

#define  O_FAST    0

#define  O_SLOW    1

#define  O_READY   2

#define  O_ERROR   3

#define  I_START   0

#define  I_ABORT   1

 

/* Text constants */

#define  TX_READY    "Startbereit|Ready to start"

#define  TX_FINISH   "Fertig!|Finished"

#define  TX_FAST     "Grobstrom...|Filling fast..."

#define  TX_SLOW     "Feinstrom...|Filling slow..."
