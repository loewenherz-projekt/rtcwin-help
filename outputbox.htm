outputbox

    

RTC Function

Opens a screen for classic display commands.

Declaration:

void outputbox (string title)

Group:

Touch screen programming

Parameters:

title

Window title

Return value:

-

 

The instruction outputbox opens a page with the headline specified by the parameter title. The page contains a drawing object and six buttons on the bottom of the page and thus emulates the standard RTC display with the six function keys F1 to F6. The function buttonpressed can be used to check if one of the six buttons was pressed. If this is the case, the button can be read using readbutton.

 

Example:

void FillingMode (void)

{

char ch = 0;

int result;

 

  outputbox ("Filling Mode;Menu;;;;;Ok"); // "Menu" =F1, "Ok" =F6

  // From here standard RTC screen commands

  font (FT_RTC_BOLD); fontsize (21);

  showtext (10, 10, "Order:");

 

  // If function key pressed: read key and process...

  for (;;)

  {

    result = getweight (indicatedscale());  // ... read scale

    if (buttonpressed ())

    {

      ch = readbutton ();

      switch (ch)

      {

        case fct1: // Return to menu

                   return;

        case fct6: // ...

                   break;

      }

    }

    /* ... */

    delay (0.1);

  }

}
