continuetimer

    

RTC Function

Resumes a timer function.

Declaration:

void continuetimer (timer t)

Group:

Timer

Parameters:

t  

Variable of type timer

Return value:

-

 

The function continuetimer resumes execution of the timer function for the provided timer variable t. The timer must be stopped via the function stoptimer. continuetimer restarts the timer with the remaining time. The function timeout may later be used to check whether the time has elapsed.

 

Example:

timer CheckTimer;

 

void Timer (void)

{

char ch = 0;

decimal time_value%5.2 = 0;

  

  clearscreen ();

  showtext (10, 10, "Delay time (s):");

  ch = input (160, 10, time_value) ;

  starttimer (CheckTimer, time_value);

  while (!timeout (CheckTimer))

  {

    showtext (10, 25, concat ("Remain time: ", adj (remain (CheckTimer), 3, 2), " s", space (10)));

    if (keypressed())

    {

      ch = readkey();

      stoptimer (CheckTimer);

      showtext (10, 25, concat ("Stop!", space (20)));

      ch = readkey();

      continuetimer (CheckTimer);

    }

    delay (0.01);

  }

  showtext (10, 25, concat ("Ready!", space (20)));

  ch = readkey ();

}
