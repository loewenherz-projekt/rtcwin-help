tinput

    

RTC Function

Entry of a variable with timeout control on the standard display.

Declaration:

char input (int x, int y, any v, float t)

Group:

Display and keyboard

Parameters:

x
y
v
t

Horizontal coordinate x in pixels
Vertical coordinate y in pixels
Variable of any type to be entered
Timeout value in seconds

Return value:

Code of terminating function key

 

The instruction tinput creates an input field for the variable specified by v at the position specified by the parameters x and y and enables data entry for the variable through the terminal keyboard or the connected PC keyboard on the default display. The function uses font 28 (RTC Mono) with the currently selected font size. The writing color is black; the background is changed to blue.

The entry of the variable fully corresponds to the function input and is terminated when the enter key, a function key or any other special key is pressed. The key code of the key pressed for termination is returned as function result.

The parameter t determines a time in seconds. If this time elapses without a key being pressed, the entry is terminated automatically. As function result, in this case 0 is returned.

If in advance the variable has been assigned an additional help text via the instruction setinfo, the F5 key can be pressed during data entry to call up that help text. Accordingly, the F5 key is labeled 'Info' and thus no more available for termination of the entry.

 

Example:

showtext (10, 10, "Order");

ch = tinput (80, 10, OrderNo, 3);

Notes:

When the entry is terminated by the timeout, eventually entered characters remain in the input field and have to be overwritten with the original value.
