bit

    

RTC Function

Checks the status of a bit in an integer variable.

Declaration:

bool bit (int bitno, int v)

Group:

Fieldbus

Parameters:

bitno
v

Bit to be checked
Variable to be checked

Return value:

Bit status

 

The function bit checks the status of a bit in the provided variable v. The bit to be checked is specified by the parameter bitno. Counting of bitno starts from 0 for the least significant bit. If the bit is set, bit returns true as function result; otherwise the result is false.

 

Example:

if (bit (I_ABORT, Inputs))
  break; /* "Abort" input active */
