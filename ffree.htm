ffree

    

RTC Function

Returns the size of the available memory of the weighing terminal.

Declaration:

int ffree (void)

Group:

File handling

Parameters:

-

 

Return value:

Size of available data memory in byte

 

The function ffree returns the number of free bytes of the battery-backed data memory.

 

Example:

#define MIN_FREE_MEMORY   20000

bool CheckInternalAvailableMemory (free)

{

  if (ffree() < MIN_FREE_MEMORY)

  {

    ShowErrorMsg ("Not sufficient memory!");

    return (false);

  }

  else

    return (true);

}

 

Note:

Assignment of internal memory is in blocks of 1024 byte. Thus, the function result of ffree is always a multiple of 1024.

The Linux based operating system compresses the battery-backed data memory for system optimization. Thus, the available memory may change if contents are modified, in particular string variables. This is why it is highly recommended to take measures even before ffree returns 0 as result, e.g. delete the oldest log files.
