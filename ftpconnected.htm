ftpconnected

    

RTC Function

Check connection status to an FTP server.

Declaration:

bool ftpconnected (int fno)

Group:

File handling

Parameters:

fno

Logical connection number

Return value:

Connection status

 

The function ftpconnected checks the connection status to an FTP server. The connection is specified by the logical connection number fno, assigned with the execution of ftpopen. If the connection is available, ftpconnected returns true as function result; otherwise the result is false.

 

Example:

/* ... */

if (ftpconnected (0))

{

  if (ftplist (0, "/"))  // open directory ...

  {

    while (ftplistread (0, filename)) // read files...

    {

    // process files contained in filename......

          

    // Rename processed file

      if (!ftprename (0, "/", filename, concat (getstddate(), "_", filename)))

        ShowErrorMsg ("FTP access denied!");

    }

  }

}

else

  ShowErrorMsg ("Connection to FTP server lost!");
