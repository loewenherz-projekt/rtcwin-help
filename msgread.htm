msgread

    

RTC Function

Reads a message from the message queue.

Declaration:

int msgread (float t)

Group:

Message handler

Parameters:

timeout

Timeout value in seconds

Return value:

Message type

 

The function msgread reads a message from the message queue. The availability of a not yet handled message matching the filter settings (set-up by the instruction msgsetfilter) can be checked beforehand using the function msgreceived. The function msgread is terminated if either a message can be read or the maximum read time specified by the parameter t has elapsed (timeout). As function result, the message is returned as follows:

 

Message

Description

0

No message

1

Service Mode has been entered

2

Service Mode has been terminated

9

Application configuration parameters requested by the WEB interface

10

Application settings parameters requested by the WEB interface

11

Java script object notification (JSON) send by process visualization

Other message numbers are reserved for internal purpose.

 

Example:

  switch (msgread (0.2))

  {

    case MSG_NONE:            break;

    case MSG_SERVICE_MODE_LEFT:

                              RestartFlag = true;

                              break;

    /* ... */

  }
