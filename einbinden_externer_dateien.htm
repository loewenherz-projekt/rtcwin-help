The Programming Language RTC     Preprocessor

Embedding of External Files

    

Using the preprocessor statement

#include  "file_name"

extended programs may be split up in modules. Thus, for example, certain drivers or function packages can be developed and maintained from a central place and made available as include file for various applications. Also, text constants for language dependent operator prompts may be excluded into a separate file for easy-to-handle translation.

 

Example:

/* Embed operator prompts from external file */

#include  "ITSQC_Prompts.h"

#include  "RTC_Constants.h"

 

 

Note:

When an external file is included in a program, all functions and definitions of that file are embedded in the program code regardless if they are needed or not. There is no dynamic adaptation or optimization.
