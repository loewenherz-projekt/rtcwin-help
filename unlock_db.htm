unlock

    

RTC Function

Releases a locked record in a database table.

Declaration:

int unlock (dbase db)

Group:

Database functions

Parameters:

db     Database table

Return value:

Function result

 

The instruction unlock searches the database table specified by the parameter db for a record matching the current content of the index field and releases it. The record is stored against the index field, i.e. the field that has been declared first. If the record is no longer locked, it can be modified or deleted. The function result returns whether the record could be found and released.

0

Record unlocked

1

Record not found

A record can be locked against modification/deletion using the instruction lock. Whether a record is locked can be checked using the function islocked.

 

Example:

/* ... */

ch = select (Product, 100, 80, 130, 80);

if (ch == ok)

{

  lock (Product);

 

  /* Use product... */

  unlock (Product); // release product

}

 

Note:

The function unlock is a method of the declaration word dbase.
