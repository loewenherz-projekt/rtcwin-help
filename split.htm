split

    

RTC Function

Separates a string into substrings and assigns them to the elements of a string array.

Declaration:

int split (string darray, string source, char sc)

Group:

String functions

Parameters:

darray
source
sc

Destination array of type string
Source string
Separator

Return value:

Returns the number of substrings

 

The function split separates the source string into substrings and then writes the individual substrings into the elements of the destination array darray. The parameter sc is used as separator character for the determination of the substrings contained in source. The first substring is written into element 0 of the destination array, the second into element 1, and so on. If the source string contains more substrings than array elements are available, the operation is aborted when the last array element is reached; if source contains less substrings, non-used array elements are filled with empty strings.

 

Example:

string Colors[10][20];

 

split (Colors[0], "red;yellow;green;blue;white", ';');
