indicatedscale

    

RTC Function

Returns the number of the currently displayed scale.

Declaration:

int indicatedscale (void)

Group:

Scale functions

Parameters:

-

Return value:

Scale number

 

The function indicatedscale returns the number of the scale that is currently shown in the scale display. If no scale is installed, the return value is 0.

 

Example:

bool SelectScale (int scale_no)

{

timer t;

  if (indicatescale (scale_no) == 0)      // Schaltet die Gewichtsanzeige auf die angegebene Waagen um

  {

    starttimer (t, 6.0);

    do

    {

      if (timeout (t))

        return (false);                   // Timeout: Waagenumschaltung fehlgeschlagen

      delay (0.1);

    }

    while (indicatedscale () != scale_no);

    return (true);                        // Waagenumschaltung erfolgreich

  }

  else

    return (false);

}
