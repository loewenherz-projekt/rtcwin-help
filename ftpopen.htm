ftpopen

    

RTC Function

Opens a connection to an FTP server.

Declaration:

bool ftpopen (int fno, string ipaddr, int port)

Group:

File handling

Parameters:

fno
ipaddr
port

Logical connection number
IP address of FTP server
FTP port

Return value:

Function result

 

The function ftpopen opens a connection to an FTP server and makes it available for subsequent file access (e.g. via ftpget) under the logical connection number specified by the parameter fno. The logical connection number fno can freely be chosen from the range 0 to 7.

The parameter ipaddr contains the IP address of the FTP server as string, whereas the integer parameter port defines the port of the FTP server. If the connection can be opened successfully, ftpopen returns true as function result; otherwise the result is false. The connection status can be checked using the instruction ftpconnected.

After file access is completed, the connection to the FTP server should be closed using ftpclose.

 

Example:

/* Get order file from FTP server */

 

#define FTP_PORT 21

 

bool GetOrders (void)

{

bool result = false;

 

  if (ftpopen (0, ServerIP, FTP_PORT))  // Open FTP connection...

  {

    if (ftpauthenticate (0, LoginName, LoginPassword))  // Log-in ...

    {

      // get new orders and copy to 'shared'...

      if (ftpget (0, "Production/Scale_1/Orders.csv", "shared/Orders.csv"))  

      {

        result = true;

        // delete order file from server

        if (! ftpremove (0, "Production/Scale_1/Orders.csv"))

          ShowErrorMsg ("FTP access denied!");

      }

      else

        ShowErrorMsg ("No new orders available!");

    }

    else

      ShowErrorMsg ("FTP access denied!");

 

    if (!ftpclose (0))

      ShowErrorMsg ("FTP access denied!");      

  }

  else

    ShowErrorMsg ("FTP server not available!");

  return (result);

}

 

Notes:

Depending on type and load of the network, opening an SFTP connection may take several seconds. The programmer should respect this in his program design.

If the FTP connection is already open when ftpopen is executed, it is automatically closed and reopened anew.
