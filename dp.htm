dp

    

RTC Variable

Number of decimals of the most recently accessed scale.

Declaration:

loc int   dp

Group:

Scale functions

 

The variable dp is a predefined variable of type int. It contains the number of decimals of the scale most recently accessed by one of the functions getweight or getapprovedweight.

The variable dp is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.

 

Example:

int Factor (void)

{

int fac = 1;

int i;

  for (i=0; i<dp; i++) fac *= 10;

  return (fac);

}

 

The variable dp may also be used for the declaration of decimal variables. During run time, the number of decimals of a variable declared in such a way corresponds to the most recently accessed scale.

 

Example:

decimal SetPoint%8.dp;
