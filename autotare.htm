autotare

    

RTC Function

Performs an automatic taring with optional timeout. Default timeout is 6 seconds

Declaration:

int autotare ([int sno], [float timeout])

Group:

Scale functions

Parameters:

sno

Optional scale number

Return value:

Error code

 

The function autotare performs an automatic taring for the scale specified by the optional parameter sno, i.e. sets the scale to net = 0. If the parameter sno is missing, always the first scale is tared instead. If the scale can be tared successfully, autotare sets the variable taremode to 1 and returns 0 as function result; otherwise the corresponding error code is returned instead.

 

Example:

if (Product.TareValue > 0)
  err = mantare (indicatedscale(), Product.TareValue) ; // Tare scale with container tare value

else

  err = autotare (indicatedscale());

 

Notes:

For automatic taring, the scale has to be settled. If the scale does not provide a stable weight within 6 seconds, a timeout error is returned.

The tare function may be locked in the calibration of the scale.
