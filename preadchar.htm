preadchar

    

RTC Function

Reads a character from a communication channel.

Declaration:

char preadchar (int port)

Group:

Communication

Parameters:

port

Logical port number

Return value:

Character code

 

The function preadchar reads a character from the communication channel specified by the logical port number port.  The port has to be opened first (e.g. using popen). If the receive buffer of the channel contains any characters not yet read, preadchar returns the corresponding character code without delay. Otherwise program execution waits until a character is received. The function preceived can be used to check the availability of a character prior to the execution of preadchar.

 

Example:

#define STX 2

 

bool WaitForStartChar (int port)

{

char ch;

  for (;;) /* Wait for start char... */

  {

    if (preceived (port))

    {

      ch = preadchar (port);    /* ... read it... */

      if (ch == STX)

        return (true);         /* Start char reiceived... */

      else

        return (false);

    }

    delay (0.05);

  }

}
