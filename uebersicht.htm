The Programming Language RTC     Preprocessor

Overview

    

As of version 2, a preprocessor is included in RTC WIN. Preprocessor statements start with the character # and are executed before the program code is compiled.

The following preprocessor statements are available:

#define

Definition of a constant

#include

Embedding of an external file

#ifdef ... #else ... #endif
#ifndef ... #else ... #endif

Conditional compiling

#pragma

Preprocessor control statement
