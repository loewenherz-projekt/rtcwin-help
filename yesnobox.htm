yesnobox

    

RTC Function

Instruction to show a pop-up message with Yes/No entry.

Declaration:

char yesnobox (string text, bool f)

Group:

Display and keyboard

Parameters:

text
f

Prompt
Flag

Return value:

Code of terminating function key

 

The instruction yesnobox produces an embossed box in the center of the application screen that contains the text string specified by the parameter text. The text is shown in black proportional characters on white background and may extend over up to three lines. Line wraps can be specified by the sequence '\n'.

Below the provided text the two options 'No' and 'Yes' are presented for selection via keys 0 'No' and 1 'Yes' or via the horizontal cursor keys. The currently valid selection is highlighted with a blue background and may be confirmed via Enter key.

The instruction yesnobox is terminated by either pressing the Enter key or the F6 key (Cancel). The key pressed for termination is returned as function result. The status yes or no can be preset and checked afterward using the Boolean variable f.

Example:

if (yesnobox ("Abort order, are you sure?", f) == ok);

  if (f) return;

 

 

Notes:

The instruction yesnobox provides multi-language operator prompting; the use of the function lsub is not necessary.

The instruction yesnobox saves the original content of the application screen and restores it after execution.
