The Programming Language RTC     The Scale Interface

Scale Functions

    

The scale interface provides the following scale functions:

Function

Description

int getweight (int sno)

Read scale and update weight and status variables.

int getapprovedweight (int sno, string custom_data, float timeout)

Request an approved weight and update weight and status variables. The weight is logged in the approved weight storage.

int autotare (int sno, float timeout)

Perform automatic taring.

int mantare (int sno, float tv)

Tare scale with the provided value tv

int clrtare (int sno)

Clear tare and set scale to gross mode.

int setzero (int sno, float timeout)

Set scale to zero.

The listed functions all refer to the scale specified by the parameter sno. If this optional parameter is not provided, the function is executed upon the first scale.

Furthermore, the following functions are available to handle the weight display.

Function

Description

int indicatescale (int sno)

Switch weight display to the specified scale.

int indicatedscale (void)

Determine number of the scale currently shown in the weight display.

void weightdisplay (int mode)

Switch weight display on (mode = 1) or off (mode = 0) (for non-approved applications only).
