outputs

    

RTC Function

Reads data from a group of digital outputs.

Declaration:

int outputs (int msb, int lsb)

Group:

External inputs/outputs

Parameters:

msb
lsb

Number of the most significant output of the group
Number of the least significant output of the group

Return value:

Status of digital outputs as binary number

 

The function outputs reads a group of digital outputs specified by the two parameters msb and lsb. msb defines the output with the highest number and lsb defines the lowest output in the group. Counting starts from 0. The status of the outputs is represented by the bits of an integer number and returned as function result. Bit #0 represents the lowest bit in the group (digital output lsb) bit #1 the next bit (output lsb+1) and so on.The value 1 reflects the status "set" and 0 the status "not set" of the corresponding output.

The case msb = lsb reads one discrete output only and thus complies with the function out.

 

Example:

SaveOuts = outputs (7, 0); // Save and clear digital outputs 0–7
pout (0, 7, 0);

// Error handling...

 

pout (SaveOuts, 7, 0); // Restore outputs

 

Note:

If outputs is used to read more outputs than available, the obsolete bits of the function result are set to 0.
