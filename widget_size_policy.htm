widget_size_policy

    

RTC Function

Sets the size policy for an object within a layout.

Declaration:

void widget_size_policy (int id, int hp, int vp)

Group:

Touch screen programming

Parameters:

id
hp
vp

Logical reference number of the object
Policy for horizontal expansion of the object
Policy for vertical expansion of the object

Return value:

-

 

The instruction widget_size_policy defines the size policy for the object specified by the parameter id. Horizontal and vertical expansion of the object within the parent layout is specified by the two parameters hp and vp as follows:

Type

Alias

Description

0

SIZE_POLICY_FIXED

The size of the object is fix and cannot be changed.

1

SIZE_POLICY_MINIMUM

The object has a minimum size, but can be enlarged if needed.

2

SIZE_POLICY_MAXIMUM

The object has a maximum size, but can be decreased if needed.

3

SIZE_POLICY_PREFERRED

The object has a preferred size, but size can be changed if needed.

4

SIZE_POLICY_EXPANDING

The object has a reasonable size but can be expanded or shrunken and then adjusts its contents accordingly.

5

SIZE_POLICY_MINIMUMEXPANDING

The object has a sufficient size, but can be expanded and then adjusts its contents accordingly.

6

SIZE_POLICY_IGNORED

The size is ignored and the object gets all available space.

 

Example:

void CreateInfoBox (void)         

{

  begindraw ();

  createwindow (SOFTWARE_VERSION_WINDOW, WINDOW_TYPE_POPUP, "");

  windowstyle  (SOFTWARE_VERSION_WINDOW, ST_BOX_STYLE);

    beginlayout (LAYOUT_TYPE_VBOX, 0);

      createwidget (GUI_OUTPUT_STATUS, WIDGET_TYPE_TEXT_OUTPUT);

        widget_size_policy    (GUI_OUTPUT_STATUS,  SIZE_POLICY_MINIMUM, SIZE_POLICY_MINIMUM);

        widget_style          (GUI_OUTPUT_STATUS,  ST_BOXTEXT_STYLE);

        widget_fontsize       (GUI_OUTPUT_STATUS,  30);                                

        widget_foreground     (GUI_OUTPUT_STATUS,  "white");   // white or #FFFFF

        widget_text_alignment (GUI_OUTPUT_STATUS,  (WIDGET_TEXT_ALIGN_HCENTER | WIDGET_TEXT_ALIGN_VCENTER));

    endlayout();

  enddraw ();

}

 

Note:

Applicable preprocessor constants (e.g. SIZE_POLICY_FIXED) are available in the file "qtmonitor.h."
