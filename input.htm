input

    

RTC Function

Entry of a variable in default display.

Declaration:

char input (int x, int y, any v)

Group:

Display and keyboard

Parameters:

x
y
v

Horizontal coordinate x in pixels
Vertical coordinate y in pixels
Variable of any type to be entered

Return value:

Code of terminating function key

 

The instruction input creates an input field for the variable specified by v at the position specified by the parameters x and y and enables data entry for the variable through the terminal keyboard or the connected PC keyboard on the default display. The function uses font 28 (RTC Mono) with the currently selected font size. The writing color is black; the background is changed to blue.

The entry is terminated when the enter key, a function key or any other special key is pressed. The key code of the key pressed for termination is returned as function result.

If in advance the variable has been assigned an additional help text via the instruction setinfo, the F5 key can be pressed during data entry to call up that help text. Accordingly, the F5 key is labeled 'Info' and thus no more available for termination of the entry.

 

Example:

showtext (10, 10, "Order");

ch = input (80, 10, OrderNo);

Input of a variable of type bool

The value of v is shown as 'N' for false and – according to the current language setting in Service Mode – with 'Y' (Yes) or e.g. 'O' ('Oui) for true. The keys '0' (false) and '1' (true) can be used for input. Alternatively, the operator can use the horizontal cursor keys to toggle between Yes and No.

Input of a variable of type string

The content of v is shown left-adjusted in the input field. The width of the input field is the length given to the variable during declaration. Data entry is from left to right.

Input of a variable of type decimal

The value of v is shown right-adjusted in the input field. The length of the input field as well as the number of decimals correspond to the format specified during declaration of the variable. During entry, first the digits left of the decimals point are entered right-adjusted with previously entered numbers being shifted left. After pressing the decimal point key, the decimal digits are entered from left to right.

Input of a variable of type int

The value of v is shown right-adjusted in the input field. The length of the input field corresponds to the format specified during declaration of the variable. If no length has been specified during declaration, the variable is shown in a field with a fix length of 10 digits. The input is right-adjusted with previously entered figures being shifted left.

Input of a variable of type float

The value of v is shown right-adjusted in the input field. The length of the input field as well as the number of decimals correspond to the format specified during declaration of the variable. If no format has been specified during declaration, the variable is shown in a field with a fix length of 12 digits. In this case, the number of decimals is fixed to four. The data entry of float variables corresponds to decimal variables.

Input of a variable of type double

The value of v is shown right-adjusted in the input field. The length of the input field as well as the number of decimals correspond to the format specified during declaration of the variable. If no format has been specified during declaration, the variable is shown in a field with a fix length of 19 digits and a fix number 8 trailing decimals. The data entry of float variables corresponds to decimal variables.
