xmlpwrite

    

RTC Function

Sends the content of an XML file through a specific port.

Declaration:

bool xmlpwrite (int id, int port)

Group:

XML interface

Parameters:

id
p

Logical reference number of the XML instance
Logical port number

Return value:

Function result

 

The function xmlpwrite writes the content of the XML instance specified by the parameter id to the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen). If the XML document can be transferred successfully, xmlpwrite returns true as function result; otherwise the result is false.

xmlpwrite (XML_HANDLE, XML_PORT);
