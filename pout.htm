pout

    

RTC Function

Write binary data to a group of digital outputs.

Declaration:

void pout (int n, int msb, int lsb)

Group:

External inputs/outputs

Parameters:

n
msb
lsb

Binary number to be sent
Number of the most significant output of the group
Number of the least significant output of the group

Return value:

-

 

The function pout outputs the integer value n on the group of digital outputs specified by the two parameters msb and lsb. msb defines the output with the highest number and lsb defines the lowest output in the group. Counting starts from 0. The parameter n is interpreted as a binary number. Bit #0 refers to the lowest digital output (lsb), bit #1 to the output lsb+1 and so on.If the bit value is 1, the corresponding output is set (on); otherwise it is cleared (off).

 

Example:

/* Output binary coded valve No. to outputs 5–8 */

pout (ValveNo, 8, 5);
