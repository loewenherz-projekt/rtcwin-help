The do-while Loop

    

The construction do ... while realizes a loop where the condition for continuation is checked after the execution of the loop. The structure is as follows:

do
  statement

while (expression);
 

Statement is executed as long as the result of the Boolean expression is true. If the result of expression is false, the loop is terminated.

·Statement may consist of multiple statements, combined by a pair of curved brackets (compound statement).
·Statement is executed at least once, even if the result of expression is already false at first check.

 

Example:

ch = 0;

do

  /* ... */

  if (keypressed()) ch = readkey();

while (ch != fct6);
