The Programming Language RTC     Communication and File Access

Standard File Access

    

Besides a serial communication, RTC also provides functions to access files:

int fopen (int fileno, string name)

Open a file for read/write access

int fclose (int fileno)

Close a given file

int fwrite (int fileno, string s, int len)

Write a string into a given file

int fread (int fileno, string s, int len)

Read a string from a given file

int fseek (int fileno, int pos, int mode)

Adjust internal read/write position of the given file

int ftell (int fileno)

Return current read/write position of the given file

int fexists (string fname)

Check existence of the specified file

int ffree (void)

Return amount of available memory

bool fcopy (string source, string dest)

Copy the content of a file to another

bool fmove (string source, string dest)

Move and/or rename file

int fremove (string fname)

Delete the specified file

 

For reading individual fields from a CSV file additional functions are available:

bool csvopen (int fileno, string name)

Open a CSV file for read access

bool csvclose (int fileno)

Close a given CSV file

bool csvread (int fileno, int *type, string *value)

Read a field from a given CSV file
