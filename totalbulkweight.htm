totalbulkweight

    

RTC Variable

Variable for the total bulk weight of a totalizing bulk weigher.

Declaration:

loc decimal   totalbulkweight %10.dp

Group:

Scale functions

 

The variable totalbulkweight is a predefined variable of type decimal that contains the representation of the total bulk weight of a totalizing scale. It is updated with the execution of the function getdischargedbulkweight. The variable has a length of 10 digits. The number of trailing decimals corresponds with the most recently accessed scale.

The variable totalbulkweight is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.
