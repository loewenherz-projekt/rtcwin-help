xmllastchild

    

RTC Function

Sets the read/write position to the last node in an XML instance.

Declaration:

bool xmllastchild (int id, string node)

Group:

XML interface

Parameters:

id
node

Logical reference number of the XML instance
Path and name of the node

Return value:

Function result

 

The function xmllastchild selects the node in the XML instance specified by the parameter id and sets the read/write position to its last entry for subsequent operations. If the entry can be selected successfully, xmllastchild returns true as function result; otherwise the result is false.

 

Example:

void WriteLogFile(string msg)

{

  xmlloadfile(XML_WEIGH_LOG, "shared/weighlog.xml")

  // lookup last entry:

  xmllastchild(XML_WEIGH_LOG, "logging");

  // append new entry:

  xmlinsertchild(XML_WEIGH_LOG, "logging", msg, 2, 1));

  xmlsavefile(XML_WEIGH_LOG, "shared/weighlog.xml")

}
