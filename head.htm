head

    

RTC Function

Replaces leading spaces in a string.

Declaration:

string head (string s, char ch)

Group:

String functions

Parameters:

s
ch

Source string
Replacement character

Return value:

Result string

 

The function head replaces the leading spaces of a string with a corresponding number of the provided replacement character ch.

 

Example:

s = head (alibino,'0')  // Replace leading spaces with zeroes
