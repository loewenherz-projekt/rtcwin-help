Fieldbus Communication

    

The communication via fieldbus is completely handled by the operating system of the weighing terminal and has to be set-up in the Service Mode accordingly, e.g. to Profibus DP. The predefined integer array dw serves as an interface to the application program.

int dw[128]

 

The elements 0 to 63 of the integer array represent output words, which correspond to the input words IW0 to IW63 of the PLC. Accordingly, the elements 64 to 127 of the array dw serve as input words corresponding to the output words OW0 to OW63 of the connected PLC.

For access to individual status bits in a data word, the following functions can be used:

bool bit (int bitno, int v)

Check status of a certain bit in an integer variable.

void setbit (int bitno, int v)

Set the specified bit in the given integer variable.

void clrbit (int bitno, int v)

Clear the specified bit in the given integer variable.
