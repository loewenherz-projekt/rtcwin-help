memcpy

    

RTC Function

Copies an integer or float variable into a string or vice versa.

Declaration:

void memcpy (float/int source, string dest)    or
void memcpy (string source, float/int dest)

Group:

Miscellaneous functions

Parameters:

source
dest

Source address
Target address

Return value:

-

 

The function memcpy copies four byte of data from the source address specified by the parameter source to the memory position specified by dest. Thus, memcpy allows to copy integer or float variables into/from a string variable.
