flashsave

    

RTC Function

Copies the application parameters contained in the working memory back to the flash memory.

Declaration:

bool flashsave (void)

Group:

System

Parameters:

-

 

Return value:

Function result

 

The function flashsave writes the copy of application parameters contained in the working area to the flash memory. Thus, all changes made on those parameters are definitively stored and save even in the unlikely event of a battery failure. If the application parameters can be written successfully, flashsave returns true as function result; otherwise the result is false.

 

Example:

#define SOFTWARE_VERSION  "IT8000E 123456 V1.00"

 

flash string SoftwareVersion[30];

flash string UserPassword[10];

flash decimal Minload%7.dp;

flash int TicketNo;

flash bool WithPrinter;

flash bool WithTransmission;

flash int HostPort;

flash int PrtPort;

 
void main (void)

{

  flashload (); // Read parameters from flash memory

  if (SoftwareVersion != SOFTWARE_VERSION)  // First initialization if variable is empty

  {

    SoftwareVersion  = SOFTWARE_VERSION;

    UserPassword     = "";  

    Minload          = 20 * division;  

    TicketNo         = 0;

    WithPrinter      = false;

    WithTransmission = false;

    HostPort         = 0;

    PrtPort          = 1;

    flashsave(); // Set parameters in flash memory

  }

}

 

Note:

Flash memory typically has a limited number of write cycles (some 10,000). Thus, it is not recommended to access the flash memory at a high rate.
