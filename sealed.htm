sealed

    

RTC Function

Indicates whether the calibration jumper of the specified scale is in place.

Declaration:

bool sealed (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Function result

 

The function sealed checks whether the scale specified by the parameter sno is sealed, i.e. whether the calibration jumper is in place. If this is the case, sealed returns true as function result; otherwise the result is false.

 

Example:

/* Application not permitted if calibration jumper 

   not in place. */

for (;;)

{

  if (sealed (1)) 

    break;

  infobox ("Calibration jumper open!");

  ch = readkey();

}

Note:

The information whether the calibration jumper is in place is obtained only once after start-up of the indicator or after exit of Service Mode, respectively. If an execution of sealed returns false as function result, subsequent calls keep returning false until the instrument is started anew or the Service Mode is terminated, even if the calibration jumper is meanwhile put into place.
