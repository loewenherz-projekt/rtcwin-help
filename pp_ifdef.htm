#ifdef

    

RTC Function

Checks a preprocessor constant.

Declaration:

#ifdef  name

Group:

Preprocessor

Parameters:

name

Name of constant

 

With the #ifdef statement, the existence of a preprocessor constant can be checked. If the constant does exist, the subsequent program lines up to the #endif statement are included in the process of compilation. If the constant does not exist, the subsequent program lines are discounted.

Between the #if and #endif statements an optional #else statement may be included. If the constant does exist, the lines between #ifdef and #else are considered for compilation. If the constant does not exist, the part between #else and #endif is used instead.

 

Example:

#ifdef REMOTE_ACCESS

  writestring (Port, "Ready to start");

  ch = preadchar (Port);

#else

  infobox ("Ready to start");

  ch = readkey();

#endif

  if (ch == ok) StartFilling();

 

Note:

# ifdef statements may be nested. However, only one preprocessor statement is accepted per line.
