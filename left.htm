left

    

RTC Function

Left adjusts a string.

Declaration:

string left (string s, int n)

Group:

String functions

Parameters:

s
n

Source string
Length of result string

Return value:

Result string

 

The function left returns a string with the specified length n, that contains the provided source string s left adjusted. Unused characters on the right of the result string are filled with spaces. If n is smaller than the provided source string, the string is truncated accordingly.

 

Example:

s = left ("Article No.", 20);
