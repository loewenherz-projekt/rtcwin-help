hidefields

    

RTC Function

Hide one or more fields in database table.

Declaration:

void hidefields (dbase db, any field_1 ... field_n)

Group:

Database functions

Parameters:

db
field  

Database table
Field to be hidden

Return value:

-

 

The function hidefields hides the table field(s) specified by the given parameter, i.e. makes them unavailable for subsequent calls of the function edit. One or more fields can be provided as parameters, separated by commas. When the function edit is executed, the listed fields are not shown anymore; only the fields not contained in the list are presented for data entry. To make all table fields available for editing the dummy parameter nofield can be used when calling the function hidefields.

 

Example:

void HideProductDBFields (void)

{

  if (!WithBagClamp && !WithAnalogFilling)

    hidefields (Product, BagClampDuration, FastAnalog, SlowAnalog);

  else if (!WithAnalogFilling)

    hidefields (Product, FastAnalog, SlowAnalog);

  else if (!WithBagClamp)

    hidefields (Product, BagClampDuration);

  else

    hidefields (Product, nofield);  // Enable all fields

}

 

Note:

The function hidefields is a method of the declaration word dbase.
