xmlfirstchild

    

RTC Function

Sets the read/write position in an XML instance to the first entry of the node.

Declaration:

bool xmlfirstchild (int id, string node)

Group:

XML interface

Parameters:

id
node

Logical reference number of the XML instance
Path and name of the node

Return value:

Function result

 

The function xmlfirstchild selects the node in the XML instance specified by the parameter id and sets the read/write position to the first entry in its list of child nodes for subsequent operations. If the entry can be selected successfully, xmlfirstchild returns true as function result; otherwise the result is false.

 

Example:

void ReadLogFile(string msg)

{

bool f;

string buf[255];

 

  xmlloadfile(XML_WEIGH_LOG, "shared/weighlog.xml")

  // lookup first entry:

  f = xmlfirstchild(XML_WEIGH_LOG, "logging");

  while (f)

  {

    // read data...

    buf = xmlgetchildvalue(XML_WEIGH_LOG, "logging");

    // ... and evaluate.
     ...

 

    f = xmlnextchild(XML_WEIGH_LOG, "logging");

  }

}
