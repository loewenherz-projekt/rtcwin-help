remain

    

RTC Function

Returns the remaining time of a running timer.

Declaration:

float remain (timer t)

Group:

Timer

Parameters:

t

Variable of type timer

Return value:

Remaining time in seconds.

 

The function remain returns the remaining time in seconds of the timer variable specified by t. Before calling remain, the timer must be started using the function starttimer. If the timer is already timed out, remain returns a negative function result specifying the time elapsed since timeout.

 

Example:

timer CheckTimer;

 

void Timer (void)

{

char ch = 0;

decimal time_value%5.2 = 0;

  

  clearscreen ();

  showtext (10, 10, "Delay time (s):");

  ch = input (160, 10, time_value) ;

  starttimer (CheckTimer, time_value);

  while (!timeout (CheckTimer))

  {

    showtext (10, 25, concat ("Remain time: ", adj (remain (CheckTimer), 3, 2), " s", space (10)));

    if (keypressed())

    {

      ch = readkey();

      stoptimer (CheckTimer);

      showtext (10, 25, concat ("Stop!", space (20)));

      ch = readkey();

      continuetimer (CheckTimer);

    }

    delay (0.01);

  }

  showtext (10, 25, concat ("Ready!", space (20)));

  ch = readkey ();

}
