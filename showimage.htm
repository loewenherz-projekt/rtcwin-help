showimage

    

RTC Function

Shows an image on the standard display.

Declaration:

void showimage (int x, int y, string fname)

Group:

Display and keyboard

Parameters:

x
y
fname

Horizontal coordinate x in pixels
Vertical coordinate y in pixels
Name of image file

Return value:

-

 

The instruction showimage shows the image specified by the parameter fname at the position specified by the two parameters x and y on the default display. fname contains the path and name of the file containing the image as bitmap in PNG (Portable Network Graphics) format.

 

Example:

showimage (10, 10, "shared/scale.png");

 

 

Notes:

The function showimage does not scale the images. It needs to be stored in the size and resolution it will be used in on the scale display. Transparency is supported.

Only .PNG graphics can be used.
