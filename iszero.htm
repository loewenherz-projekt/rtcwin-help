iszero

    

RTC Variable

Status variable for scale within zero range.

Declaration:

loc bool   iszero

Group:

Scale functions

 

The value of the Boolean variable iszero is true, if the scale was within zero range when the functions getweight or getapprovedweight were executed last; otherwise the value of iszero is false.

The variable iszero is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.

 

Example:

/* Clear tare when scale empty */

if (getweight (indicatedscale()) == 0 && iszero)

  err = clrtare (indicatedscale());
