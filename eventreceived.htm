eventreceived

    

RTC Function

Checks if an event was raised.

Declaration:

int eventreceived (void)

Group:

Touch screen programming

Parameters:

-

Return value:

result of type int

 

The function eventreceived checks if an object (widget) has raised an event. If that is the case, eventreceived returns true as function result; otherwise the result is false.

eventreceived only checks the presence of pending events without handling them. If an event was raised, subsequent calls of eventreceived return true until the event is read (e.g. using readevent) or until the event buffer is cleared using dropevents.

As function result, eventreceived returns the event type as follows:

0No event received
1Event from local monitor received
2Event from remote monitor received

 

Example:

void WeighingApplication (void)

{

int source, type, ident;

string event_data[4][255];

 

  selectpage (PAGE_WEIGHING_MODE);

  dropevents (); // Drops all unprocessed events.

  for (;;)

  {

    if (eventreceived() > 0) // Get all events...

    {

      readevent (source, type, ident, event_data); 

      if (source == EVENT_CLASS_APPLICATION) // Only events raised by the application...

      {

        switch (type)

        {

          case EVENT_TYPE_BUTTON_PRESS : 

                     switch (ident)

                     {

                       case ID_BUTTON_0 : return; // Pressed Button_0 return to main menu

                                          break;

                       case ID_BUTTON_1 : /* ... */

                                          break;

                     }

                     break;

          

          case EVENT_TYPE_TEXT_ENTERED :

                     /* ... */

                     break;

          /* ... */

        }

      }

    }

    /* ... */

    delay (0.1);

  }

}
