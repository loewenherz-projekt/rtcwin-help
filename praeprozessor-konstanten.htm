The Programming Language RTC     Tables

Preprocessor Constants

    

To adapt application programs to different hardware platforms or interpreter versions, special predefined preprocessor constants are available.

 

Name

Comment

__IT9000E__

Device is IT9000E

__IT8000E__

Device is IT8000E

__IT6000E__

Device is IT6000E

__SIMULATION__

Device is RTC SIMULATOR

__DATE__

Date when application was compiled

__TIME__

Time when application was compiled

__LINE__

Program line

__FILE__

File name

__RTC_VERSION__

Version string of interpreter, e.g. "4.1.56"

__RTC_MAJOR__

RTC interpreter main version, e.g. "4"

__RTC_MINOR__

RTC interpreter subversion, e.g. "1"

__RTC_PATCH_LEVEL__

RTC interpreter patch level, e.g. "56"

__MACADDRESS__

MAC address of weighing terminal

__PLATFORM__

Device description, e.g. "IT8000E"

 

Example:

/* Output short beep... */

#ifndef __IT9000E__

  if (WithBeeper) beep (2700, 0.2);

#endif
