messagebox

    

RTC Function

Shows a pop up window with a message.

Declaration:

void messagebox (string msg, string buttons)

Group:

Touch screen programming

Parameters:

msg
buttons

Message text
List with button texts

Return value:

-

 

The instruction messagebox opens a window showing the text provided by the parameter msg. The text is shown in black proportional characters and may contain line break commands "\n". The parameter buttons contains the texts for up to six function keys shown at the bottom line of the window. The individual button texts have to be separated by semicolons. After pressing one of the function keys, the message disappears and the pre-defined variable button then contains the key code of the pressed function key. The key codes comply with the codes of the function keys F1 to F6 from left to right.

If the surface of the window is clicked rather than one of the buttons, the window disappears as well and the key code of the clear key is returned in the variable button. Thus, it is possible to create message windows without keys by passing an empty string as parameter buttons.

 

Example:

messagebox("Print again?", "Yes;No");
