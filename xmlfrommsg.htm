xmlfrommsg

    

RTC Function

Transfers an XML document from the message queue to an XML instance.

Declaration:

bool xmlfrommsg (int id, string *data)

Group:

XML interface

Parameters:

id
data

Logical reference number of the XML instance
Variable for transferring metadata

Return value:

Function result

 

The function xmlfrommsg reads an XML document from the message queue. The availability of a not yet handled message matching the filter settings (set-up by the instruction msgsetfilter) can be checked beforehand using the function msgreceived. If the XML document can be read successfully, it is written into the XML instance specified by the parameter id for subsequent evaluation. Possibly available additional data, e.g. information about how to interpret the document, is copied into the provided string variable data. If the XML document can be transferred successfully, xmlfrommsg returns true as function result; otherwise the result is false.

 

Example:

xmlfrommsg (XML_CONF_HANDLE, data);

if (data == "Send")

  SendConfiguration();

else

  ReceiveConfiguration();

 

 

Note:

The XML instance is automatically cleared before new content is written to it. Calling xmlclear is not necessary.
