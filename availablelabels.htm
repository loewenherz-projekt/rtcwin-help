availablelabels

    

RTC Function

Returns the number of layouts available for a specific print format.

Declaration:

int availablelabels (string format)

Group:

Communication

Parameters:

format

Print format

Return value:

Number of available layout files

 

The function availablelabels returns the number of layouts that are available for the stated print format. The print format is referenced by the name that was assigned during definition using form. The layouts have to be saved in the internal directory shared with the extension lab. The name of the layout file has to start with the format name if the layout is to be assigned to a specific print format.

 

Example:

i = availablelabels (Singleticket); 

if (i > 1)

{

  savescreen (1);

  background (CL_BACKGROUND);

  fontsize (15);

  font (FT_RTC_BOLD);

  posto (5, 43);

  boxto (310, 100, 2);

  showtext (20, 50, "Select print format:");

  Layoutname = inputlabel (20, 72, Singleticket, Layoutname); 

  restorescreen (1);

  print (PrtPort, Singleticket, Layoutname);

}
