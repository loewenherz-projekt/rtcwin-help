ain

    

RTC Function

Reads an analog input.

Declaration:

float ain (int no)

Group:

External inputs/outputs

Parameters:

no

Number of analog input

Return value:

Input signal

 

The function ain reads the analog input specified by the provided parameter no. Counting starts from 0 for the first analog input. The analog input signal is – according to current Service Mode settings – converted into a percentage 0 to 100 % and returned as function result.

 

Example:

/* Read target from first analog input */

Target = MaxTarget * ain (0) / 100.0;
