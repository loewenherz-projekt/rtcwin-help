posto

    

RTC Function

Set cursor position on the standard display.

Declaration:

void posto (int x, int y)

Group:

Display and keyboard

Parameters:

x
y

Horizontal coordinate x in pixels
Vertical coordinate y in pixels

Return value:

-

 

The instruction posto defines the start position for the subsequent drawing commands lineto, boxto, clearto, etc. The position is specified via the two provided coordinates x and y.

 

Example:

posto (10, 20);
lineto (240, 20); /* Draw horizontal line */
