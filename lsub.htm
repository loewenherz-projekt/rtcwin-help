lsub

    

RTC Function

Separates a substring according to the selected language.

Declaration:

string lsub (string s)

Group:

String functions

Parameters:

-    

String

Return value:

Result string

 

The function lsub returns a substring according to the currently selected language. The language substrings contained in s are separated by the character '|' (ASCII 124). If s contains fewer substrings than required for proper language selection, the first substring is returned instead.

 

Example:

showtext (20, 10, lsub ("Article File|Archivo de Artículos"));

 

Notes:

The function lsub does not interpret the language name set in Service Mode but rather returns the according substring, e.g. the second substring if the second language is set. The pragma statement determines the selectable language options and their order in the Service Mode.

The first substring of a multi-lingual text should always be the default language, e.g. English.
