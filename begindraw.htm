begindraw

    

RTC Function

Starts buffering of subsequent display functions.

Declaration:

void begindraw (void)

Group:

Display and keyboard

Parameters:

-

 

Return value:

-

 

The instruction begindraw starts buffering of display functions, i.e. subsequent write and draw operations are not executed but stored in a buffer. The concluding enddraw statement terminates buffering and executes the buffered instructions in one go. This results in a reduction of execution time and avoids flickering where extensive screen masks have to be shown.

 

Example:

begindraw();

  clearscreen();

  font (24);

  fontsize (13);

  showtext (10, 0, "Weighing Mode");

  posto (10, 18); lineto (310, 18);

  font (25);

  showtext (10, 22, "Order No.");

  showtext (10, 40, "Customer");

  font (29);

  showtext (100, 22, OrderNo);

  showtext (100, 40, Client.No);

  showtext (130, 40, Client.Name);

  /* ... */

enddraw();

 

Note:

The buffering storage for write operations is limited. If the storage is full and cannot hold any further operations, it is emptied by executing the instructions without enddraw.Thus, it may be useful to update the display in several steps when executing extensive write or draw operations.
