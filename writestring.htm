writestring

    

RTC Function

Outputs a string on a communication channel.

Declaration:

int writestring (int port, string s)

Group:

Communication

Parameters:

port
s

Logical port number
String to be sent

Return value:

Error code

 

The function writestring transmits the provided string s on the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen).

Data transmission is carried out using the communication protocol set up in Service Mode. If the string can be transmitted successfully, writestring returns 0 as function result; otherwise the corresponding error code is returned instead.

 

Example:

/* Data transmission */

bool DataTransmission (int port, string weighing_data)

{

string receive_buffer[128] = "";

timer  t;

 

  if (ReadComFifo())

  {

    pclear (port); // clear receive buffer 

    if (writestring (port, weighing_data) == 0)

    {

      starttimer (t, 10.0);

      while (!stringreceived (port)) // Wait for answer...

      {

        if (timeout (t)) 

        {

          ShowErrorMsg ("Error: Timeout");

          return (false);  

        }    

        delay (0.01);

      } 

      if (readstring (port, receive_buffer) == 0)

      {

        if (receive_buffer == "OK")

          return (true);  

      }

      else

        ShowErrorMsg ("Error: Data transmission");

    }

    else

      ShowErrorMsg ("Error: Data transmission");

  }

  return (false);

}
