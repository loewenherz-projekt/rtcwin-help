select

    

RTC Function

Selection from a database table.

Declaration:

char select (dbase db, int x1, int y1, int x2, int y2)

Group:

Display and keyboard

Parameters:

x1
y1
x2
y2
db

Horizontal coordinate x of the index field in pixels
Vertical coordinate y of the index field in pixels
Horizontal coordinate x of the search field in pixels
Vertical coordinate y of the search field in pixels
Database table

Return value:

Code of terminating function key

 

The instruction select creates a selection dialog for the provided database table db in the default display. For this, an input field is provided for the index (i.e. the table field declared first) at the position specified by the two parameters x1 and y1 where the operator can enter data through the terminal keyboard or a connected PC keyboard. The text is written using the currently selected font size, character set and colors. The background color changes to blue.

If the entry matches the index field of a record contained in the database table db, the second field of this record is shown for confirmation at the position specified by the parameters x2 and y2. Again, the currently selected font and attributes including the background color are used.

Besides the normal entry of the index field, the operator can use the cursor right and left keys to browse through the available records contained in the database table. With every key press the record with the next higher (cursor right) – or lower (cursor left) – index field comes up.

The entry is terminated when the enter key, a function key or any other special key is pressed. The code of the key pressed for termination is returned as function result.

If in advance the index field of the database table has been assigned an additional help text via the instruction setinfo, the F5 key can be pressed during selection to call up that help text. Accordingly, the F5 key is labeled 'Info' and thus no more available for termination of the function select.

 

Example:

showtext (10, 10, "Client");

ch = select (Client, 0, 10, 120, 10);

Note:

The function select is a property of the declaration word dbase.

Alternative Search in a Database Table

Besides the above-described method to select a record through its index, the function select also provides the option to select a record via the second field, the so-called search field. After pressing the F3 function key a list of table entries comes up, sorted by the search field in ascending order.

 

The operator can select a record with the cursor up and down keys. The selected record is indicated by bold characters. The cursor right and left keys can be used to shift the listed records horizontally to view further fields of the table. By pressing the enter key the selection is made and the record is shown in the main select dialog (see also searchbox).
