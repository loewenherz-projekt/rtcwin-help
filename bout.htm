bout

    

RTC Function

Sets or clears the status of a digital output.

Declaration:

void bout (int bitno, bool stat)

Group:

External inputs/outputs

Parameters:

bitno
stat

Number of digital output
Output status to be set

Return value:

-

 

The function bout sets the output specified by the parameter bitno. Counting starts from 0. The output is set (on) if the parameter stat = true; otherwise if the parameter stat = false the output is cleared (off).

 

Example:

#define DISABLE 0

#define ENABLE  1

#define OUT0_READY_TO_START 0 

#define IN0_START_COMMAND 0

 

/* Wait for start */

char WaitForStart (void)

{

char ch = 0;

 

  bout (OUT0_READY_TO_START, ENABLE);

  while ((ch != ok) && (ch != fct6))

  {

    if (keypressed()) 

      ch = readkey();

      

    if (bin (IN0_START_COMMAND)) 

      ch = ok;

    delay (0.05);

  }

  return (ch);

}
