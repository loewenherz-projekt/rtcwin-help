gettime

    

RTC Function

Reads the internal real-time clock and updates the date and time variables.

Declaration:

void gettime (void)

Group:

System

Parameters:

-

Return value:

-

 

The function gettime reads the internal real-time clock and updates the variables date and time. The string variable date has a fix length of 8 or 10 characters and contains the date with two- or four-digit year formatted as set in Service Mode. The string variable time has a fix length of 5 or 8 characters and contains the time in "HH:MM" or "HH:MM:SS" format as set in Service Mode.
