getapprovedweight

    

RTC Function

Performs an approved weighing cycle and updates all the weight variables.

Declaration:

int getapprovedweight ([int sno])

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Error code

 

The function getapprovedweight reads the scale specified by the parameter sno and waits for the scale to stabilize (issettled=true). If the scale delivers stable readings within 6 seconds, the weight variables are updated and the weighing is recorded in the approved weight storage (alibi memory). The variable alibino thereafter contains the identification number the weighing is stored against.

If getapprovedweight can be executed successfully, the weight variables gross, net and tare contain the corresponding weights as numeric values. The variable nethires contains the net weight in a 100-times-higher resolution. In addition, the string variables fgross, fnet and ftare are updated with weights readily formatted for printout. Besides the weight variables, also the status variables issettled, iszero, taremode and range and the variable date are updated.

If the scale can be read successfully, getapprovedweight returns 0 as function result; otherwise the corresponding error code is returned instead and the variable alibino contains 0.

 

Example:

/* Read approved weight */

char Registration (void)

{

int  err;

char ch = 0;

 

  infobox ("Registration...");
  do

  {

    err = getweight (indicatedscale());

    if (keypressed()) ch = readkey();

  }

  while (err == 0 && !issettled && ch != fct6);

    if (err != 0 || ch == fct6) 

      return (fct6);

  err = getapprovedweight (indicatedscale());
  /* Print weighing ticket... */

}

 

Note:

The function gettime has to be used if also the variable time needs to be updated. However, it has to be taken into account, that this also updates the variable date, which then might change and no longer match the record in the approved weight storage (alibi memory).
