The Programming Language RTC     Printout Formatting

Definition of a Layout

    

The declaration word form is followed by the name of the print format and – embraced in curved brackets – the individual print fields. Each field consists of a position in line and column numbers and the content, which can either be a text or a variable.

The print field may be assigned one or more of the following attributes.

     bold

Emphasized printing in bold characters

     underlined

Underlined characters

     condensed

Printing with reduced width

     italic

Printing in italics

Alternatively, the printer control sequence can be specified directly, e.g. if the required print attribute is not contained in the list above. To do so, the ASCII code of the individual control characters is entered in decimal notation and enclosed in angle brackets. Where the sequence is composed of more than one character, they have to be separated by commas. Typically, the sequence to enable a function is placed in front of a print field; the command to disable the function is put behind the field.
