getfilledbulkweight

    

RTC Function

Captures the weight of the filled weigh hopper of a totalizing bulk weigher.

Declaration:

int getfilledbulkweight (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Error code

 

The function getfilledbulkweight captures the weight of the filled weigh hopper of a totalizing bulk weigher and stores it in an internal variable. A subsequent call of getdischargedbulkweight then reads the weight of the emptied hopper, calculates the difference and adds it to the sum in the variable totalbulkweight.

The instruction getfilledbulkweight reads the scale specified by the parameter sno and waits for the scale to stabilize (issettled = true). If the scale returns a stable weight, the weight values are updated as described in getweight.

If the function getfilledbulkweight can be executed successfully, getfilledbulkweight returns 0 as function result; otherwise the corresponding error code is returned instead.
