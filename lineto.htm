lineto

    

RTC Function

Draws a line.

Declaration:

void lineto (int x, int y)

Group:

Display and keyboard

Parameters:

x
y

Horizontal coordinate x of the end of the line
Vertical coordinate y of the end of the line

Return value:

-

 

The instruction lineto draws a line on the application screen from the current cursor position to the position specified by the coordinates x and y. The line has a thickness of one pixel and is drawn using the currently selected drawing color. The instruction lineto only contains the coordinates of the end position of the line. I.e. subsequent calls of lineto can be used to draw complex forms (see polygon).

 

Example:

posto(40, 20);   /* Determine start point */

lineto(240, 20); /* Draw horizontal line */
