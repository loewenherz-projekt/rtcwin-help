inputlabel

    

RTC Function

Selection of a layout file in default display.

Declaration:

string inputlabel (int x, int y, form format, string layout]

Group:

Display and keyboard

Parameters:

x
y
format
layout

Horizontal coordinate x in pixels
Vertical coordinate y in pixels
Print format
Name of layout file

Return value:

Name of selected layout file

 

The instruction inputlabel creates a select dialog for the print format specified by format at the position specified by the two parameters x and y on the default display. The operator can then select a layout from the layout files assigned to the print format using the cursor right and left keys of the weighing terminal keyboard or the connected PC keyboard. The text is written using the currently selected font size, character set and colors. The background color changes to blue.

The print format is referenced by the name that was assigned during definition using form. The layouts have to be saved in the internal directory shared with the extension lab. The name of the layout file has to start with the format name if the layout is to be assigned to a specific print format. The number of available layout files can be evaluated using the instruction availablelabels.

 

Example:

i = availablelabels (Singleticket); 

if (i > 1)

{

  savescreen (1);

  background (CL_BACKGROUND);

  fontsize (15);

  font (FT_RTC_BOLD);

  posto (5, 43);

  boxto (310, 100, 2);

  showtext (20, 50, "Select print format:");

  Layoutname = inputlabel (20, 72, Singleticket, Layoutname); 

  restorescreen (1);

  print (PrtPort, Singleticket, Layoutname);

}
