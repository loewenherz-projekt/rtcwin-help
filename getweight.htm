getweight

    

RTC Function

Reads the scale and updates the weight variables.

Declaration:

int getweight ([int sno])

Group:

Scale functions

Parameters:

sno       Optional scale number

Return value:

Error code

 

The function getweight reads the scale specified by the parameter sno and updates the weight variables. If the parameter sno is missing, the first scale is read instead.

The weight variables gross, net and tare contain the corresponding weights as numeric values. The variable nethires contains the net weight in a 100-times-higher resolution. In addition, the string variables fgross, fnet and ftare are updated with weights readily formatted for printout. Besides the weight variables, also the status variables issettled, iszero, taremode and range are updated.

If the scale can be read successfully, getweight returns 0 as function result; otherwise the corresponding error code is returned instead.

 

Example:

/* Read approved weight */

char Registration (void)

{

int err;

char ch = 0;

 

  infobox ("Registration...");
  do

  {

    err = getweight (indicatedscale());

    if (keypressed()) ch = readkey();
    delay(0.1);

  }

  while (err == 0 && !issettled && ch != fct6);

    if (err != 0 || ch == fct6) 

      return (fct6);

  err = getapprovedweight (indicatedscale());
  /* Print weighing ticket... */

}
