abortinput

    

RTC Function

Aborts an operator prompt.

Declaration:

void abortinput (void)

Group:

Touch screen programming

Parameters:

-

 

Return value:

-

 

The function abortinput terminates the operator input under program control – e.g. out of a background task. The input object of type WIDGET_TYPE_TEXT_INPUT immediately terminates the input mode and returns the event EVENT_TYPE_TEXT_ABORTED.
