The Programming Language RTC     Databases

Database Functions

    

RTC provides a range of powerful functions that apply to database tables, i.e. to the data type dbase:

 

clear

Clears and initializes a database table.

insert

Inserts a record in a database table. If the table already contains a record with identical index field (first field), it is overwritten.

find

Finds the record matching the index field.

findnext

Finds the record succeeding the index field.

findprevious

Finds the record preceding the index field.

findfield

Finds the record matching the specified database field.

findnextfield

Finds the record succeeding the specified database field.

findprevfield

Finds the record preceding the specified database field.

delete

Deletes the record matching the specified index field.

records

Determines the number of records currently contained in the table.

edit

Opens a complete dialog for creating/editing records via display/keyboard of the weighing terminal.

select

Creates a display/input field on the screen for selecting a record from the table.

searchbox

Opens a complete dialog for searching and selecting a record via display/keyboard of the weighing terminal.

lock

Locks the record matching the specified index field, so it can no longer be modified or deleted.

unlock

Unlocks the record matching the specified index field.

save

Writes the specified database table into a CSV file.

load

Loads the specified database table from a CSV file.
