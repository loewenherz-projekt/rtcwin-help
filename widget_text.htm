widget_text

    

RTC Function

Sets the text of an input or output object.

Declaration:

void widget_text (int id, string text,  [int row], [int column])

Group:

Touch screen programming

Parameters:

id
text
row
column

Logical reference number of the object
Text to be shown
Optional row number
Optional column number

Return value:

-

 

The instruction widget_text sets the text the object specified by the parameter id. The object has to be a text object e.g. of type WIDGET_TYPE_TEXT_OUTPUT, WIDGET_TYPE_TEXT_INPUT or a list or table object of type WIDGET_TYPE_LIST or WIDGET_TYPE_TABLE.

If the object is a list or table object, the two optional parameters row and column have to be used to address the respective position of the text. If the object only contains one text, the parameters can be omitted.

 

Beispiel:

void CreateMyInputBox (void)

{

  begindraw ();

  createpage (PAGE_MY_INPUT_BOX);

    beginlayout (LAYOUT_TYPE_VBOX, 0, 2);

 

      beginlayout (LAYOUT_TYPE_HBOX, 0);  // Layout for headline   

        createwidget (ID_HEADLINE, WIDGET_TYPE_TEXT_OUTPUT);

          widget_fontsize       (ID_HEADLINE,  30); 

          widget_fontface       (ID_HEADLINE,  FT_RTC);

          widget_size_policy    (ID_HEADLINE,  SIZE_POLICY_MINIMUMEXPANDING, SIZE_POLICY_MINIMUM); 

          widget_background     (ID_HEADLINE,  "#5a7cf9");  // SysTec Default

          widget_foreground     (ID_HEADLINE,  "white");

          widget_text_alignment (ID_HEADLINE,  WIDGET_TEXT_ALIGN_LEFT);

          widget_text           (ID_HEADLINE,  lsub (TX_INPUTBOX));

      endlayout ();

 

      /* Further input or output objects */

 

      beginlayout (LAYOUT_TYPE_HBOX, 0, 2); // Layout for Codepage selection

        createwidget (ID_CODE_PAGE_PROMPT, WIDGET_TYPE_TEXT_OUTPUT);

          widget_text (ID_CODE_PAGE_PROMPT, "Codepage", 0);   

                 

        createwidget (ID_CODE_PAGE_SELECTION, WIDGET_TYPE_COMBOBOX);

          widget_text      (ID_CODE_PAGE_SELECTION, "None", 0);

          widget_text      (ID_CODE_PAGE_SELECTION, "850",  1);

          widget_text      (ID_CODE_PAGE_SELECTION, "852",  2);

          widget_text      (ID_CODE_PAGE_SELECTION, "866",  3);

          widget_set_index (ID_CODE_PAGE_SELECTION, 2, 0);

      endlayout ();

 

    endlayout ();

  enddraw ();

}

 

Note:

The instruction widget_text cannot be used for a drawing area of type WIDGET_TYPE_SKETCH. Use the instruction showtext instead.
