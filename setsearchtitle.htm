setsearchtitle

    

RTC Function

Assigns a database table a headline for the search function.

Declaration:

void setsearchtitle (dbase db, string title)

Group:

Display and keyboard

Parameters:

db
title

Database table
Title string

Return value:

-

 

The function setsearchtitle assigns the given database table a title string for the function searchbox. The given title string contains the headers of the individual table fields, separated by semicolons. The title is written on top in black bold characters, underlined, followed by the list of records.

If the database table is not assigned a title string, a headline is not printed and the list of records starts from the topmost position of the application screen.

 

Example:

dbase "Customer file" Client[200]

 

{

  "Client No."    decimal  No%3.0; selectmode = input;

  "Name"          string   Name[24];

  " "             string   Name2[24];

  "Street"        string   Street[24];

  "ZIP code"      string   Zip[5];

  "City"          string   City[24];

}

 

setsearchtitle (Client, "No.;Name;;Street;ZIP;City");
