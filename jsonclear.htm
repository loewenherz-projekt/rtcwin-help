jsonclear

    

RTC Function

Deletes the content of a JSON statement list.

Declaration:

bool jsonclear (int id)

Group:

JSON interface

Parameters:

id

Logical reference number of the JSON statement list

Return value:

Function result

 

The function jsonclear clears the content of a JSON statement list specified by the parameter id. The logical reference number id has a range from 0 to 7 and addresses one of 8 possible JSON statement lists.

 

Example:

/* Enable input on website */

int JSON_EnableInput(int handle, string input_id, bool stat)

{

int     index = 0;

string  cmd[10] = "setvalues";

 

  jsonclear(handle);

  if (stat)

  {

    JSON_SetHTMLValue(handle, cmd, index++, input_id, "disabled", "false");

    JSON_SetHTMLValue(handle, cmd, index++, input_id, "focus", "");

  }

  else

    JSON_SetHTMLValue(handle, cmd, index++, input_id, "disabled", "true");

  return(jsontomsg (handle, EVENT_APPLICATION_WS_SERVER, MSG_APPL_JSON, ""));

}

 

 

Note:

The functions jsonloadfile, jsonfromstring and jsonfrommsg automatically clear the statement list before new content is written. Calling jsonclear is not necessary.
