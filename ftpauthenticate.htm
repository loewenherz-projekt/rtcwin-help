ftpauthenticate

    

RTC Function

Performs a log-in on an FTP server.

Declaration:

bool ftpauthenticate (int fno, string name, string pw)

Group:

File handling

Parameters:

fno
name
pw

Logical connection number
Log-in name
Password

Return value:

Function result

 

The function ftpauthenticate performs a log-in on an FTP server for the connection specified by the parameter fno, provided that a log-in is required. The parameter fno is the logical connection number assigned with the execution of the function ftpopen. The parameters name and pw specify the user name and password required as log-in details. If the log-in can be carried out successfully, ftpauthenticate returns true as function result; otherwise the result is false.

 

Example:

/* Get order file from FTP server */

 

#define FTP_PORT 21

 

bool GetOrders (void)

{

bool result = false;

 

  if (ftpopen (0, ServerIP, FTP_PORT))  // Open FTP connection...

  {

    if (ftpauthenticate (0, LoginName, LoginPassword))  // Log-in ...

    {

      // get new orders and copy to 'shared'...

      if (ftpget (0, "Production/Scale_1/Orders.csv", "shared/Orders.csv"))  

      {

        result = true;

        // delete order file from server

        if (! ftpremove (0, "Production/Scale_1/Orders.csv"))

          ShowErrorMsg ("FTP access denied!");

      }

      else

        ShowErrorMsg ("No new orders available!");

    }

    else

      ShowErrorMsg ("FTP access denied!");

 

    if (!ftpclose (0))

      ShowErrorMsg ("FTP access denied!");      

  }

  else

    ShowErrorMsg ("FTP server not available!");

  return (result);

}

 

Note:

If the FTP server allows anonymous log-in, calling ftpauthenticate is not necessary.
