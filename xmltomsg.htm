xmltomsg

    

RTC Function

Sends an XML document of an XML instance to the message handler.

Declaration:

bool xmltomsg (int id, int dest, int type, string data, bool auto_id)

Group:

XML interface

Parameters:

id
dest
type
data
auto_id

Logical reference number of the XML instance
Destination of the XML document
Message type
Additional data for destination
Automatic assignment of ID

Return value:

Function result

 

The function xmltomsg sends the document of the XML instance specified by the parameter id to the message handler for subsequent transfer to another instance. The parameter dest specifies the destination of the XML document as follows:

 

Destination

Description

0

Undefined

3

Application (interpreter)

8

Web interface

9

JSON server (Java script object notification)

Other destinations exist for internal communication. The parameter type returns the message type as follows:

 

Message

Description

9

The XML document contains the configuration parameters requested by the WEB interface of the application.

10

The XML document contains the settings requested by the WEB interface of the application.

11

Java script object notification (JSON) for the process visualization.

Other message numbers are reserved for internal purpose. The parameter data can be used to send additional data (metadata), e.g. information about how to process the XML document. The last parameter auto_id activates the automatic assignment of unique IDs; it should always be true.

If the XML document can be transferred to the message handler successfully, xmltomsg returns true as function result; otherwise the result is false.

 

Example:

xmlfrommsg (XML_CONF_HANDLE, data);

if (data == "Send")

{

  WriteBasicSettingsToXML();

  xmltomsg (XML_CONF_HANDLE, msgsrc(), MSG_APPL_CONFIG, "", true);

}

else

  ReceiveConfiguration();
