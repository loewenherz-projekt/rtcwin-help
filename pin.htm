pin

    

RTC Function

Reads data from a group of digital inputs.

Declaration:

int pin (int msb, int lsb)

Group:

External inputs/outputs

Parameters:

msb
lsb

Number of the most significant input of the group
Number of the least significant input of the group

Return value:

Status of digital inputs as binary number

 

The function pin reads a group of digital inputs specified by the two parameters msb and lsb. msb defines the input with the highest number and lsb defines the lowest input in the group. Counting starts from 0. The status of the inputs is represented by the bits of an integer number and returned as function result. Bit #0 represents the lowest bit in the group (digital input lsb), bit #1 the next bit (input lsb+1) and so on.The value 1 reflects the status "set" and 0 the status "not set" of the corresponding input.

The case msb = lsb reads one discrete input only and thus complies with the function bin.

 

Example:

/* Read material No. through inputs 5–8 */

MatNo = pin (8, 5);

 

Note:

If pin is used to read more inputs than available, the obsolete bits of the function result are set to 0.
