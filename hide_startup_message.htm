hide_startup_message

    

RTC Function

Hides the power-up message from the screen.

Declaration:

void hide_startup_message (void)

Group:

Touch screen programming

Parameters:

-

 

Return value:

-

 

The instruction hide_startup_message hides the power-up message displayed by show_startup_message.

 

Example:

void main(void)
{

  show_startup_message ("", "<H1 ALIGN=CENTER><font size=5>Starting up....</font size></H1>")

  CreatePages();
  Init();

  hide_startup_message();
  MainApplication();

}
