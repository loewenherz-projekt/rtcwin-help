widget_enable

    

RTC Function

Activates/deactivates an object.

Declaration:

void widget_enable (int id, bool enable)

Group:

Touch screen programming

Parameters:

id
enable

Logical reference number of the object
Object status to be set

Return value:

-

 

The instruction widget_enable activates or deactivates the object specified by the parameter id. If the parameter enable is true, the object is activated and can be used by the operator. If enable is false, the object is deactivated (grayed) and cannot be used by the operator.

 

Example:

if (StartConditionOK ())       

  widget_enable (START_BUTTON, true);

else
  widget_enable (START_BUTTON, false);
