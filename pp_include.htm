#include

    

RTC Function

Embeds an external file into the program.

Declaration:

#include  fname

Group:

Preprocessor

Parameters:

fname

Name of external file

 

With the instruction #include an external file can be included in the program design. The external file is specified by the given name fname (in quotation marks or angle brackets) and is embedded in the program file at the position of the #include statement. The external file has to be contained in the current project.

 

Example:

#include  "Prompts.c"   /* Language-dependent operator prompts */

 

Notes:

For encrypted binary files, the statement #binary has to be used.
