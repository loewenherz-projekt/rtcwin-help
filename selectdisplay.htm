selectdisplay

    

RTC Function

Selects the display and keypad for the following read and write operations.

Declaration:

bool selectdisplay (int dno)

Group:

Display and keyboard

Parameters:

dno

Number of the display 0 (main) or 1 (secondary display)

Return value:

result of type bool

 

The function selectdisplay selects the display specified by the parameter dno for subsequent write and draw operations. Valid values are 0 for the primary and 1 for the secondary display. The display of the main unit and the remote indicator have to be configured as primary and secondary, accordingly.

Switching to another display also selects the associated keyboard for subsequent data entry. If the display can be selected successfully (display available and ready), selectdisplay returns true as function result; otherwise the result is false.

 

Example:

...

selectdisplay (0);

if (keypressed())          // Key press on main display?

{

  selectdisplay (1);        // switch to secondary display ...

  infobox ("Locked!");    // … and show message.

  selectdisplay (0);

  WeighingMode();          // Weighing mode on main display

}

else

{

  selectdisplay (1);

  if (keypressed())        // Key press on secondary display?

  {

    selectdisplay (0);      // switch to main display...

    infobox ("Locked!");  // … and show message.

    selectdisplay (1);

    WeighingMode();        // Weighing mode on secondary display

  }

}
