arc

    

RTC Function

Draws an elliptical arc.

Declaration:

void arc (int w, int h, int w0, int w1)

Group:

Display and keyboard

Parameters:

w
h
w0
w1

Horizontal width of the ellipse (X direction)
Vertical height of the ellipse (Y direction)
Start angle of the segment
End angle of the segment

Return value:

-

 

The instruction arc draws an elliptical arc. The current cursor position specifies the center of the reference ellipse whereas the two parameters w and h specify their width and height.The two other parameters specify start and end angle of the arc. Angles start with 0° for the rightmost position on the horizontal axis and are counted counterclockwise.

 

Example:

posto (20, 20);

arc (40, 20, 0, 45);
