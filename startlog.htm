startlog

    

RTC Function

Starts real-time logging of weight readings.

Declaration:

void startlog (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

-

 

The function startlog starts logging of weight readings for the scale specified by the parameter sno. During logging all weights together with the internal system time are written in real-time into a ring buffer. Logging can be terminated using the instruction stoplog. With the function readlog, the logged weights can then be read for evaluation.

The ring buffer has a capacity of 24000 records and thus – for example – allows weight logging for more than 100 seconds at an update rate of 225 Hz.

 

Example:

/* Capture dynamic weight */

void CatchWeight (void)

{

timer t;

 

  starttimer (t, MeasureTime);

  startlog (1);

  for (;;)

    if (timeout (t)) break;

  stoplog (1);

  /*  Evaluate ring buffer and determine weight... */

}
