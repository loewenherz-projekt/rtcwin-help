The Programming Language RTC   Loops

The for Loop

    

The for loop is basically a while loop with integrated initialization and maintenance of the loop variable. The principal structure is:

for (Expression_1; Expression_2; Expression_3)
  Statement

 

Where

·Expression_1

Initialization of loop variable.
·Expression_2

Condition for continuing the loop.
·Expression_3

Maintenance of the loop variable at the end of a loop cycle.

 

In clear language: Expression_1 contains the initialization of the loop variable. Statement is executed as long as the result of the Boolean expression Expression_2 is true. If the result of Expression_2 is false, execution of the loop is terminated. Expression_3 is executed at the end of every statement loop cycle and – for example – is used to increment the loop variable.

·There is no semicolon behind Expression_3.
·Each of the three expressions has to be present (exception: all three expressions are missing). Multiple loop variables are not permitted.
·The for loop must not be empty, i.e. statement has to be appended. Statement may consist of multiple statements, combined by a pair of curved brackets (compound statement).

 

Example:

string LeadingZero (string s)

{

int i;

  for (i = 0; s[i] == ' ' && i < length(s); i++)

    s[i] = '0';

  return (s);  

}

 

The following example shows a special form of the for loop and is the realization of an endless loop:

 

Example:

for (;;)
{

  /*... */

}
