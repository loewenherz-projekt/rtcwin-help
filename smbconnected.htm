smbconnected

    

RTC Function

Checks the status of a secure communication channel.

Declaration:

bool smbconnected (string path)

Group:

File handling

Parameters:

path

Directory path

Return value:

Connection status

 

The function smbconnected checks the status of a secure communication channel. The directory path is defined by the parameter path that was returned as the function result of smbconnect. If the connection is still available, smbconnected returns true as function result; otherwise the result is false.
