keymatrix_generate

    

RTC Function

Creates a window with a matrix of buttons.

Declaration:

void keymatrix_generate (int id, int rows, int cols)

Group:

Touch screen programming

Parameters:

id
rows
cols

Logical reference number of the window
Number of matrix rows
Number of matrix columns

Return value:

-

 

The instruction keymatrix_generate creates a page with a given number of buttons. The parameter id can be freely chosen by the programmer and has to be unique. It is used to identify and reference this page for later use.

The buttons are arranged in a matrix of rows and columns. The number of rows and columns (cols) is defined by the corresponding parameters. Each button can be assigned an individual text using the instruction keymatrix_text. The color of the buttons can be specified using keymatrix_color.

In addition to the generated matrix of buttons, the screen also contains an extra button in its lower left corner named "back," which returns the F1 key code when it is pressed.

The screen can be activated using the instruction keymatrix_select and then waits for any of the buttons being pressed.

 

Example:

void CreateSettingsMenu (void)

{

  keymatrix_generate (PAGE_MAIN_SETTINGS, 4, 1);  // 4 buttons one below the other

    keymatrix_color (PAGE_MAIN_SETTINGS, 0, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 0, "Produkte");

    keymatrix_color (PAGE_MAIN_SETTINGS, 1, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 1, "Taraspeicher");

    keymatrix_color (PAGE_MAIN_SETTINGS, 2, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 2, "Einstellungen");

    keymatrix_color (PAGE_MAIN_SETTINGS, 2, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 2, "Datensicherung");

}

void Settings (void)

{

char ch = 0; 

int key = 0;

 

  ch = keymatrix_select (PAGE_MAIN_SETTINGS, "Data input", key);

  if (ch == ok)

    switch (key)

    {

      case 0:  EditProducts();

               break;

      case 1:  EditTareValues();

               break;

      case 2:  Settings();

               break;

      case 3:  DataBackup();

               break;

    }

}
