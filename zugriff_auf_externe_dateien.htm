The Programming Language RTC     Communication and File Access

Access to External Files

    

Access to external files is possible via FTP (File Transfer Protocol), SFTP (Secure File Transfer Protocol) or through a SAMBA server.

For access via FTP, the following functions can be used:

bool ftpopen (int handle, string host, int port)

Open an FTP connection for subsequent file transfer

bool ftpauthenticate (int handle, string name, string pw)

Perform a log-in on an FTP server

bool ftpconnected (int handle)

Check connection status to an FTP server

bool ftpclose (int handle)

Close connection to an FTP server

bool ftpget (int handle, string source, string dest)

Copy a file from an FTP server

bool ftpput (int handle, string source, string dest)

Copy a file to an FTP server

bool ftpremove (int handle, string fname)

Remove a file from an FTP server

bool ftplist (int handle, string path)

Open a directory on an FTP server

bool ftplistread (int handle, string *name)

Read a file name from an FTP server

bool ftprename (int handle, string path, string oldname, string newname)

Rename a file on the FTP server.

 

For access via the secured protocol SFTP, the following functions can be used:

bool sftpopen (int handle, string hostname, int port, string user, string pw)

Open an SFTP connection for subsequent file transfer

bool sftpconnected (int handle)

Check connection status to an SFTP server

bool sftpclose (int handle)

Close connection to an SFTP server

bool sftpget (int handle, string source, string dest)

Copy a file from an SFTP server

bool sftpput (int handle, string source, string dest)

Copy a file to an SFTP server

bool sftpremove (int handle, string fname)

Remove a file from an SFTP server

bool sftplist (int handle, string path)

Open a directory on an SFTP server

bool sftplistread (int handle, string *name)

Read a file name from an SFTP server

bool sftprename (int handle, string oldname, string newname)

Rename a file on the SFTP server.

 

For access through a SAMBA server, an SMB connection to the server has to be established first. The files can then be accessed by the standard functions for file access.

string smbconnect (string host, string share, string user, string pw, string *res)

Provide access to external network drive

bool smbconnected (string path)

Check connection status to an external network drive

void smbdisconnect (string host)

Disconnect from an external network drive
