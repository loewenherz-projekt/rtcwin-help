Forward Declarations

    

Functions have to be declared prior to the first usage. In some situations, this is not possible, for instance when the function accesses other functions that can only be defined in a later part of the program. If this is the case, the function can be announced using a so-called forward declaration.

The forward declaration consists of the name of the function, its parameter list and the type of the return value; the forward declaration is concluded by a semicolon.

Type Name ( Parameters );

 

The actual function can be defined in the code below with type and parameters matching the settings of the forward declaration.

 

Example:

// Forward declaration

void StartUp (void);

 

 

/* Configuration menu */

{

      TX_BASIC_SETTINGS              BasicSettings();

      TX_FACTORY_DEFAULTS            LoadFactoryDefaults();

    }

    StartUp();  // Enable modifications

  }  

}
