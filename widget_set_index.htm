widget_set_index

    

RTC Function

Activates an entry or a field of a list or table object.

Declaration:

void widget_set_index (int id, int row, int col)

Group:

Touch screen programming

Parameters:

id
row
col

Logical reference number of the object
Entry or table row that is to be activated
Table column that is to be activated

Return value:

-

 

The instruction widget_set_index selects the list or table object specified by the parameter id and activates it. If the object is a table object of type WIDGET_TYPE_TABLE, the field is addressed by the two parameters row and col. If the object is a list object of type WIDGET_TYPE_LIST or WIDGET_TYPE_COMBOBOX, the entry is addressed by the parameter row and the parameter col is ignored.

 

Beispiel:

// Activate field 2 (852) in object list 

beginlayout (LAYOUT_TYPE_HBOX, 0, 2);

  createwidget (ID_CODE_PAGE_PROMPT, WIDGET_TYPE_TEXT_OUTPUT);

    widget_text (ID_CODE_PAGE_PROMPT, "Codepage", 0);   

                 

  createwidget (ID_CODE_PAGE_SELECTION, WIDGET_TYPE_COMBOBOX);

    widget_text      (ID_CODE_PAGE_SELECTION, "None", 0);

    widget_text      (ID_CODE_PAGE_SELECTION, "850",  1);

    widget_text      (ID_CODE_PAGE_SELECTION, "852",  2);

    widget_text      (ID_CODE_PAGE_SELECTION, "866",  3);

    widget_set_index (ID_CODE_PAGE_SELECTION, 2, 0);

endlayout ();
