updateio

    

RTC Function

Updates the digital I/Os.

Declaration:

void updateio (void)

Group:

External inputs/outputs

Parameters:

-

 

Return value:

-

 

The function updateio synchronizes the digital I/Os of the software application and the physical I/Os. Thus, the state of the I/Os is available without delay for other background tasks.

 

Example:

#define IO_MAX 15

/* Set outputs... */

pout (Outputs, IO_MAX, 0);

updateio ();

 

Note:

There is usually no need to use the function updateio as the interpreter automatically updates the digital I/Os every 10 milliseconds.
