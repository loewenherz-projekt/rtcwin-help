widget_attribute

    

RTC Function

Sets the properties of an object.

Declaration:

void widget_attribute (int id, int opt, any val)

Group:

Touch screen programming

Parameters:

id
opt
val

Logical reference number of the object
Property to be modified
Value the property is set to

Return value:

-

 

The instruction widget_attribute sets specific properties of the object specified by the parameter id. The parameter opt determines the property to be modified with the value specified by the parameter val as follows:

Property

Alias

Description

0

WIDGET_ATTRIBUTE_NONE

No property

1

WIDGET_ATTRIBUTE_ROWS

Sets the number of rows in a table or list object of type WIDGET_TYPE_TABLE or WIDGET_TYPE_LIST to the value specified by val.

2

WIDGET_ATTRIBUTE_COLUMNS

Sets the number of columns in a table object of type WIDGET_TYPE_TABLE to the value specified by val.

3

WIDGET_ATTRIBUTE_VERTICAL_HEADER

Sets the row titles (descriptions) in a table object of type WIDGET_TYPE_TABLE to the value specified by val. The individual title texts contained in val are separated by semicolons.

4

WIDGET_ATTRIBUTE_HORIZONTAL_HEADER

Sets the column titles in a table or list object of type WIDGET_TYPE_TABLE or WIDGET_TYPE_LIST to the value specified by val. The individual title texts contained in val are separated by semicolons.

If a property is set for an object that does not support this property, the function widget_attribute has no effect.

 

Example:

// Prepare article table...

void CreateMyTable (void)

{

  begindraw ();

  createpage (PAGE_MY_TABLE);

    beginlayout (LAYOUT_TYPE_VBOX, 0, 2); 

      createwidget(ID_TABLE_GUI, WIDGET_TYPE_TABLE);

        widget_fontsize (ID_TABLE_GUI, 20);

        widget_attribute (ID_TABLE_GUI, WIDGET_ATTRIBUTE_COLUMNS, 3);

        widget_attribute (ID_TABLE_GUI, WIDGET_ATTRIBUTE_HORIZONTAL_HEADER, "Article-Nr.;Description;Target");

        widget_attribute (ID_TABLE_GUI, WIDGET_ATTRIBUTE_ROWS, 5);

        // Current consec is displayed, if ID5 isn´t predefined, 

        widget_attribute (ID_TABLE_GUI, WIDGET_ATTRIBUTE_VERTICAL_HEADER, "ID1;ID2;ID3;ID4"); 

 

        // Predefinded default descriptions 

        widget_text (ID_TABLE_GUI, "Article 1",   0, 0); // Row=1, Column=0 

        widget_text (ID_TABLE_GUI, "Descr.1",     0, 1); 

        widget_text (ID_TABLE_GUI, "Target 1",    0, 2); 

        widget_text (ID_TABLE_GUI, "Article 2",   1, 0); 

        widget_text (ID_TABLE_GUI, "Descr.2",     1, 1); 

        widget_text (ID_TABLE_GUI, "Target 2",    1, 2); 

        widget_text (ID_TABLE_GUI, "Article 3",   2, 0); 

        widget_text (ID_TABLE_GUI, "Descr.3",     2, 1); 

        widget_text (ID_TABLE_GUI, "Target 3",    2, 2); 

        widget_text (ID_TABLE_GUI, "Article 4",   3, 0); 

        widget_text (ID_TABLE_GUI, "Descr.4",     3, 1); 

        widget_text (ID_TABLE_GUI, "Target 4",    3, 2); // Row=3, Column=2 

    endlayout ();  

  enddraw ();

}

 

 

Note:

Applicable preprocessor constants (e.g. WIDGET_ATTRIBUTE_ROWS) are available in the file "qtmonitor.h."
