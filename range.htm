range

    

RTC Variable

Current weighing range of a multiple-range or multi-interval scale.

Declaration:

loc int   range

Group:

Scale functions

 

The variable range is a predefined variable of type integer showing the number of the weighing range the scale was in when the functions getweight or getapprovedweight were executed last. If the read scale is neither a multiple-range nor a multi-interval scale, the content of range is 0.

The variable range is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.
