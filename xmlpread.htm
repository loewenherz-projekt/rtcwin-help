xmlpread

    

RTC Function

Reads an XML document from a port into an XML instance.

Declaration:

bool xmlpread (int id, int port, float t)

Group:

XML interface

Parameters:

id
p
t

Logical reference number of the XML instance
Logical port number
Timeout value

Return value:

Function result

 

The function xmlpread reads an XML document from the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen). The function ends when either the first node is closed or the maximum receive timet has elapsed (timeout). If the XML document can be received successfully, it is made available for subsequent evaluation in the XML instance specified by the parameter id. In that case, xmlpread returns true as function result; otherwise the result is false.

 

Example:

/* If XML data was found, read it... */

if (preceived(port))

{

  xmlpread (XML_HANDLE, XML_PORT, 5);
  /* Process XML data...

  /* ... */

}

 

Note:

The XML instance is automatically cleared before new content is written to it. Calling xmlclear is not necessary.
