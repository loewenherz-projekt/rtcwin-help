flashset

    

RTC Function

Set a parameter in flash memory.

Declaration:

bool flashset (string key, string value)

Group:

System

Parameters:

key
value

Parameter name
Parameter value

Return value:

Function result

 

The function flashset searches the working area of the application flash memory for the parameter specified by key. Beforehand the instruction flashload has to be executed to load the working area with a copy of the application parameters stored in flash memory. If the specified parameter can be found in the working area, it is set with the provided parameter value. If the specified parameter does not exist in the flash memory, it is created with the provided value.  If the parameter can be set or created successfully, flashset returns true as function result; otherwise the result is false. To definitively save the parameter, the instruction flashsave has to be executed to copy the content of the working area back into flash memory.

 

Example:

/* Set a parameter in flash memory */
bool SetFlashParameter (void)

{

  if (! flashset ("MaxFillings", trim (MaxFillings)))

    return (false);

  /* ... */

}
