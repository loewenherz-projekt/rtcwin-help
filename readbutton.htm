readbutton

    

RTC Function

Waits until one of the function buttons of an outputbox is pressed.

Declaration:

char readbutton (void)

Group:

Touch screen programming

Parameters:

-

 

Return value:

Key code

 

The function readbutton checks if one of the six function buttons of an outputbox is pressed. If that is the case, readbutton returns the corresponding key code of the function buttons F1–F6 without delay just as for the standard non-touch devices. Otherwise, program execution waits until the operator presses a button. The function buttonpressed can be used to check the availability of a button prior to the execution of readbutton.

 

Example:

void FillingMode (void)

{

char ch = 0;

int result;

 

  outputbox ("Filling Mode;Menu;;;;;Ok"); // "Menu" =F1, "Ok" =F6

  // From here standard RTC screen commands

  font (FT_RTC_BOLD); fontsize (21);

  showtext (10, 10, "Order:");

 

  // If function key pressed: read key and process...

  for (;;)

  {

    result = getweight (indicatedscale());  // ... read scale

    if (buttonpressed ())

    {

      ch = readbutton ();

      switch (ch)

      {

        case fct1: // Return to menu

                   return;

        case fct6: // ...

                   break;

      }

    }

    /* ... */

    delay (0.1);

  }

}
