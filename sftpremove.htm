sftpremove

    

RTC Function

Deletes a file from the SFTP server.

Declaration:

bool sftpremove (int sfno, string fname)

Group:

File handling

Parameters:

sfno
fname

Logical connection number
Name of the file to be deleted

Return value:

Function result

 

The function sftpremove deletes the file specified by the parameter fname from the SFTP server. For this purpose, an SFTP connection has to be established beforehand via sftpopen. The parameter sfno is the logical connection number assigned with the execution of the function sftpopen.

The parameter fname contains the full file name and path (if applicable) of the file that is to be deleted. If the file can be deleted successfully, sftpremove returns true as function result; otherwise the result is false.

 

Example:

/* Get order file from FTP server */

 

#define SFTP_PORT 22

 

bool GetOrders (void)

{

bool result = false;

 

  if (sftpopen (0, ServerIP, SFTP_PORT))  // Open FTP connection...

  {

    // get new orders and copy to 'shared'...

    if (ftpget (0, "Production/Scale_1/Orders.csv", "shared/Orders.csv"))  

    {

      result = true;

      // delete order file from server

      if (! ftpremove (0, "Production/Scale_1/Orders.csv"))

        ShowErrorMsg ("SFTP access denied!");

    }

    else

      ShowErrorMsg ("No new orders available!");

  }

  else

    ShowErrorMsg ("SFTP access denied!");

 

  if (!ftpclose (0)) // Close FTP connection...

    ShowErrorMsg ("SFTP access denied!");      

  }

  else

    ShowErrorMsg ("SFTP server not available!");

  return (result);

}

 

Note:

After file deletion is completed, the SFTP connection should be closed using sftpclose.
