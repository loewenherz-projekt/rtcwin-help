trylock

    

RTC Function

Checks the status of a semaphore variable and locks it when possible.

Declaration:

bool trylock (semaphore sp)

Group:

Multitasking

Parameters:

sp

Name of semaphore variable

Return value:

Status

 

The instruction trylock checks the status of the given semaphore variable sp. If the semaphore variable is free, it is immediately locked and true is returned as function result. If the semaphore variable is already locked, the result is false.

 

Example:

semaphore Sema1;

 

void UpdateWeightStatus (int scl_no)

{

  if (trylock (Sema1)) // Access to shared resource

  {

    if (getweight (scl_no) == 0)

    {

      GrossWeight[scl_no-1] = gross;

      TareWeight[scl_no-1] = tare;

      NetWeight[scl_no-1] = net;

      IsSettled[scl_no-1] = issettled;

    }

    unlock (Sema1);   // Release semaphore variable

  }

}

 

void Scale1_Task (void)

{

  for (;;)

  {

    UpdateWeightStatus (1);

    /* ... */

  }

}
