indicatescale

    

RTC Function

Switches the scale display to the specified scale.

Declaration:

int indicatescale (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Function result

 

The function indicatescale brings the scale specified by the parameter sno into the scale display. If the display can switch to the specified scale successfully, indicatescale returns true as function result; otherwise the result is false.

 

Example:

bool SelectScale (int scale_no)

{

timer t;

  if (indicatescale (scale_no) == 0)      // Switches the scale to the specified scale

  {

    starttimer (t, 6.0);

    do

    {

      if (timeout (t))

        return (false);                   // Timeout: Scale switching failed

      delay (0.1);

    }

    while (indicatedscale () != scale_no);

    return (true);                        // Scale switching successfully

  }

  else

    return (false);

}

 

Interpreter version:

In interpreter versions before version 6, the function result of indicatescale is void, i.e. the function does not return any result. The display is switched to the first scale instead when attempting to call a nonexistent scale.
