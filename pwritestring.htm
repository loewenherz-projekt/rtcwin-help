pwritestring

    

RTC Function

Outputs a string on a communication channel.

Declaration:

int pwritestring (int port, string s, int len)

Group:

Communication

Parameters:

port
s
len

Logical port number
String to be sent
Number of characters to be send

Return value:

Error code

 

The function pwritestring transmits the provided string s on the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen). If the string can be transmitted successfully, pwritestring returns 0 as function result; otherwise the corresponding error code is returned instead.

The parameter len specifies the number of characters to transmit. If the string is longer than len, only the corresponding part of the string is transmitted. If the string is shorter than len, it is filled up with a corresponding number of zero characters #0.

 

Example:

if (pwritestring (port, SendBuffer, length (SendBuffer)) != 0)

  ShowErrorMsg ("Error data transmission");

 

Note:

The function pwritestring is not convenient for sending binary strings. For this purpose, the function pwritechar should be used.
