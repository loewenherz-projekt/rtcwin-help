#warning

    

RTC Function

Outputs a warning before compilation.

Declaration:

#warning  message

Group:

Preprocessor

Parameters:

message

Warning to be shown during compilation

 

With the preprocessor statement #warning a message can be shown as warning or hint in the status window of RTC WIN.

 

Example:

void main (void)

{

#warning "Output on network printer not yet tested..."

 

  Init();

  for (;;)

    selectbox (TX_SELECT_FUNCTION)

    {

      TX_WEIGHING_MODE   WeighingMode();

      TX_SETTINGS        Settings();

    }

}
