dw

    

RTC Variable

Shared memory area for fieldbus communication.

Declaration:

int   dw [128]

Group:

Fieldbus

 

For the exchange of data via fieldbus, a shared memory area is used. This area is designed as array of data words. The assignment of the individual data words is part of the application. The program can access this memory for read and write as usual.

Exchange of the contents of data words with the host system is automatically carried out by the operating system using the fieldbus communication protocol set up in Service Mode.

The data words dw[0] to dw[63] are used as RTC output words and correspond to the input words IW0 to IW63 of the host system. The data words dw[64] to dw[127] are input words and are read only. They correspond to the output data words OW0 to OW63 of the host system.
