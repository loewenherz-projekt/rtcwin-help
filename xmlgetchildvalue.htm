xmlgetchildvalue

    

RTC Function

Reads the value of the current entry of a node in an XML instance.

Declaration:

string xmlgetchildvalue (int id, string node)

Group:

XML interface

Parameters:

id
node

Logical reference number of the XML instance
Path and name of the node

Return value:

Value of the entry; text

 

The function xmlgetchildvalue reads the value of the currently selected entry of the given node from an XML instance specified by the parameter id. The node is specified by the parameter node which contains the full path and name of the node.

 

Example:

if (xmlnodeexists (handle, "response/error_info"))

{

  infodata = xmlgetchildvalue(handle, "response/error_info/error_text")

  infobox(infodata);

  readkey();

}
