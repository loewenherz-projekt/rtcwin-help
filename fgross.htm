fgross, ftare, fnet

    

RTC Variable

Weight variables formatted for approval.

Declaration:

loc string   fgross[x]
loc string   ftare[x]
loc string   fnet[x]

Group:

Scale functions

 

The variables fgross, fnet and ftare are predefined variables of type string. They are used for the approved representation of weights. They are updated by the functions getweight and getapprovedweight and have a fix length that is calculated as follows:

 
+
+
+
+
+

Number of digits to represent the maximum capacity of the scale
Number of decimal digits plus 1 for the decimal point, if applicable
One digit for the sign (minus or space)
Two characters for the weight unit
One space character between value and weight unit
Three characters after the weight unit (space plus 'N', 'T', 'PT or space only)

For a weighing range of 1500 kg and a resolution of 0.5 kg, the length of the formatted weight variables results in 12 digits. If the representation of weights with approval signs is selected in Service Mode (Approval Signs = Y), the length of the variable increases by another digit.

The variables fgross, fnet and ftare are task local variables, i.e. they may be used in multiple tasks, however assign different physical memory areas for each task.

For unformatted numeric representation of the weights, the variables gross, net and tare are available.

 

Example:

void UpdateWeightDisplay(void)

{

string buffer[40];

 

  buffer = concat ("Brutto:",space (7), fgross);

  showtext (20, 30, buffer);

  buffer = concat ("Netto: ", space (7), fnet);

  showtext (20, 50, buffer);

  buffer = concat ("Tara:  " ,space (7), ftare);

  showtext (20, 70, buffer);

}
