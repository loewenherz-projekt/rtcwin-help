savescreen

    

RTC Function

Saves the complete content of the screen.

Declaration:

void savescreen (int sno)

Group:

Display and keyboard

Parameters:

sno

Backup number

Return value:

-

 

The instruction savescreen saves the complete content of the current application screen against the backup number specified by the parameter sno. The backup number can freely be chosen from the range 0 to 7. The screen content can be restored later on using the function restorescreen by specifying the backup number.

 

Example:

...

if (find (Product) != 0)

{

  savescreen (1);

  infobox ("Product not found!");

  ch = waitkey (3);

  restorescreen (1);

}
