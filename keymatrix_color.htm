keymatrix_color

    

RTC Function

Specifies the color of the button in a keymatrix.

Declaration:

void keymatrix_color (int id, int key, int color)

Group:

Touch screen programming

Parameters:

id
key
color

Logical reference number of the window
Reference number of the button
Color as RGB code

Return value:

-

 

The instruction keymatrix_color defines the background color of the button specified by the parameter key in the keymatrix referenced by id. Counting of key starts from 0 for the leftmost button in the top row and continues row by row from left to right. The color is defined by a hex number as follows:

 

Value from 0x00 to 0xff for blue

Value from 0x00 to 0xff for green

Value from 0x00 to 0xff for red

 

Example:

void CreateSettingsMenu (void)

{

  keymatrix_generate (PAGE_MAIN_SETTINGS, 4, 1);  // 4 buttons one below the other

    keymatrix_color (PAGE_MAIN_SETTINGS, 0, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 0, "Produkte");

    keymatrix_color (PAGE_MAIN_SETTINGS, 1, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 1, "Taraspeicher");

    keymatrix_color (PAGE_MAIN_SETTINGS, 2, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 2, "Einstellungen");

    keymatrix_color (PAGE_MAIN_SETTINGS, 2, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 2, "Datensicherung");

}

void Settings (void)

{

char ch = 0; 

int key = 0;

 

  ch = keymatrix_select (PAGE_MAIN_SETTINGS, "Data input", key);

  if (ch == ok)

    switch (key)

    {

      case 0:  EditProducts();

               break;

      case 1:  EditTareValues();

               break;

      case 2:  Settings();

               break;

      case 3:  DataBackup();

               break;

    }

}
