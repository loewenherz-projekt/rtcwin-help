widget_format

    

RTC Function

Sets the format of an input field.

Declaration:

void widget_format (int id, int format, int len, int decs)

Group:

Touch screen programming

Parameters:

id
format
len
decs

Logical reference number to access the object
Input format
Length of the input field
Number of decimals

Return value:

-

 

The function widget_format defines the format of the object of type WIDGET_TYPE_TEXT_INPUT specified by the parameter id. The maximum length of the entry (maximum number of characters) is defined using the parameter len. When type format INPUT_FORMAT_DECIMAL is used, the parameter decs is used to define the number of decimals; for all other formats this parameter is ignored.

The parameter format specifies the input field type as follows:

Type

Alias

Description

0

INPUT_FORMAT_STRING

Normal alphanumeric text input from left to right

1

INPUT_FORMAT_PASSWORT

Hidden text input. Characters are shown as asterisks.

2

INPUT_FORMAT_DATE

Date input as specified in Service Mode group General

3

INPUT_FORMAT_TIME

Time input as specified in Service Mode group General

4

INPUT_FORMAT_INTEGER

Numeric input without decimals (integer)

5

INPUT_FORMAT_BOOL

Boolean input (Yes / No)

6

INPUT_FORMAT_FLOAT

Numeric input with optional decimals (floating-point number)

7

INPUT_FORMAT_DECIMAL

Numeric input with fix number of decimals as specified by the parameter decs

8

INPUT_FORMAT_NUMERIC_STRING

Alphanumeric text input from left to right, but numeric keypad is shown as default.

 

Example:

// Create input field...
void CreateInputWidget (int id, string label_text)

{
  createwidget (id, WIDGET_TYPE_TEXT_INPUT);

    widget_fontsize   (id, 24);  

    widget_fontface   (id, FT_RTC_BOLD)

    widget_dimensions (id, 50, 50); 

    widget_label      (id, label_text); // Operator prompt

    widget_format     (id, INPUT_FORMAT_DECIMAL, 6, 2);

}

 

Note:

Applicable preprocessor constants (e.g. INPUT_FORMAT_STRING) are available in the file "qtmonitor.h."
