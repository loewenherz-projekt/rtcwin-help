out

    

RTC Function

Returns the status of a digital output.

Declaration:

bool out (int bitno)

Group:

External inputs/outputs

Parameters:

bitno

Number of digital output

Return value:

Status of digital output

 

The function out checks the status of the digital output specified by the parameter bitno. Counting starts from 0. If the output is set, out returns true as function result; otherwise the result is false.

 

Example:

#define OUT0_WATCH_DOG 0

timer WatchdogTimer;

 

/* Handle watchdog output */

void CheckWatchDog (void)

{

  if (timeout (WatchdogTimer))

  {

    bout (OUT0_WATCH_DOG,! out (OUT0_WATCH_DOG));

    starttimer (WatchdogTimer, 1.0);

  }

}

 

Note:

If the output specified by the parameter bitno is not available, out returns false as function result.
