Totalizing Scales

    

RTC provides a set of instructions especially tailored for totalizing scales, where the total weight after a series of tips is subject of weights and measures:

Function

Description

decimal totalbulkweight %10.dp

Variable for the totalized bulk weight over all tips.

int startbulkweighing (int sno)

Resets the total bulk weight and starts totalizing.

int getfilledbulkweight (int sno)

Requests the approved weight of the filled weigh hopper and updates all weight and status variables.

int getdischargedbulkweight (int sno)

Requests the approved weight of the discharged weigh hopper and updates all weight and status variables. The difference between filled and discharged weight of the hopper is added to the total bulk weight.

int storetotalbulkweight (int sno)

Stores the total bulk weight in the approved weight storage and updates the ID (alibi number).
