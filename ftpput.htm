ftpput

    

RTC Function

Drops a file onto the FTP server.

Declaration:

bool ftpput (int fno, string sname, string dname)

Group:

File handling

Parameters:

fno
sname
dname

Logical connection number
Path and name of the source file
Path and name of the target file

Return value:

Function result

 

The function ftpput drops the file specified by the parameter sname on an FTP server. For this purpose, a connection has to be established beforehand via ftpopen and – if applicable – a log-in performed using ftpauthenticate. The parameter fno is the logical connection number assigned with the execution of the function ftpopen.

The parameter sname contains the full name and path of the source file as follows:

internal/name.ext

The file name with the extension ext is contained in the battery-backed memory of the weighing terminal and cannot be accessed externally.

shared/name.ext

The file name with the extension ext is contained in the directory "shared". This folder can be accessed externally via FTP access (login: "shared," password as set in Service Mode).

usbstick/.../name.ext

The file name with the extension ext is contained on a USB memory stick. Further subdirectories may be specified as applicable.

Files on the USB memory stick can be accessed externally via FTP (login: "usbshared," password as set in Service Mode).

sharename/.../name.ext

The file name with the extension ext is contained on a network drive. sharename is the symbolic path name as it was returned by the instruction smbconnect. Further subdirectories may be specified as applicable.

If the file name does not specify a directory, the folder 'internal' is used as default.

The parameter dname contains the full name and path (if applicable) of the target file on the FTP server.

If the file can be saved successfully on the FTP server, ftpput returns true as function result; otherwise the result is false.

 

Note:

After file copy is completed, the FTP connection should be closed using ftpclose.
