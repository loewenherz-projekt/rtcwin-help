fcopy

    

RTC Function

Copies the content of a file to another.

Declaration:

bool fcopy (string f_source, string f_dest)

Group:

File handling

Parameters:

f_source
f_dest

Name of source file
Name of target file

Return value:

Function result

 

The function fcopy copies the content of the file specified by the parameter f_source into the target file specified by the parameter f_dest. The parameters f_source and f_dest contain the full file names and paths as follows:

internal/name.ext

The file name with the extension ext is contained in the battery-backed memory of the weighing terminal and cannot be accessed externally.

shared/name.ext

The file name with the extension ext is contained in the directory "shared". This folder can be accessed externally via FTP access (login: "shared," password as set in Service Mode).

usbstick/.../name.ext

The file name with the extension ext is contained on a USB memory stick. Further subdirectories may be specified as applicable.

Files on the USB memory stick can be accessed externally via FTP (login: "usbshared," password as set in Service Mode).

sharename/.../name.ext

The file name with the extension ext is contained on a network drive. sharename is the symbolic path name as it was returned by the instruction smbconnect. Further subdirectories may be specified as applicable.

If the file name does not specify a directory, the folder 'internal' is used as default.

If the target file does not exist, it is created by fcopy. If the file can be copied successfully, fcopy returns true as function result.  If the specified directory does not exist or the file cannot be copied, fcopy returns false, accordingly.

 

Example:

/* Copy log file to USB stick */

bool CopyDataToLogFile (string source_name, string dest_name)

{

string source_file[1023] = concat ("shared/", source_name, ".csv");

string destination_file[1023] = concat ("usbstick/", dest_name, ".csv");

 

  infobox ("Kopiere Log-Datei...");

  if (fexists (destination_file) == 1)

    if (fremove (destination_file) != 0)

    {

      ShowErrorMsg ("Zugriff verweigert");

      return (false);

    }

 

  if (! fcopy (source_file, destination_file))

  {

    ShowErrorMsg ("Datei konnte nicht kopiert werden!");

    return (false);

  }

  else

    return (true);

}
