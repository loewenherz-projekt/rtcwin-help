bool

    

RTC Declaration Word

Declaration of a Boolean variable.

Declaration:

bool name

Group:

Declaration words

Parameters:

name

Variable or function name

 

The instruction bool declares a variable with given name or the return value of a function with name as "Boolean." The type bool can only contain one of the two values false or true. During declaration, the variable name may be assigned an initial value.

Where a Boolean variable or return value is assigned to an integer variable, false is converted to 0 and true to 1. Contrariwise, the integer value 0 is interpreted as false and all other values as true.

 

Example:

bool WithPrinter;

bool IsEmpty = false;
