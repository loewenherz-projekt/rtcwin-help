background_task

    

RTC Function

Defines a program part that is executed in the background.

Declaration:

void background_task (void)

Group:

System

Parameters:

-

 

Return value:

-

 

background_task is the name of a function yet to be declared in the application program that is cyclically called up by the operating system and executed in the background (background task). This function is meant to contain all functions that are to be executed independently from the current status of the application, e.g. constant check of a timer in the background. The mere fact that this function is given the name background_task causes this part of the program to be run in the background.

 

Example:

// The following #pragma statement prevents the automatic Background_Task start after system startup

#pragma (backgroundtask_autostart, "0") 

 

#define OUT0_WATCH_DOG 0

timer WatchdogTimer;

 

void background_task (void)

{

  /* Handle watchdog output */

  if (timeout (WatchdogTimer))

  {

    bout (OUT0_WATCH_DOG, ! out (OUT0_WATCH_DOG));

    starttimer (WatchdogTimer, 1.0);

  }

}

 

void main (void)

{

int result;

  GeneralInit ();

  result = starttask (background_task); // Background_task Start after initialization

  /* ... */

}

 

Notes:

The background task must not contain any loops or timer functions that may delay the execution of the background task.
