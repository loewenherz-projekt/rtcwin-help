The Programming Language RTC     String Functions

Formatting Strings

    

For the formatting of string variables and expressions, a number of handy functions are contained in RTC.

ucase converts all characters of a string into uppercase. Contrariwise, the function lcase can be used where all characters have to be converted into lowercase.

For truncation of leading and trailing spaces in a string variable, the function trim can be used. Where only leading or trailing spaces need to be cut off, correspondingly, the functions ltrim or rtrim are available.

But also for the opposite operations, there are adequate functions: left, right and mid add spaces to a string to left-, right- or center-adjust the contents to a given width. If the leading spaces are not to be truncated but converted into another character, e.g. into leading zeroes, the function head can be used.
