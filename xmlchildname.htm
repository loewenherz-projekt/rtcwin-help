xmlchildname

    

RTC Function

Determines the name of an entry in an XML instance.

Declaration:

string xmlchildname (int id, string node)

Group:

XML interface

Parameters:

id
node

Logical reference number of the XML instance
Path and name of the node

Return value:

Function result

 

The function xmlchildname is used to determine the name of the entry at the current read/write position of the node specified by node in the XML instance specified by the parameter id. The current position has to be set beforehand, e.g. by using xmlfirstchild.
