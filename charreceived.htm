charreceived

    

RTC Function

Checks if any received characters are available.

Declaration:

bool charreceived (int port)

Group:

Communication

Parameters:

port

Logical port number

Return value:

Status

 

The function charreceived checks the receive buffer of the communication channel specified by the logical port number port if it contains any characters not yet read. The port has to be opened first (e.g. using popen). If one or more characters are available, charreceived returns true as function result; otherwise the result is false.

The function charreceived only checks the availability of a character without reading it. If a character was received, subsequent calls of charreceived always return true until all characters have been read (e.g. using preadchar).

 

Example:

#define STX 2

 

bool WaitForStartChar (int port)

{

char ch;

  for (;;) /* Wait for start char... */

  {

    if (charreceived (port))

    {

      ch = readchar (port);    /* ... read it... */

      if (ch == STX)

        return (true);         /* Start char reiceived... */

      else

        return (false);

    }

    delay (0.05);

  }

}
