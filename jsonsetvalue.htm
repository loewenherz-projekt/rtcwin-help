jsonsetvalue

    

RTC Function

Sets the value of an object in a JSON statement list.

Declaration:

bool jsonsetvalue (int id, string object, string value)

Group:

JSON interface

Parameters:

id
object
value

Logical reference number of the JSON statement list
Object in JSON statement list
Value of object

Return value:

Function result

 

The function jsonsetvalue sets the value of the given object in the JSON statement list specified by the parameter id to the value specified by the parameter value.

If the value of the object can successfully be loaded into the statement list, jsonsetvalue returns true as function result; otherwise the result is false.

 

Example:

jsonclear(JSON_HANDLE);

jsonsetvalue (JSON_HANDLE, "cmd", "wss-connection-status");

jsontomsg(JSON_HANDLE, EVENT_APPLICATION_WS_SERVER, MSG_WEB_LINK, "");
