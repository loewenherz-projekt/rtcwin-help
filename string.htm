string

    

RTC Declaration Word

Declaration of a formatted text variable.

Declaration:

string name [len]

Group:

Declaration words

Parameters:

name
len

Variable or function name
Length

 

The instruction string declares a variable with given name or the return value of a function name as character string (text). During declaration, also the maximum length of the variable is specified. The type string represents a character string with dynamic length that is assigned a constant memory section of 1–2047 bytes. The actual length of the string can be determined using the function length. During declaration, the variable name may be assigned an initial value. At a length of 1, the type string is compatible to the data type char.

If a variable of type string is assigned a text, the complete character string must be enclosed in double quotes. Inside a string, the following escape sequences are allowed:
