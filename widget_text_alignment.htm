widget_text_alignment

    

RTC Function

Sets the text alignment of an object.

Declaration:

void widget_text_alignment (int id, int al)

Group:

Touch screen programming

Parameters:

id
al

Logical reference number of the object
Text alignment

Return value:

-

 

The instruction widget_text_alignment sets the text alignment policy the object specified by the parameter id. The parameter al defines the alignment as follows:

 

Type

Alias

Description

1

WIDGET_TEXT_ALIGN_LEFT

Text aligned left-justified

2

WIDGET_TEXT_ALIGN_RIGHT

Text aligned right-justified

4

WIDGET_TEXT_ALIGN_HCENTER

Text horizontally centered

8

WIDGET_TEXT_ALIGN_JUSTIFY

Text justified to full object width

20

WIDGET_TEXT_ALIGN_TOP

Text aligned to the top of the object

40

WIDGET_TEXT_ALIGN_BOTTOM

Text aligned to the bottom of the object

80

WIDGET_TEXT_ALIGN_VCENTER

Text vertically centered

 

Example:

void CreateInfoBox (void)         

{

  begindraw ();

  createwindow (SOFTWARE_VERSION_WINDOW, WINDOW_TYPE_POPUP, "");

  windowstyle  (SOFTWARE_VERSION_WINDOW, ST_BOX_STYLE);

    beginlayout (LAYOUT_TYPE_VBOX, 0);

      createwidget (GUI_OUTPUT_STATUS, WIDGET_TYPE_TEXT_OUTPUT);

        widget_size_policy    (GUI_OUTPUT_STATUS,  SIZE_POLICY_MINIMUM, SIZE_POLICY_MINIMUM);

        widget_style          (GUI_OUTPUT_STATUS,  ST_BOXTEXT_STYLE);

        widget_fontsize       (GUI_OUTPUT_STATUS,  30);                                

        widget_foreground     (GUI_OUTPUT_STATUS,  "white");  // white or #FFFFF

        widget_text_alignment (GUI_OUTPUT_STATUS,  (WIDGET_TEXT_ALIGN_HCENTER | WIDGET_TEXT_ALIGN_VCENTER));

    endlayout();

  enddraw ();

}

 

Notes:

Options can be combined as long as they do not conflict. Objects are automatically assigned to the layout in which they are created.

Applicable preprocessor constants (e.g. WIDGET_TEXT_ALIGN_LEFT) are available in the file "qtmonitor.h."
