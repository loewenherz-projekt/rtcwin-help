requestinput

    

RTC Function

Enables data input by the operator.

Declaration:

string requestinput (int kmode, string prompt, string preset, int format, int len, int decs)

Group:

Touch screen programming

Parameters:

kmode
prompt
preset
format
len
decs

Keyboard mode
Operator prompt
Preset value for the input
Input format
Length of the input field
Number of decimals

Return value:

Text entered by the operator

 

The instruction requestinput opens the window for data input by the operator. The input type is specified by the parameter format as follows:

Type

Alias

Description

0

INPUT_FORMAT_STRING

Normal text input

1

INPUT_FORMAT_PASSWORT

Hidden password entry showing asterisks

2

INPUT_FORMAT_DATE

Date input

3

INPUT_FORMAT_TIME

Time input

4

INPUT_FORMAT_INTEGER

Numeric input without decimals

5

INPUT_FORMAT_BOOL

Yes / No input

6

INPUT_FORMAT_FLOAT

Numeric input with optional decimals

7

INPUT_FORMAT_DECIMAL

Numeric input with fixed decimals

8

INPUT_FORMAT_NUMERIC_STRING

Numeric input as string

The parameter len defines the maximum length of the input (maximum number of characters). When the input type INPUT_FORMAT_DECIMAL is used, the parameter decs is used to define the number of decimals; for all other formats this parameter is ignored.

The parameter kmode specifies which keypad is used for the input:

Type

Alias

Description

0

KEYBOARD_MODE_ALPHANUMERIC

Alphanumeric keyboard

1

KEYBOARD_MODE_NUMERIC

Numeric keyboard

The parameter preset can be used to preset the input field with an initial value. The operator can accept this value with the enter key or press the clear key and enter a new value.

The parameter prompt can be used to put a fix text in front of the input field.

The instruction requestinput returns a string containing the operator input as function result. If data entry was aborted by the operator, the function result is an empty string.

 

Example:

BatchNo = requestinput (KEYBOARD_MODE_NUMERIC, "Order No.", "",
                        INPUT_FORMAT_NUMERIC, 13, 0);

 

Note:

Applicable preprocessor constants (e.g. INPUT_FORMAT_STRING) are available in the file "qtmonitor.h."
