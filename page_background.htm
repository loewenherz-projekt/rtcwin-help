page_background

    

RTC Function

Specifies the background color of a page.

Declaration:

void page_background (int id, int color)

Group:

Touch screen programming

Parameters:

id
color

Logical reference number of the page
Color code

Return value:

-

 

The instruction page_background defines the background color of the page specified by the parameter id. The color is specified by the provided parameter color and may be passed as 8-bit or 32-bit value. When provided as 8-bit value, the color is taken from a lookup table. Examples are:

56

black

 

 

88

red

72

dark red

63

green

60

dark green

95

yellow

76

dark yellow

216

blue

136

dark blue

248

magenta

152

dark magenta

223

cyan

140

dark cyan

255

white

 

 

32–55

grayscale from black to white

Alternatively, the background color can be given as 32-bit value. When shown as hexadecimal number, the digits are interpreted as follows:

 

Fixed value 00

Value from 0x00 to 0xff for blue

Value from 0x00 to 0xff for green

Value from 0x00 to 0xff for red

 

Example:

void CreatePage (void) 

{

  begindraw();

  createpage (PAGE_1);

  page_background (PAGE_1, "#d0e0ff");  // SysTec Default

    beginlayout (LAYOUT_TYPE_VBOX, 0, 2);

    /* ... */

    endlayout();

  enddraw();

}
