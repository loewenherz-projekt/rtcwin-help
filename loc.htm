loc

    

RTC Declaration Word

Declaration of a task local variable.

Declaration:

loc  decl  name

Group:

Declaration words

Parameters:

decl       Declaration word name
name     Variable name

 

The instruction loc declares a variable with given name as task local. loc precedes the literal declaration word decl. Task local variables may be used in multiple tasks, however assign different physical memory areas for each task. Thus, they cannot be accessed from another task.

 

Example:

loc  decimal SetPoint%6.dp;
