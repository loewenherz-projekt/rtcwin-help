paintbox

    

RTC Function

Opens a window with a drawing object.

Declaration:

void paintbox (int width, int height)

Group:

Touch screen programming

Parameters:

width
height

Width of the object in pixels
Height of the object in pixels

Return value:

-

 

The instruction paintbox opens a window with the size specified by the two parameters width and height (in pixels). The window is used as display which is compatible to the standard RTC display. Unlike the instruction outputbox, the window does not have function keys and stays visible until paintbox is called once again with the parameters width and height set to 0,0.

 

Example:

void StartUpMessage (void)              

{

  paintbox (185, 100);

    fontsize(22);

    showtext (10, 20, lsub (TX_PROGRAM_VERSION));

    fontsize (16);

    showtext (39, 60, "(c) SysTec GmbH");

    showtext (38, 80, "www.systecnet.com");

    delay (2);

  paintbox (0, 0);

}
