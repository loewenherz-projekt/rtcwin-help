Main program

 

The first line of the main program is the program header, consisting of the type declaration void, the name main and a list of parameters in parentheses, which in this case is empty (void). The instructions within the curved brackets build the program code that normally consists of multiple instructions and functions calls.

Above the function main, further functions with any names may be defined and called up from within the main program and/or frequently used variables may be declared. Thus, the program can be structured in manageable pieces and arranged clearly.

·Variables and functions have to be declared prior to the first usage.
·Already existing names or reserved tokens are not permitted as variable and function names.
·Names of variables and functions may consist of the characters A–Z or a–z, the numbers 0–9 and the underline character. Any special characters are not permitted. The first character of the name must not be a number.
·The length of variable and function names is limited to 64 characters. However, account must be taken of the fact that the name of variables local to a function is composed of the function name and the variable name itself. For task local variables, also the name of the task is included in the full name.
