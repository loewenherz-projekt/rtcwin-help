fread

    

RTC Function

Reads a string from a file.

Declaration:

int fread (int fno, string *s, int len)

Group:

File handling

Parameters:

fno
s
len

Logical file number
String variable for characters read
Number of characters to read

Return value:

Number of read characters

 

The function fread reads a string from the file specified by the logical file number fno. Beforehand the file has to be opened using the instruction fopen and the read position has to be set using the instruction fseek. The function is terminated when either the number of characters specified by the parameter len is read or the end-of-file is reached. If a string can be read successfully, it is written into the provided variable s. In that case, fread returns the actual number of read characters as function result; otherwise the return value is 0. After execution of fread, the internal read/write pointer is moved behind the last read character.

 

Example:

/* Read parameters from USB memory */

bool ReadFile (string directory)

{

string data_buffer[1023] = ""; 

 

  if (fopen (1, concat (directory, "WeighLog.csv")) == 0) // directory = "usbstick/"

  {

    if (ftell (1) > 0)                                    // Read current read/write position

    {

      if (fseek (1, 0, 0) == 0)                           // Move cursor to the beginning of file 

      {

        while (fread (1, data_buffer, 1023) > 0)          // Read data from file

        {

          /* Process parameter  */

        }

        return (true);

      }  

      else

      {

        ShowErrorMsg ("Zugriff verweigert");

        return (false);

      }

    } 

    else

    {

      ShowErrorMsg ("Zugriff verweigert");

      return (false);

    }   

  }

  else  

  {

    ShowErrorMsg ("Fehler Log-Datei");

    return (false);

  }

}

 

Note:

There is no end character defined for the function fread, i.e. zero characters and other control characters can also be read from the file.
