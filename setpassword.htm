setpassword

    

RTC Function

Set password for Service Mode access.

Declaration:

void setpassword (string pw)

Group:

System

Parameters:

pw

Service Mode password

Return value:

-

 

The instruction setpassword installs the provided string pw as password for the access of the Service Mode of the weighing terminal. As password pw, any alphanumeric string is applicable.

 

Example:

setpassword ("wbx-7503"); // Set Service Mode password

 

Notes:

If an empty string is installed as password, the Service Mode is accessible without password entry.

There is no master password to overwrite the service password.
