readstring

    

RTC Function

Reads a string from a communication channel.

Declaration:

int readstring (int port, string *s)

Group:

Communication

Parameters:

port
s

Logical port number
String variable for characters received

Return value:

Error code

 

The function readstring reads a string from the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen). Data transmission is carried out using the communication protocol set up in Service Mode. The function is terminated, if – according to the transmission protocol – a complete data sentence is received. The function stringreceived may be used beforehand to check the availability of a complete data string.

If a string can be received successfully, it is written into the provided variable s. In that case, readstring returns 0 as function result; otherwise the corresponding error code is returned instead.

 

Example:

/* Data transmission */

bool DataTransmission (int port, string weighing_data)

{

string receive_buffer[128] = "";

timer  t;

 

  if (ReadComFifo())

  {

    pclear (port); // clear receive buffer 

    if (writestring (port, weighing_data) == 0)

    {

      starttimer (t, 10.0);

      while (!stringreceived (port)) // Wait for answer...

      {

        if (timeout (t)) 

        {

          ShowErrorMsg ("Error: Timeout");

          return (false);  

        }    

        delay (0.01);

      } 

      if (readstring (port, receive_buffer) == 0)

      {

        if (receive_buffer == "OK")

          return (true);  

      }

      else

        ShowErrorMsg ("Error: Data transmission");

    }

    else

      ShowErrorMsg ("Error: Data transmission");

  }

  return (false);

}
