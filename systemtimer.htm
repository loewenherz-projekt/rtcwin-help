systemtimer

    

RTC Function

Returns the current system time.

Declaration:

int systemtimer (void)

Group:

Timer

Parameters:

-

Return value:

System time in 10 ms ticks

 

The function systemtimer returns the current system time. The return value is the number of 10 ms ticks since last system start.

 

Example:

float FlowRate (float measuring_time)

{

int start_time;

float start_weight;  

 

  getweight (indicatedscale ());

  start_time = systemtimer();

  start_weight = gross;

  delay (measuring_time);

  getweight (indicatedscale());

  return ((gross - start_weight) / (systemtimer() - start_time) * 100.0); // kg/s

}
