#define

    

RTC Function

Defines a preprocessor constant.

Declaration:

#define  name  [wert]

Group:

Preprocessor

Parameters:

name
value

Name of constant
Value of constant

 

With #define a preprocessor constant with the stated name can be defined and assigned an optional value. If the constant is assigned a value, all evidences of name are replaced by this value before the literal compilation of the program takes place. If name is not assigned a value, it can be used in combination with the #ifdef statement e.g. to control the compilation process. In this context, a preprocessor constant can also be undefined using #undef.

If the constant value is a text string, it may be wrapped into the next line using the backslash character.

Example:

#define  TESTMODE   // For test

#define  A_FAST    0

#define  TX_MIN_LOAD_INFO   "Enter the minimum weight above that the\n" \ 

                            "scale is considered loaded."

 

void FillingProcess (decimal target)

{

  while (net < target)  

  {

    bout (O_FAST, true);

    getweight (indicatedscale());

#ifdef TEST_MODE

    showtext (10, 40, adj (net - target, 8, dp));

#endif TEST_MODE

  }

  bout (O_FAST, false);

}

 

#undef TEST_MODE

 

Note:

Predefined preprocessor constants are available to customize application programs for different hardware platforms or interpreter versions.
