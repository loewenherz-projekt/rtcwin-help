xmlclear

    

RTC Function

Deletes the content of an XML instance.

Declaration:

void xmlclear (int id)

Group:

XML interface

Parameters:

id

Logical reference number of the XML instance

Return value:

-

 

The function xmlclear clears the content of an XML instance specified by the parameter id. The logical reference number id has a range from 0 to 7 and addresses one of 8 possible XML instances.

 

Example:

/* create XML file for the settings */

void XML_CreateSettingsFile(void)

{

  xmlclear (XML_HANDLE);

  XML_CreateGroup(XML_HANDLE, "Settings", "Configuration");

  XML_WriteGeneralSettings();

  xmlsavefile (XML_HANDLE, "Settings.xml");

}

 

 

Note:

The functions xmlloadfile, xmlloadstring, xmlfrommsg and xmlpread automatically clear the XML instance before a new content is written. Calling xmlclear is not necessary.
