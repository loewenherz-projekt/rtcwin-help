print

    

RTC Function

Output a print format on a communication channel.

Declaration:

void print (int port, form format, [string layout])

Group:

Communication

Parameters:

port
format
layout

Logical port number
Print format
Name of optional layout file

Return value:

-

 

The function print outputs the provided print format on the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen). The print format is referenced by the name that was assigned during definition using form.

 

Example:

form Header

{

  1 50 "Ticket No." 

  1 66 FTicketNo

  3 50 "Date"

  3 66 date

}

 

void PrintTicket (int port)

{

int i;

 

  gettime();

  TicketNo++;

  showtext (20, 90, "Printing ...");

  print (port, Header);

  for (i = 1; i <= PosNo; i++)

  {

    /* ... print details */

  }

}

 

If the shared directory of the weighing terminal contains a layout file with the same name as the print format and with the file extension 'lab' (e.g. 'Header.lab'), this layout is send to the printer instead of the print format, replacing any placeholders by the corresponding variables. Thus, an external layout for a label printer can be embedded in the program.

The instruction print may also be assigned the name of a layout file as additional parameter to enable printing in different formats.
