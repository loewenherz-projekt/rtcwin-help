starttask

    

RTC Function

Starts a function as background task.

Declaration:

int starttask (procname p)

Group:

Multitasking

Parameters:

p

Name of function without parameters

Return value:

task ID

 

The instruction starttask starts the given parameter-free function as a task. The function is specified by the parameter p. Thereafter, the function p is executed in the background simultaneously with the main task. If the function can be started as a task successfully, starttask returns a value greater or equal 0 as function result; otherwise the function result is -1.

 

Example:

void ZeroCheck_Task (void)

{

int result = 0;

 

  for (;;)

  {

    result = getweight (1);

    if (gross > ZeroWeight)

      bout (IN0_IS_SCALE_EMPTY, false);

    else

      bout (IN0_IS_SCALE_EMPTY, true);

    /* ... */

    delay (0.1);

  }

}

 

void main (void)

{

int result;

  GeneralInit ();

  result = starttask (ZeroCheck_task);

  /* ... */

}

 

Note:

The function meant to run as a task in the background should be designed as an endless loop. Otherwise the function terminates after being executed once and has to be started anew.
