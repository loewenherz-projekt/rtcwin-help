flashremove

    

RTC Function

Removes a parameter from flash memory.

Declaration:

bool flashremove (string key)

Group:

System

Parameters:

key

Parameter name

Return value:

Function result

 

The function flashremove searches the working area of the application flash memory for the parameter specified by key. Beforehand the instruction flashload has to be executed to load the working area with a copy of the application parameters stored in flash memory. If the specified parameter can be found in the working area, it is removed and flashload returns true as function result; otherwise the result is false. To definitively delete the parameter from the application flash memory, the instruction flashsave has to be executed to copy the content of the working area back into flash memory.
