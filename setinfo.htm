setinfo

    

RTC Function

Assigns an additional help text to a variable or database field.

Declaration:

void setinfo (any var, string text)

Group:

Display and keyboard

Parameters:

var
text

Variable or database field
Additional help text

Return value:

-

 

The instruction setinfo assigns the variable var the info text specified by the parameter text as additional help text. If done so, pressing the F5 key when entering a variable during the execution of the functions input or inputbox calls up the additional help text. In that case, the F5 key is labeled 'Info.' If the help text is assigned a database field (dbname.fieldname), the help function is available during the execution of the functions select and edit.

After pressing the F5 key, an embossed box pops up in the center of the application screen that contains the provided text string. The text is shown in black proportional characters on white background and may extend over up to four lines. Line wraps can be specified by the sequence '\n'. Press F5 key again to remove the help box.

 

Example:

nov decimal  MinLoad%6.dp;  // Minimum load

 
setinfo (MinLoad, "Enter the minimum weight above that the\n"+
                  "scale is considered loaded.");
