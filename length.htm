length

    

RTC Function

Determines the length of a string.

Declaration:

int length (string s)

Group:

String functions

Parameters:

-    

String

Return value:

Length of s

 

The function length returns the number of characters contained in the string s.

 

Example:

/* convert all characters of s into upper case */

for (i = 0; i < length (s); i++)

  if (s[i] >= 'a' && s[i] <= 'z')

    s[i] -= 32;

 

Note:

The function length calculates the length of a string only up to the first <NUL> character and thus is inappropriate to determine the length of a string containing binary data.
