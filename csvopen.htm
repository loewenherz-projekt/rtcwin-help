csvopen

    

RTC Function

Opens a CSV file.

Declaration:

bool csvopen (int fno, string fname)

Group:

File handling

Parameters:

fno
fname

Logical file number
File name

Return value:

Function result

 

The function csvopen opens a file and makes it available under the provided logical file number fno for subsequent read and write operations using csvread. The logical file number fno can freely be chosen from the range 0 to 7.

The parameter fname contains the full file name as follows:

internal/name.ext

The file name with the extension ext is contained in the battery-backed memory of the weighing terminal and cannot be accessed externally.

shared/name.ext

The file name with the extension ext is contained in the directory "shared". This folder can be accessed externally via FTP access (login: "shared," password as set in Service Mode).

usbstick/.../name.ext

The file name with the extension ext is contained on a USB memory stick. Further subdirectories may be specified as applicable.

Files on the USB memory stick can be accessed externally via FTP (login: "usbshared," password as set in Service Mode).

sharename/.../name.ext

The file name with the extension ext is contained on a network drive. sharename is the symbolic path name as it was returned by the instruction smbconnect. Further subdirectories may be specified as applicable.

internal/name.ext

The file name with the extension ext is contained in the battery-backed memory of the terminal and cannot be accessed externally.

shared/name.ext

The file name with the extension ext is contained in the directory "shared". This folder can be accessed externally via FTP access (login: "shared," password as set in Service Mode).

usbstick/.../name.ext

The file name with the extension ext is contained on a USB memory stick. Further subdirectories may be specified as applicable.

sharename/.../name.ext

The file name with the extension ext is contained on a network drive. sharename is the symbolic path name as it was returned by the instruction smbconnect. Further subdirectories may be specified as applicable.

If the file name does not specify a directory, the folder 'internal' is used as default. If the file can be opened successfully, csvopen returns true as function result; otherwise the result is false.

 

Example:

bool ReadCSVFile (string directory, string file_name)

{

int type;

string data[1023] = "";

 

  if (csvopen (1, concat (directory, file_name, ".csv"))) // Open CSV file

  {

    while (csvread (1, type, data))                       // Read until end of file...

    {

      switch (type)

      {

        case 1 : // Process data...

                 break;

                 

        case 2 : // Line break...

                 break;

 

        default :// Error

                 return (false);  

 

      }

    }

    

    if (csvclose (1))

      return (true);

    else

    {

      ShowErrorMsg ("Fehler Log-Datei");

      return (false);

    }

  }

  else

  {

    ShowErrorMsg ("Fehler Log-Datei");

    return (false);

  }

} 

 

Notes:

After opening a file, the internal read pointer is set to the beginning of the file.

Depending on type and performance of the connection, opening of a file may take several seconds. The programmer should respect this in his program design.
