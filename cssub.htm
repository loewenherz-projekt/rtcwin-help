cssub

    

RTC Function

Separates a substring from a string.

Declaration:

string cssub (string s, char separator, int n)

Group:

String functions

Parameters:

s
sc
n

Source string
Separating character
Number of substring

Return value:

Result string

 

The function cssub returns the substring specified by the integer parameter n. Counting starts from 0. The substrings contained in s are separated by the separator character sc. If s contains less than n+1 substrings, an empty string is returned as function result. If the separator specified by sc cannot be found in s, the function result is the complete string in case of n = 0 or an empty string in case of n > 0.

 

Example:

i = 0;
for (;;)
{

  s0 = cssub (ReceiveBuffer, ';', i++);

  if (s0 == "") break;

  /* ... */

}
