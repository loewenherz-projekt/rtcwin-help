iso2utf

    

RTC Function

Converts a string from ISO encoding to UTF-8 encoding.

Declaration:

string iso2utf (string s)

Group:

String functions

Parameters:

-    

Source string

Return value:

Result string

 

The function iso2utf converts the string s from ISO 8859 into UTF-8 encoding. For the encoding of the given string the ISO 8859 table is used, which is installed using the pragma statement, e.g. character set 15 for Western Europe.
