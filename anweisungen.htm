The Programming Language RTC   Program Design

Statements

    

The instruction part between the two curved brackets defines the action of the program or function. It typically consists of multiple individual statements and expressions. In detail:

·Assignments

i = 0;

A variable is assigned a constant value.

j = ConsNo;

A variable is assigned the content of another variable.

i = j + 1;

A variable is assigned the result of a mathematical operation.

 

·Function calls

PrintHeader();

Call of the function PrintHeader (without parameters).

mantare (2, 12.5);

Call of the function mantare with two parameters.

i = length (s);

Call of the function length. The function result is assigned to the variable i.

 

·Control structures

if (i > 0) j = i * 2;

Conditional assignment or function call.

while (ch != 0)
 ch = readkey();

Repetitive statement or function call.

 

·Compound statements

{
getweight (1);
 PosNo = i + 1;
 PrintTicket();
}

By a pair of curved brackets, multiple statements and function calls may be combined to a so-called compound statement.

 

Every statement or assignment must be terminated by a semicolon.
