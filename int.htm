int

    

RTC Declaration Word

Declaration of an integer variable.

Declaration:

int name [%len.0]       (decimals are always 0 for int)

Group:

Declaration words

Parameters:

name

Variable or function name

 

The instruction int declares a variable with given name or the return value of a function name as "integer." The type int has a resolution of 32 bit and thus represents the numbers in the range of

-2147483648 to 2147483647

or

0x00000000 to 0xFFFFFFFF

During declaration, the variable name may be assigned an initial value.

 

Example:

int ConsNo;

int i = 0;
