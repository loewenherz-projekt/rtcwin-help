getdischargedbulkweight

    

RTC Function

Captures the weight of the filled weigh hopper of a totalizing bulk weigher.

Declaration:

int getdischargedbulkweight (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Error code

 

The function getdischargedbulkweight captures the weight of the discharged weigh hopper of a totalizing bulk weigher. The instruction getdischargedbulkweight must precede a call of getfilledbulkweight to capture the weight of the filled weigh hopper.

The instruction getdischargedbulkweight reads the scale specified by the parameter sno and waits for the scale to stabilize (issettled = true). If the scale returns a stable weight, the weight values are updated as described in getweight. In addition, the difference between the weights of the filled weigh hopper determined by getfilledbulkweight and the current weight of the empty hopper is calculated and added to the variable totalbulkweight.

If the function getdischargedbulkweight can be executed successfully, getdischargedbulkweight returns 0 as function result; otherwise the corresponding error code is returned instead.
