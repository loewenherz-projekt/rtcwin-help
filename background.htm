background

    

RTC Function

Specifies the background color.

Declaration:

void background (int color)

Group:

Display and keyboard

Parameters:

color

Color code

Return value:

-

 

The instruction background specifies the background color for the consecutive write or draw instruction. The color is specified by the provided parameter color and may be passed as 8-bit or 32-bit value. When provided as 8-bit value, the color is taken from a lookup table. Examples are:

56

black

 

 

88

red

72

dark red

63

green

60

dark green

95

yellow

76

dark yellow

216

blue

136

dark blue

248

magenta

152

dark magenta

223

cyan

140

dark cyan

255

white

 

 

32–55

grayscale from black to white

RTC dialog functions use the following default background colors:

020

light blue/gray

Normal background

238

blue

Background of input/selection (e.g. input, select)

029

gray

Background of infobox, yesnobox, codebox

255

white

Background of additional help text (setinfo)

 

Alternatively, the background color can be given as 32-bit value. When shown as hexadecimal number, the digits are interpreted as follows:

 

Fixed value 00

Value from 0x00 to 0xff for blue

Value from 0x00 to 0xff for green

Value from 0x00 to 0xff for red

 

Example:

background (255);

clearscreen();     /* clear screen to white */
