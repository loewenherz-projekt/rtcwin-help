double

    

RTC Function

Converts an expression into data type double.

Declaration:

int double (any v)

Group:

Type conversion

Parameters:

v

Variable or function with result of any type

 

The function double converts the variable of any type specified by the parameter v into the data type double. If v is a variable of type bool, true is converted into 1 and false into 0, respectively.

If a string is passed as parameter v, all leading non-numeric characters are ignored. The subsequently following numerals up to the next non-numeric character form a number, which then is converted into double as described above.

Instead of a variable v, also a function or an algebraic expression may be provided as parameter.

 

Example:

CorrFactor = CorrFactor * double (MeasuredQty / DisplayedQty);
