division

    

RTC Variable

Increments of the most recently accessed scale.

Declaration:

loc decimal   division %8.dp

Group:

Scale functions

 

The variable division is a predefined variable of type decimal containing the increments of the scale most recently accessed by one of the functions getweight or getapprovedweight. In case of a multi-interval or multiple range scale, the variable division contains the increments of the currently valid range. The variable division has a length of 8 digits. The number of trailing decimals corresponds with the most recently accessed scale.

The variable division is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.

 

Example:

/* Round variable to the increments of scale */

decimal RoundWeight (decimal weight)

{

int i;

  if (weight < 0)

    i = int (weight / division - 0.5);

  else

    i = int (weight / division + 0.5);

 

  return (i * division);

}
