xmlsavefile

    

RTC Function

Saves the content of an XML instance to a file.

Declaration:

void xmlsavefile (int id, string fname)

Group:

XML interface

Parameters:

id
fname

Logical reference number of the XML instance
File name

Return value:

-

 

The function xmlsavefile saves the content of the XML instance specified by the parameter id into the file specified by the parameter fname. The logical reference number id has a range from 0 to 7 and addresses one of 8 possible XML instances. The parameter fname contains the full file name as follows:

internal/name.ext

The file name with the extension ext is contained in the battery-backed memory of the terminal and cannot be accessed externally.

shared/name.ext

The file name with the extension ext is contained in the directory "shared". This folder can be accessed externally via FTP access (login: "shared," password as set in Service Mode).

usbstick/.../name.ext

The file name with the extension ext is contained on a USB memory stick. Further subdirectories may be specified as applicable.

Files on the USB memory stick can be accessed externally via FTP (login: "usbshared," password as set in Service Mode).

sharename/.../name.ext

The file name with the extension ext is contained on a network drive. sharename is the symbolic path name as it was returned by the instruction smbconnect. Further subdirectories may be specified as applicable.

If the file name does not specify a directory, the folder 'internal' is used as default.

 

Example:

/* Send XML file...  */

xmlsavefile (XML_HANDLE, "shared/settings.xml");

 

Note:

The target file does not need to be opened before executing xmlsavefile.
