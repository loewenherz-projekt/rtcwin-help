The Programming Language RTC     Printout Formatting

Overview

    

With RTC, the creation of a layout for a printout appears to be quite comfortable. The layout of a printout is created and assigned to a format name by the declaration word form. To output the form to a printer the function print is used that is given the format name as parameter.

Furthermore labels can be imported and printed that have been created externally, e.g. using the program NiceLabel.
