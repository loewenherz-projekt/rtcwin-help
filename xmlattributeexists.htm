xmlattributeexists

    

RTC Function

Checks if a node in an XML instance has a specific attribute set.

Declaration:

bool xmlattributeexists (int id, string node, string attr)

Group:

XML interface

Parameters:

id
node
attr

Logical reference number of the XML instance
Path and name of the node
Attribute name

Return value:

Function result

 

The function xmlattributeexists searches the XML instance specified by the parameter id for the given node and checks if it has the specified attribute attr. The node is specified by the parameter node which contains the full path and name of the node.

If the node does contain the specified attribute, xmlattributeexists returns true as function result; otherwise the result is false.

 

Example:

int ReadMessage(string *msg)

{

int err=0;

 

  if (xmlattributeexists(XML_MSG, "message", "error")

  // Error: return error code

    err = int(xmlgetattribute(XML_MSG, "message", "error"));

  else

    // read message...

    msg = xmlgetnodevalue(XML_MSG, "message");

  return(err);

}
