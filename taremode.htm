taremode

    

RTC Variable

Shows the tare mode of the most recently accessed scale.

Declaration:

loc int   taremode

Group:

Scale functions

 

The value of the variable taremode reflects the tare status of the scale most recently accessed by one of the functions getweight or getapprovedweight. The status is shown as follows:

0  

Scale not tared (gross mode).

1  

Scale is auto tared (net mode).

2  

Scale tared with preset tare value (manual tare).

The variable taremode is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.
