preceived

    

RTC Function

Checks if any received characters are available.

Declaration:

int preceived (int port)

Group:

Communication

Parameters:

port

Logical port number

Return value:

Number of received characters

 

The function preceived checks the receive buffer of the communication channel specified by the logical port number port if it contains any characters not yet read. The port has to be opened first (e.g. using popen). If the receive buffer contains any characters, preceived returns the number of characters as function result; otherwise the result is 0.

The function preceived only checks the availability of a character without reading it. If a character was received, subsequent calls of charreceived always return a result greater zero until all characters have been read (e.g. using preadchar).

 

Example:

#define STX 2

 

bool WaitForStartChar (int port)

{

char ch;

  for (;;) /* Wait for start char... */

  {

    if (preceived (port))

    {

      ch = preadchar (port);    /* ... read it... */

      if (ch == STX)

        return (true);         /* Start char reiceived... */

      else

        return (false);

    }

    delay (0.05);

  }

}
