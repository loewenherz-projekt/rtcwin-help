widget_fontsize

    

RTC Function

Sets the font size of an object.

Declaration:

void widget_fontsize (int id, int size)

Group:

Touch screen programming

Parameters:

id
size

Logical reference number of the object
Font size in pixels

Return value:

-

 

The instruction widget_fontsize defines the font size in pixels to be used in the object specified by the parameter id. The font itself as well as possibly selected attributes remain unchanged.

 

Example:

void CreateButtonWidget (int id, string button_text)

{

  createwidget (id, WIDGET_TYPE_BUTTON);

    widget_dimensions (id, 100, 55);      

    widget_fontsize   (id, 24);  

    widget_fontface   (id, FT_RTC_BOLD);

    widget_foreground (id, "white");

    widget_background (id, "darkblue");

    widget_style      (id, ST_BUTTON_STYLE);

    widget_text       (id, button_text);

}

 

Note:

The fonts for the touch screen indicators are True Type fonts and therefore available at any size.
