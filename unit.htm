unit

    

RTC Variable

Weight unit of the most recently accessed scale.

Declaration:

loc string   unit[2]

Group:

Scale functions

 

The variable unit is a predefined variable of type string. It contains the weight unit of the scale most recently accessed by one of the functions getweight or getapprovedweight. The variable has a fix length of two characters. The units are "kg", "g ", "t " or "lb."

The variable unit is a task local variable, i.e. it may be used in multiple tasks, however assigns different physical memory for each task.

 

Example:

if (unit == "kg") RefWeight *= 1000;  // Reference weight always in grams
