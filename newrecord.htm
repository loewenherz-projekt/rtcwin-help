newrecord

    

RTC Function

Dialog for the entry of a single record in a database table.

Declaration:

char newrecord (dbase db)

Group:

Display and keyboard

Parameters:

db

Name of database table

Return value:

Code of terminating function key

 

The instruction newrecord generates a complete dialog for creating a new record in a database table. The structure of the dialog – as it appears on the application screen – complies with inputbox. The index field must be pre-set beforehand and cannot be changed by the function newrecord.

The function newrecord defines five function keys as follows:

F5

Display an optional help text (see setinfo)

F6

Exit dialog

The dialog is terminated when the enter key is pressed after the entry of the last table field or with the F6 function key. The key code of the key pressed for termination is returned as function result.

 

Example:

ch = input (100, 40, Client.No);

if (find (Client) != 0)

  if (yesnobox ("Client not found.\nCreate new record?", f) == ok)

    if (f) 

      ch = newrecord (Client);

    else

      ch = 0;

 

Notes:

The instruction newrecord provides multi-language operator prompting; the use of the function lsub is not necessary.

The instruction newrecord does not save the original content of the application screen or the previously selected font attributes. This has to be considered in the design of the program.
