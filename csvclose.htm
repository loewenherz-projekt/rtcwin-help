csvclose

    

RTC Function

Closes a CSV file.

Declaration:

bool csvclose (int fno)

Group:

File handling

Parameters:

fno

Logical file number

Return value:

Function result

 

 

The function csvclose closes the file specified by the parameter fno. fno is the logical file number that was assigned by the function csvopen. If the file can be closed successfully, csvclose returns true as function result; otherwise the result is false.

 

Example:

bool ReadCSVFile (string directory, string file_name)

{

int type;

string data[1023] = "";

 

  if (csvopen (1, concat (directory, file_name, ".csv"))) // Open CSV file

  {

    while (csvread (1, type, data))                       // Read until end of file...

    {

      switch (type)

      {

        case 1 : // Process data...

                 break;

                 

        case 2 : // Line break...

                 break;

 

        default :// Error

                 return (false);  

 

      }

    }

    

    if (csvclose (1))

      return (true);

    else

    {

      ShowErrorMsg ("Fehler Log-Datei");

      return (false);

    }

  }

  else

  {

    ShowErrorMsg ("Fehler Log-Datei");

    return (false);

  }

}
