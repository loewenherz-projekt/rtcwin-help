xmlsetattribute

    

RTC Function

Sets an attribute of a node in an XML instance.

Declaration:

bool xmlsetattribute (int id, string node, string attr, string value)

Group:

XML interface

Parameters:

id
node
attr
value

Logical reference number of the XML instance
Path and name of the node
Attribute name
Value of the attribute

Return value:

Function result

 

The function xmlsetattribute searches the XML instance specified by the parameter id for the given node. The node is specified by the parameter node which contains the full path and name of the node. If the node does contain the specified attribute, it is set to the provided value. If the node does not have the requested attribute, it is created.

If the attribute of the node can be set successfully, xmlsetattribute returns true as function result; otherwise the result is false.

 

Example:

void WriteErrorMessage(string msg)

{

 

  xmlsetattribute(XML_MSG, "message", "type", 2)); //Type=error

  xmlsetnodevalue(XML_MSG, "message", msg);

  xmlpwrite (XML_MSG, XML_PORT);

}
