The Programming Language RTC     Variables and Constants

Arrays

    

Where multiple implementations of the identical variable are needed, e.g. for multiple setpoint values, the variable may be declared as array. Doing so, only the first variable element is declared, followed by the number of required implementations.

Type  Name [ Number ]

Type is the data type specifying the value facet of the variable; Name is the name of the variable. Number is an integer value greater 0 that specifies the number of array elements.

 decimal FixTare%6.1[8];

Declaration of a decimal array for 8 fix tare values (length of 6 and one decimal each)

 string InfoText[4][40];

String array for four 40-character information texts (see notes)

 

Arrays may also be two-dimensional.

 int Matrix[4][6];

Declaration of a matrix with 4 lines and 6 columns

Arrays with three and more dimensions are not supported. String arrays can only have one dimension since they already have to be considered as char arrays.

Reference to a certain array element is accomplished via the name of the array followed by the ID of the element in square brackets. The ID starts with zero for the first element.

FixTare[2];

Access to the third (!) fix tare value

 for (i = 0; i < 4; i++)
    InfoText[i] =  "";

Clearing of the four info texts

 Matrix[1][3];

Access to the fourth element of the second line

Arrays may also be assigned start values during declaration:

 float Fixpoints[3] = {0.5,
    1.75, 3};

Declaration of a three-element array Fixpoints with start values 0.5, 1.75 and 3.

 int Matrix[2][3] = {10, 11,
    12, 20, 21, 22};

Declaration of a two-dimensional array matrix with the two lines containing the values 10, 11, and 12 and 20, 21 and 22.

 

Notes:

During the declaration of a string variable, strictly spoken an array of type char is declared. This explains why the maximum string length has to be specified in square brackets.

For the same reason in a string array (i.e. a two-dimensional char arrays), the order of string length (columns) and number of array elements (lines) is apparently interchanged.
