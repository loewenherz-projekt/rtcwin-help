decimal

    

RTC Declaration Word

Declaration of a formatted floating-point variable.

Declaration:

decimal name %len.dec

Group:

Declaration words

Parameters:

name
len
dec

Variable or function name
Length
Number of decimals

The instruction decimal declares a variable with given name or the return value of a function name as floating-point number. During declaration, the variable is also formatted for input and output in respect of length and number of decimals. The length len is understood as overall length including decimals and the decimal point itself, as applicable, whereas dec only defines the number of trailing decimals.

Where an integer variable is used instead of a discrete number of trailing decimals, the variable is only formatted during run time. Where the variable dp is used, the number of trailing decimals is set during run time according to the currently selected scale.

The type decimal approximately covers the range of ±10–10 to ±1010 with an accuracy of 10 digits. During declaration, the variable name may be assigned an initial value. The types decimal, float and double may be assigned each other.

 

Example:

decimal MaxValue%8.dp;

decimal SettleTime%4.1 = 5.0;

 

Note:

When the length is specified, the comma also has to be taken into account. The format %4.1 thus defines a variable with two digits before and one digit after the decimal point.
