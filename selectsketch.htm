selectsketch

    

RTC Function

Selects a drawing area for all following draw and write operations.

Declaration:

void selectsketch (int id)

Group:

Touch screen programming

Parameters:

id

Logical reference number of the object

Return value:

-

 

The instruction selectsketch selects a drawing area of type WIDGET_TYPE_SKETCH on the current screen for subsequent standard RTC draw and write operations.

 

Example:

void StartUpMsg(void)         

{

  selectwindow(STARTUP_ID);

  selectsketch(STARTUP_ID + 1);

  clearscreen();

  font(24);

  fontsize(27);

  showtext(80, 20,"   IT8000ET Demo");

  fontsize(22);

  showtext(80, 50,"Version 1.00   02.2012");

  showtext(80, 100,"    (c) SysTec GmbH");

  showtext(80, 130,"   www.systecnet.com");

  showwindow(STARTUP_ID);

}

 

Note:

If a screen contains only one drawing object, it is automatically selected and the use of selectsketch is not necessary.
