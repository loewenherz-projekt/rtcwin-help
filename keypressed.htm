keypressed

    

RTC Function

Checks if a key is pressed.

Declaration:

bool keypressed (void)

Group:

Display and keyboard

Parameters:

-

Return value:

result of type bool

 

The function keypressed checks whether the keyboard buffer contains any characters not yet read. If that is the case, keypressed returns true as function result; otherwise the result is false.

keypressed only checks the keyboard buffer without reading any character. If a key was pressed on the keyboard, subsequent calls of keypressed return true until the character was read (e.g. using readkey).

 

Example:

/* Empty keyboard buffer... */

while (keypressed()) readkey();
