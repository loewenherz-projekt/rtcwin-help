find

    

RTC Function

Searches a record in a database table.

Declaration:

int find (dbase db)

Group:

Database functions

Parameters:

db

Database table

Return value:

Function result

 

The function find searches the database table specified by the parameter db for a record matching the current content of the given index field. The record is stored against the index field, i.e. the field that has been declared first. The function result returns whether the record was found.

0

Record found

1

Record not found

2

Record not found and database table full

 

Example:

FirstWeight.No = IdentNo;

if (find (FirstWeight) == 0)

{

  /* Perform second weighing */

  delete (FirstWeight);

}

else

  ShowMessage ("First weight not found!");

 

Note:

The function find is a method of the declaration word dbase.
