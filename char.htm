char

    

RTC Declaration Word

Declaration of a "char"type variable.

Declaration:

char name

Group:

Declaration words

Parameters:

name    

Variable or function name

 

The function char declares a variable with given name or the return value of a function with name as "char." The type char can contain any character from the character set or the corresponding ASCII code. During declaration, the variable name may be assigned an initial value. The data type char is compatible to the type string with a length of 1.

Where a variable or return value of type char is assigned an integer variable, the integer variable thereafter contains the ASCII code of the character. Contrariwise, the lower 8 bit of the integer variable are interpreted as ASCII code.

The following assignments are allowed for the data type char:

     'A'

The character itself in single quotes

     255

The ASCII code of the character in decimal

     0xff

The ASCII code of the character in hexadecimal

     '\n'

The character as escape symbol (refer to string)

For the function keys of the keyboard, the following constants are defined:

ok

The ok and enter keys of the terminal as well as the return key of a connected PC keyboard

cup

Cursor-up key of the weighing terminal or the connected PC keyboard

cdown

Cursor-down key of the weighing terminal or the connected PC keyboard

cright

Cursor-right key of the weighing terminal or the connected PC keyboard

cleft

Cursor-left key of the weighing terminal or the connected PC keyboard

info

Info key of the weighing terminal

fct1 ... fct6

The function keys F1 to F6 of the weighing terminal or the connected PC keyboard

For the scale function keys, the following constants are available:

tar

Tare key of the weighing terminal or the F10 key of the connected PC keyboard

ptar

Preset tare key of the weighing terminal or the F8 key of the connected PC keyboard

set0

Zero setting key of the weighing terminal or the F11 key of the connected PC keyboard

scale

Scale select key of the weighing terminal or the F12 key of the connected PC keyboard

sel

Toggle key of the weighing terminal or the F9 key of the connected PC keyboard
