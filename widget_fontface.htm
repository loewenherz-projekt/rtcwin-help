widget_fontface

    

RTC Function

Sets the font of an object.

Declaration:

void widget_fontface (int id, int font)

Group:

Touch screen programming

Parameters:

id
font

Logical reference number of the object
Font

Return value:

-

 

The instruction widget_fontface defines the font to be used in the object specified by the parameter id. The parameter font determines the font to be used as follows. Font size and color remain unchanged and have – if required – to be set using the instructions widget_fontsize and widget_foreground.

Number

Font

Comment

24

RTC

Proportional bitmap font, display font

28

RTC Mono

Monospaced bitmap font, display font

Fonts 24 (RTC) and 28 (RTC Mono) contain the character tables ISO 8859-1 to -5, -7, -11 and -15 with all European characters as well as Cyrillic, Greek and Thai characters.

If applicable, one of the following attribute values may be added to the font number:

Number

Attributes

Comment

0

normal

1

bold

 

Example:

void CreateButtonWidget (int id, string button_text)

{

  createwidget (id, WIDGET_TYPE_BUTTON);

    widget_dimensions (id, 100, 55);      

    widget_fontsize   (id, 24);  

    widget_fontface   (id, FT_RTC_BOLD);

    widget_foreground (id, "white");

    widget_background (id, "darkblue");

    widget_style      (id, ST_BUTTON_STYLE);

    widget_text       (id, button_text);

}
