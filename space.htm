space

    

RTC Function

Creates a string of spaces.

Declaration:

string space (int n)

Group:

String functions

Parameters:

n

Length of result string

Return value:

Result string

 

The function space creates a string with the given length n that only contains space characters.

 

Example:

buffer = concat ("Gross:", space(7), fgross);

showtext (20, 30, buffer);
