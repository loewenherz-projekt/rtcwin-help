xmlpreviouschild

    

RTC Function

Sets the read/write position to the previous entry of the node of an XML instance.

Declaration:

bool xmlpreviouschild (int id, string node)

Group:

XML interface

Parameters:

id
node

Logical reference number of the XML instance
Path and name of the node

Return value:

Function result

 

The function xmlpreviouschild sets the read/write position by one position for subsequent access to the entries of a node. The parameter node contains the full path and name of the given node in the XML instance specified by the parameter id. If a previous entry can be selected successfully, xmlpreviouschild returns true as function result; otherwise the result is false.
