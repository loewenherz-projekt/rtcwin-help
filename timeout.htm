timeout

    

RTC Function

Checks a timer for time-out.

Declaration:

bool timeout (timer t)

Group:

Timer

Parameters:

t  

Variable of type timer

Return value:

Timer status.

 

The function timeout checks the timeout status of the provided timer variable t. If the timer has timed out, true is returned as function result; otherwise the return value is false. Before calling time-out, the timer must be started using the function starttimer.

 

Example:

void Filling (void)

{

  starttimer (BatchTimer, MaxFillingTime);

  while (!timeout (BatchTimer))

  {

    /* Filling loop... */

    break;

  }

  ShowErrorMsg ("Max. filling time exceeded!");

}
