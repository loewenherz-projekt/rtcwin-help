keymatrix_select

    

RTC Function

Opens a page with a matrix of buttons and waits for a button to be activated.

Declaration:

char keymatrix_select (int id, string title, int *key)

Group:

Touch screen programming

Parameters:

id
title
key

Logical reference number of the page
Page title
Variable for the number of the button pressed

Return value:

Key code

 

The instruction keymatrix_select activates the key matrix specified by the parameter id. The page with a matrix of buttons has to be generated beforehand using the instruction keymatrix_generate. The page may be assigned a headline using the parameter title.

If one of the buttons is pressed, keymatrix_select returns the key code of the pressed button. The integer variable key then returns the number of the button that was pressed. Counting of key starts from 0 for the leftmost button in the top row and continues row by row from left to right.

If the extra button in the lower left corner of the screen is pressed, the key code of the F1 function key is returned.

 

Example:

void CreateSettingsMenu (void)

{

  keymatrix_generate (PAGE_MAIN_SETTINGS, 4, 1);  // 4 buttons one below the other

    keymatrix_color (PAGE_MAIN_SETTINGS, 0, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 0, "Produkte");

    keymatrix_color (PAGE_MAIN_SETTINGS, 1, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 1, "Taraspeicher");

    keymatrix_color (PAGE_MAIN_SETTINGS, 2, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 2, "Einstellungen");

    keymatrix_color (PAGE_MAIN_SETTINGS, 2, 0x80C0FF) // Schaltfläche light blue

    keymatrix_text  (PAGE_MAIN_SETTINGS, 2, "Datensicherung");

}

void Settings (void)

{

char ch = 0; 

int key = 0;

 

  ch = keymatrix_select (PAGE_MAIN_SETTINGS, "Data input", key);

  if (ch == ok)

    switch (key)

    {

      case 0:  EditProducts();

               break;

      case 1:  EditTareValues();

               break;

      case 2:  Settings();

               break;

      case 3:  DataBackup();

               break;

    }

}
