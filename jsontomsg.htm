jsontomsg

    

RTC Function

Sends a JSON statement list to the message handler.

Declaration:

bool jsontomsg (int id, int dest, int type, string data)

Group:

JSON interface

Parameters:

id
dest
type
data

Logical reference number of the JSON statement list
Destination of the JSON statement list
Message type
Additional data for destination

Return value:

Function result

 

The function jsontomsg sends the content of the JSON statement list specified by the parameter id to the message handler for subsequent transfer to another instance. The parameter dest specifies the destination of the statement list as follows:

 

Destination

Description

0

Undefined

3

Application (interpreter)

8

Web interface

9

JSON server (Java script object notification)

Other destinations exist for internal communication. The parameter type specifies the message type as follows:

Message

Description

9

The data contain the configuration parameters requested by the WEB interface of the application.

10

The data contain the settings requested by the WEB interface of the application.

11

Java script object notification (JSON) for the process visualization.

14

Message for the web server.

Other message numbers are reserved for internal purpose. If necessary, the parameter data can be used to send additional data (metadata) to the destination.

If the JSON statement list can be transferred to the message handler successfully, jsontomsg returns true as function result; otherwise the result is false.

 

Example:

#define EVENT_APPLICATION_WS_SERVER   9

#define MSG_WEB_LINK                  14

 

jsonsetvalue (JSON_HANDLE, "cmd", "wss-connection-status");

jsontomsg(JSON_HANDLE, EVENT_APPLICATION_WS_SERVER, MSG_WEB_LINK, "");
