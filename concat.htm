concat

    

RTC Function

Converts various data types into strings and concatenates them to a result string.

Declaration:

string concat (any v0, … vn)

Group:

String functions

Parameters:

v

Source element

Return value:

Result string

 

The function concat converts the provided source elements of any data type into strings and combines them to a result string. According to their data type, the individual elements are converted as follows:

string

The string is embedded into the result string without modification (up to the first Null character).

char

The ASCII code of the character is embedded into the result string as one byte.

int

The integer number is converted into a string with a fix length of 10 characters. Leading non-used digits are shown as spaces. If a format is specified during declaration of an integer variable, it is included in the result string accordingly.

decimal

Variables are converted according to their format specified during declaration.

float

The floating-point number is converted into a string with a total length of 12 characters. It has a fix number of four decimals. Leading non-used digits are shown as spaces. If a format is specified during declaration of a float variable, it is represented accordingly in the result string.

double

The floating-point number is converted into a string with nine digits before and eight digits after the decimal point. Leading non-used digits are shown as spaces. If a format is specified during declaration of a double variable, it is represented accordingly in the result string.

bool

The Boolean number is converted into '0' or '1' according to the status false or true.

 

Example:

writestring (Host, concat (date, ";", time, ";", OrderNo, ";"',

                           Client.No, ";", PosNo, ";",

                           Article.No, ";", Parts, ";", tare, ";", net));

 

Note:

Numeric elements of the types int, float, double or decimal may also be formatted using the function adj.
