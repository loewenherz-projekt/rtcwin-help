storetotalbulkweight

    

RTC Function

Stores the totalized weight in the approved weight storage.

Declaration:

int storetotalbulkweight (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Error code

 

The function storetotalbulkweight concludes totalizing of weights of a totalizing bulk weigher and stores the current content of the variable totalbulkweight in the approved weight storage (alibi memory). The variable alibino thereafter contains the identification number the weights total is stored against.

If the function can be completed successfully, storetotalbulkweight returns 0 as function result; otherwise the corresponding error code is returned instead and the variable alibino contains 0.
