activatekey

    

RTC Function

Induces a key-press under program control.

Declaration:

void activatekey (char ch)

Group:

Display and keyboard

Parameters:

ch

Key code of the actuated key

Return value:

-

 

The function activatekey induces a key-press under program control – e.g. out of a background task. The key code is provided as parameter ch. The key-press is read and evaluated with the next keyboard access, e.g. when the function readkey is executed. Whether the key has been read can be checked using the function keyactivated.

 

Example:

if (cmd = CM_CANCEL) // Terminate operator input...

{

  activatekey (fct6);

  while (!keyactivated ()) delay (0.1);

}

 

Note:

Prior to the execution of activatekey, a check should be carried out whether the previously induced key has already been processed (read). If this is not the case, the previously induced key will be overwritten by the new key and get lost.
