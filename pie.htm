pie

    

RTC Function

Draws a segment of an ellipse.

Declaration:

void pie (int w, int h, int a0, int a1)

Group:

Display and keyboard

Parameters:

w
h
a0
a1

Horizontal width of the ellipse (X direction)
Vertical height of the ellipse (Y direction)
Start angle of the segment
End angle of the segment

Return value:

-

 

The instruction pie draws a segment of an ellipse. The current cursor position specifies the center of the reference ellipse whereas the two parameters w and h specify their width and height.The two other parameters specify start and end angle of the segment. Angles start with 0° for the rightmost position on the horizontal axis and are counted counterclockwise. The segment is filled with the current drawing color.

 

Example:

posto(20, 20);

pie(40, 20, 0, 45);
