infobox

    

RTC Function

Instruction to show a pop-up message.

Declaration:

void infobox (string text)

Group:

Display and keyboard

Parameters:

text

Prompt

Return value:

-

 

The instruction infobox produces an embossed box in the center of the application screen that contains the text string specified by the parameter text. The text is shown in black proportional characters on light gray background and may extend over up to four lines. Line wraps can be specified by the sequence '\n'.

 

Example:

infobox ("Please place filled container on\nweighing platform.");

 

 

Note:

The instruction infobox provides multi-language operator prompting; the use of the function lsub is not necessary.

The instruction infobox does not save the original content of the application screen. I.e. the programmer cannot expect that text previously written into the display is still present after the execution of infobox and must design his program accordingly.
