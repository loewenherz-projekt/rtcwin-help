RTC Variable

Identification number of the approved weight storage.

Declaration:

decimal   alibino%4.0 / %6.0

Group:

Scale functions

 

The variable alibino is a predefined variable of type decimal. After execution of the function getapprovedweight, alibino contains the identification number against that the weighing was stored in the approved weight storage (alibi memory). The identification number (alibi number) has 4 or 6 digits depending on the configuration and is incremented automatically. A 4-digit identification number starts with 1 after a new date. So, strictly speaking, the approved alibi number is combined from the date and the variable alibino. A 6-digit identification number is incremented automatically after each weighing and only starts again with 1 after passing 999999.
