jsontype

    

RTC Function

Determines the type of an object in a JSON statement list.

Declaration:

int jsontype (int id, string object)

Group:

JSON interface

Parameters:

id
object

Logical reference number of the JSON statement list
Object in JSON statement list

Return value:

Object type

 

The function jsontype returns the type of the given object in the JSON statement list specified by the parameter id. The function result contains the object type as follows:

Result

Type

Description

0

VALUE_TYPE_STRING

Object is a text.

1

VALUE_TYPE_NUMBER

Object is a number or a numeric value.

2

VALUE_TYPE_OBJECT

Object is an 'object', e.g. a bitmap file.

3

VALUE_TYPE_ARRAY

Object is an array.
