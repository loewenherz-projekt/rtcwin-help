lock

    

RTC Function

Locks a semaphore variable.

Declaration:

void lock (semaphore sp)

Group:

Multitasking

Parameters:

sp

Name of semaphore variable

Return value:

-

 

The instruction lock locks the semaphore variable specified by sp. A locked semaphore variable can be released using the function unlock.

If the semaphore variable was already locked when lock is executed, further program execution waits until the semaphore variable is free. Where a possible delay of the program cannot be accepted, the function trylock may be used instead.

 

Example:

semaphore Sema1;

 

bool WeighingProcess (void)

{

int result; 

  lock (Sema1);

    result = getapprovedweight (indicatedscale ());

    if (result == 0)

      UpdateParameter ();

  unlock (Sema1);

  return (result);

}
