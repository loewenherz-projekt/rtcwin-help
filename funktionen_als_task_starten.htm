The Programming Language RTC     Multitasking

Run a Function as a Task

    

Besides the option to link certain functions to the fix system background task, it is also possible to run any function without parameters as a task. For this purpose, the following instructions are available:

int starttask (procname p)

Installs and runs the given function without parameters as a background task.

void stoptask (procname p)

Stops execution of the given background task.

For access to commonly used system resources so-called semaphores can be defined and used for synchronization. The following instructions can be used:

semaphore name;

Declares a semaphore variable with the provided name.

void lock (semaphore sp)

Checks the status of the given semaphore variable and locks it if possible. If the semaphore variable is already locked, further program execution is delayed until the semaphore is released.

bool trylock (semaphore sp)

Checks the status of the given semaphore variable and locks it if possible. If the semaphore variable can be locked successfully, the function returns true as function result; otherwise the result is false.

void unlock (semaphore sp)

Unlocks the given semaphore variable.
