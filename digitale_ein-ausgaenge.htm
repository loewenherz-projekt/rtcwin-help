The Programming Language RTC     External Inputs/Outputs

Digital Inputs/Outputs

    

RTC provides a range of functions for the handling of the digital inputs and outputs. They allow to read inputs and set outputs both individually and in groups.

void bout (int bitno, bool stat)

Set the specified digital output.

bool bin (int bitno)

Read the specified digital input.

bool out (int bitno)

Read the specified digital output.

void pout (int n, int msb, int lsb)

Write a value to a group of digital outputs. The group is specified by the most and least significant bits.

int pin (int msb, int lsb)

Read a group of digital inputs. The group is specified by the most and least significant bits.

int outputs (int msb, int lsb)

Read a group of digital outputs. The group is specified by the most and least significant bits.

The configuration of digital inputs and outputs is contained in the Service Mode of the weighing terminal. Thus, the functions listed above are independent of the type of I/O module.
