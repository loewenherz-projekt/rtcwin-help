beginrow

    

RTC Function

Starts a new row inside a grid.

Declaration:

void beginrow (void)

Group:

Touch screen programming

Parameters:

-

 

Return value:

-

 

The instruction beginrow starts a new row in a layout of type LAYOUT_TYPE_GRID. As in a layout, all following objects up to the statement endrow are combined and treated as a logical entity. The usage of beginrow ... endrow is mandatory for a grid layout.

 

Example:

enum

{

  PAGE_1,

  PAGE_2,

}

 

enum

{

  ID_BUTTON_0,

  ID_BUTTON_1,

  ID_BUTTON_2,

  ID_BUTTON_3,

  ID_BUTTON_4,

  ID_BUTTON_5,

}

 

void CreatePage (void) 

{

  begindraw();

  createpage (PAGE_2);

  page_background (PAGE_2, "#d0e0ff");  // SysTec Default

    // Outer layout as grid:

    beginlayout (LAYOUT_TYPE_GRID);

      // First line: three buttons horizontally aligned

      beginrow();

        createwidget (ID_BUTTON_0, WIDGET_TYPE_BUTTON);

        createwidget (ID_BUTTON_1, WIDGET_TYPE_BUTTON);

        createwidget (ID_BUTTON_2, WIDGET_TYPE_BUTTON);

      endrow();  

      // Second line: two buttons horizontally aligned

      beginrow();

        createwidget (ID_BUTTON_3, WIDGET_TYPE_BUTTON);

        createwidget (ID_BUTTON_4, WIDGET_TYPE_BUTTON)

      endrow();  

    endlayout(); // close outer layout  

  enddraw();

}  

 

The five objects are arranged in the layout as follows:

Button 0

Button 1

Button 2

Button 3

Button 4

 

Notes:

Applicable preprocessor constants (e.g. LAYOUT_TYPE_GRID) are available in the file "qtmonitor.h."
