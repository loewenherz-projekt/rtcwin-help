boxto

    

RTC Function

Draws a rectangular box.

Declaration:

void boxto (int x, int y, int style)

Group:

Display and keyboard

Parameters:

x
y
style

Horizontal coordinate x of opposite corner
Vertical coordinate y of opposite corner
Box type

Return value:

-

 

The instruction boxto draws a rectangular box on the application screen starting at the current cursor position. The diagonally opposite corner is specified by the coordinates x and y. This corner of the box section is the endpoint. The rectangle is filled with the current drawing color. The parameter style determines the box type as follows:

 

style

Description

0

No box

1

Flat box (no frame), inside background color

2

Embossed box, inside background color

3

Engraved box, inside background color

4

Embossed frame, inside unchanged

5

Engraved frame, inside unchanged

6

Embossed box with thin frame, inside background color

7

Engraved box with thin frame, inside background color

8

Embossed thin frame, inside unchanged

9

Engraved thin frame, inside unchanged

10

Box with engraved frame, inside background color

11

Box with embossed frame, inside background color

12

Engraved frame, inside unchanged

13

Embossed frame, inside unchanged

14

Box with thin black frame, inside background color

 

Example:

/* Show alarm message */

begindraw();

  background (88);

  foreground (255);

  posto (40, 20);

  boxto (280, 80, 2);

  fontsize (19);

  showtext (65, 40, "Scale in overload!");

enddraw();
