The Programming Language RTC     Output to the Display

Dialog Functions

    

Besides the basic functions for output to the display, RTC provides a set of powerful macros. Some of them are complete dialogs and accordingly affect settings of colors, fonts, character sizes and others.

void inputbox (string title) ...

Complete dialog for variable and parameter entry. Function keys may also be assigned additional functions.

char select (dbase db, int x1, int y1,
int x2, int y2)

Dialog for selection of a record from a database table.

void searchbox  (dbase db)

Provides a list of available records of the specified database table with the option to browse and select a record.

void selectbox (string title) ...

Select menu to choose a function from a set of options (e.g. main or supervisor menu).

void edit (dbase db)

Complete dialog to create and edit a record for/of the specified database table.

void infobox (string text)

Pop-up message box with specified text.

char yesnobox (string text, bool f)

Create a yes/no query box.

char codebox  (string prompt,
string pw)

Input box for hidden password entry.

void fctinfo (int n, string info)

Create a label for a function key (soft key).

void setinfo (any v, string infotext)

Assign an additional help text to the variable v infotext.

void setunit (any v, string unittext)

Assign a unit sign to the variable v unittext.

void setsearchtitle (dbase db,
string titletext)

Assign a headline to the database table db for the use of searchbox.
