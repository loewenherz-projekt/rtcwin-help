xmlinsertchild

    

RTC Function

Inserts a new node or text into an XML instance.

Declaration:

bool xmlinsertchild (int id, string path, string name, int type, int position)

Group:

XML interface

Parameters:

id
path
name
type
position

Logical reference number of the XML instance
Path of the node
Text or name of the node
Type of child
Insert position

Return value:

Function result

 

The function xmlinsertchild inserts a text or a node specified by the parameter name into the XML instance specified by the parameter id. The parameter type defines the entry type as follows:

type = 0

Empty; nothing is inserted into the XML instance.

type = 1

Entry is a node. The node is inserted as child node of the parent specified by path.

type = 2

Entry is a text. The text is inserted at the position specified by the parameter path.

The parameter position specifies whether the new entry is written before (position = 0) or after (position = 1) the current position. The current position has to be set beforehand, e.g. by using xmllastchild.

If the file can be inserted successfully into the XML instance, xmlinsertchild returns true as function result; otherwise the result is false.

 

Example:

void WriteLogFile(string msg)

{

  xmlloadfile(XML_WEIGH_LOG, "shared/weighlog.xml")

  // lookup last entry:

  xmllastchild(XML_WEIGH_LOG, "logging");

  // append new entry:

  xmlinsertchild(XML_WEIGH_LOG, "logging", msg, 2, 1));

  xmlsavefile(XML_WEIGH_LOG, "shared/weighlog.xml")

}
