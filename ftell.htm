ftell

    

RTC Function

Returns the current read/write position of an opened file

Declaration:

int ftell (int fno)

Group:

File handling

Parameters:

fno

Logical file number

Return value:

Current read/write position

 

The instruction ftell returns the current read/write position of the file specified by the logical file number fno. The file has to be opened first using the instruction fopen. The returned value is understood as number of characters counted from the beginning of the file. Thus, a function result of 0 indicates that the current read/write position is at the beginning of the file.

 

Example:

/* Read parameters from USB memory */

bool ReadFile (string directory)

{

string data_buffer[1023] = ""; 

 

  if (fopen (1, concat (directory, "WeighLog.csv")) == 0) // directory = "usbstick/"

  {

    if (ftell (1) > 0)                                    // Read current read/write position

    {

      if (fseek (1, 0, 0) == 0)                           // Move cursor to the beginning of file 

      {

        while (fread (1, data_buffer, 1023) > 0)          // Read data from file

        {

          /* Process parameter  */

        }

        return (true);

      }  

      else

      {

        ShowErrorMsg ("Zugriff verweigert");

        return (false);

      }

    } 

    else

    {

      ShowErrorMsg ("Zugriff verweigert");

      return (false);

    }   

  }

  else  

  {

    ShowErrorMsg ("Fehler Log-Datei");

    return (false);

  }

}
