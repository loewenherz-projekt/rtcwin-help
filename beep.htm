beep

    

RTC Function

Outputs an acoustic signal.

Declaration:

void beep (int freq, float time)

Group:

System

Parameters:

freq
time

Frequency in Hertz
Duration in seconds

Return value:

-

 

The function beep activates the integrated beeper for a period of time in seconds specified by the parameter time. The parameter freq determines the frequency and thus the pitch of the acoustic signal.

 

Example:

/* Output beep for 200ms */
beep (2300, 0.2);

 

Note:

The function beep is supported by the indicators IT6000E and IT8000E only. The volume of the acoustic signal depends on the frequency and the type of beeper.
