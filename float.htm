float

    

RTC Declaration Word

Declaration of a floating-point variable.

Declaration:

float name [%len.dec]

Group:

Declaration words

Parameters:

name
len
dec    

Variable or function name
Optional specification of length
Optional number of decimal digits

 

The instruction float declares a variable with given name or the return value of a function with given name as floating-point number. The type float covers the real numbers in the range of

± 3.4 * 10–38 to ± 3.4 * 1038

with a precision of 7–8 digits. The data type double can be used for higher precision. During declaration, the variable name may be assigned an initial value. The types float, decimal and double may be assigned each other.

During declaration, the variable can optionally be formatted for input and output in respect of length and number of decimals. The length len is understood as overall length including decimals and the decimal point itself, as applicable, whereas dec only defines the number of trailing decimals.

Example:

float MaxValue%8.dp;

float pi = 3.1415;

 

Note:

When the optional length is specified, the comma also has to be taken into account. The format %4.1 thus defines a variable with two digits before and one digit after the decimal point.
