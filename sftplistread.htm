sftplistread

    

RTC Function

Reads a file name from an SFTP server.

Declaration:

bool sftplistread (int sfno, string *name)

Group:

File handling

Parameters:

sfno
fname

Logical connection number
String variable for the file name read

Return value:

Function result

 

The function sftplistread reads the name of the next file contained in the directory on the SFTP server. Beforehand the connection to the SFTP server has to be established via sftpopen and the files contained on the server be made available using sftplist. The parameter sfno is the logical connection number assigned with the execution of the function sftpopen. If any (further) file exists in the given directory on the SFTP server, the name is returned under the provided parameter fname and sftplistread returns true as function result. In this case, the internal read pointer is moved to the next file for subsequent calls of the function sftplistread. If no (further) file name can be read, sftplistread returns false.

 

Example:

/* ... */

if (sftpconnected (0))

{

  if (sftplist (0, "shared"))   // open directory

  {

    while (sftplistread (0, filename))  // read files...

    {

    // process files contained in filename......

          

    // Rename processed file

      if (!sftprename (0, concat ("shared/", filename), concat ("shared/", getstddate(), "_", filename)))

        ShowErrorMsg ("SFTP access denied!");

    }

  }

}

else

  ShowErrorMsg ("Connection to SFTP server lost!");
