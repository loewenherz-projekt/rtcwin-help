show_startup_message

    

RTC Function

Shows a power-up message on the screen.

Declaration:

void show_startup_message (string title, string text)

Group:

Touch screen programming

Parameters:

title
text

Title of message
Message text

Return value:

-

 

The instruction show_startup_message creates a frame on the screen containing the headline specified by the parameter title and the message specified by text. show_startup_message can be used as a first statement when the program is started to display a corresponding message during a longer start-up phase.

The parameter title may be an empty string. In that case, the text is shown borderless. The parameter text may also contain HTML formatting commands.

The message can be removed using the function hide_startup_message.

 

Example:

void main(void)
{

  show_startup_message ("", "<H1 ALIGN=CENTER><font size=5>Starting up....</font size></H1>")

  CreatePages();
  Init();

  hide_startup_message();
  MainApplication();

}
