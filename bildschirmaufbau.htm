The Programming Language RTC     Touch Screen Programming

Screen Layout

    

First step of touch screen programming is to create one (or multiple) pages or screens using the instruction createpage on which then the various objects (the widgets) are placed. This allows to create individual screens for the various program parts, e.g. weighing mode or settings menu.

In addition, the instruction createwindow may be used to create a window, which then contains the individual objects. When called up, the window is shown on top of the currently selected page and in difference to a page does not necessarily use the complete display space.

To position the elements on a page or a window pre-defined layouts are available, which combine logically associated objects, determine their order and size and help to align the objects horizontally, vertically or in a grid. The assignment of objects to a layout is made using beginlayout and endlayout. Layouts may also be nested, i.e. inside of a layout containing horizontally aligned objects other layouts may be placed containing vertically aligned objects. The following example shows a possible layout:

Example:

enum

{

  PAGE_1,

  PAGE_2,

}

 

enum

{

  ID_BUTTON_0,

  ID_BUTTON_1,

  ID_BUTTON_2,

  ID_BUTTON_3,

  ID_BUTTON_4,

  ID_BUTTON_5,

}

 

void CreatePage (void) 

{

  begindraw();

  createpage (PAGE_1);

  // Create outer vertical layout

  beginlayout (LAYOUT_TYPE_VBOX);

    // First line: three buttons horizontally aligned

    beginlayout (LAYOUT_TYPE_HBOX);

      createwidget (ID_BUTTON_0, WIDGET_TYPE_BUTTON);

      createwidget (ID_BUTTON_1, WIDGET_TYPE_BUTTON);

      createwidget (ID_BUTTON_2, WIDGET_TYPE_BUTTON);

    endlayout();  

    // Second line: two buttons horizontally aligned

    beginlayout (LAYOUT_TYPE_HBOX);

      createwidget (ID_BUTTON_3, WIDGET_TYPE_BUTTON);

      createwidget (ID_BUTTON_4, WIDGET_TYPE_BUTTON);

    endlayout();  

  endlayout(); // close outer layout  

  enddraw();

}

 

void main (void)

{

  Init ();

  recorddisplay

  {

    CreatePage ();  // Create a new screen

  }

  /* ... */

}

Alternatively, the objects can also be placed in a fixed grid. The individual grid rows are then defined using the instructions beginrow and endrow:

 

Example:

void CreatePage (void) 

{

  begindraw();

  createpage (PAGE_2);

  // Outer layout as grid:

  beginlayout (LAYOUT_TYPE_GRID);

    // First line: three buttons horizontally aligned

    beginrow();

      createwidget (ID_BUTTON_0, WIDGET_TYPE_BUTTON);

      createwidget (ID_BUTTON_1, WIDGET_TYPE_BUTTON);

      createwidget (ID_BUTTON_2, WIDGET_TYPE_BUTTON);

    endrow();  

    // Second line: two buttons horizontally aligned

    beginrow();

      createwidget (ID_BUTTON_3,WIDGET_TYPE_BUTTON);

      createwidget (ID_BUTTON_4,WIDGET_TYPE_BUTTON)

    endrow();  

  endlayout(); // close outer layout  

  enddraw();

}
