recorddisplay

    

RTC Function

Combines a group of subsequent display functions.

Declaration:

void recorddisplay (void)
{
 statement( )
 statement( )
 ...
}

Group:

Touch screen programming

Parameters:

-

 

Return value:

-

 

The instruction recorddisplay combines a set of display layout instructions to be treated as a logical entity. Thus, the output of the display instructions required for the current screen is repeated when a second display connects to the weighing terminal.

 

Example:

// Combine screen commands

recorddisplay

{

  CreateSimpleWeighingPage();

  CreateInfoBox();

  CreateFillingPage();

  CreateOperatorListPage();

  setdisplayoption (DISPLAY_OPTION_SCALE_BUTTONS, SCALE_BUTTON_ZERO         |

                                                  SCALE_BUTTON_SELECT       |

                                                  SCALE_BUTTON_PRESET_TARE  |

                                                  SCALE_BUTTON_TARE);

}
