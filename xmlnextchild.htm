xmlnextchild

    

RTC Function

Sets the read/write position to the next entry of a node of an XML instance.

Declaration:

bool xmlnextchild (int id, string node)

Group:

XML interface

Parameters:

id
node

Logical reference number of the XML instance
Path and name of the node

Return value:

Function result

 

The function xmlnextchild sets the read/write position to the next entry for subsequent access to the entries of a node. The parameter node contains the full path and name of the given node in the XML instance specified by the parameter id. If a next entry can be selected successfully, xmlnextchild returns true as function result; otherwise the result is false.

 

Example:

void ReadLogFile(string msg)

{

bool f;

string buf[255];

 

  xmlloadfile(XML_WEIGH_LOG, "shared/weighlog.xml")

  // lookup first entry:

  f=xmlfirstchild(XML_WEIGH_LOG, "logging");

  while (f)

  {

    // read data...

    buf = xmlgetchildvalue(XML_WEIGH_LOG, "logging");

    // ... and evaluate.
     ...

 

    f = xmlnextchild(XML_WEIGH_LOG, "logging");

  }

}
