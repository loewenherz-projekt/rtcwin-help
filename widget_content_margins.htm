widget_content_margins

    

RTC Function

Sets the outer margins of an object.

Declaration:

void widget_content_margins (int id, int left, int top, int right, int bottom)

Group:

Touch screen programming

Parameters:

id
left
right
top
bottom

Logical reference number of the object
Left margin in pixels
Right margin in pixels
Top margin in pixels
Bottom margin in pixels

Return value:

-

 

The instruction widget_content_margins sets the outer margins of the object specified by the parameter id according to the parameters left, right, top and bottom. The parameters specify the gap in pixels to the left-hand, right-hand, upper and lower neighbor object. If a parameter is -1, the margins remains unchanged.
