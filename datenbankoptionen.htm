The Programming Language RTC     Databases

Options

    

The behavior of the functions select and edit can be customized through optional control statements. The control statements are appended to the field in the table declaration as below.

 

dbase "Article"   Article[100]

{

   "Article No."  decimal  No%3.0;      selectmode=input; 

   "Name"         string   Name[25];
   "Price/kg"     decimal  Price%6.2;

   "Consumption"    decimal  Total%8.dp;  editmode=skip;

}

 

The following control statements are available:
 

Statement

Applies to

Description

selectmode=input

select

During selection, a new record can be created after pressing F4 function key.

editmode=skip

edit

The field is invisible during entry.

editmode=showonly

edit

The field can only be viewed but not entered.

editmode=select(db)

edit

Entry of this field is made via selection from another database table.

range=expr

edit

The entry is only accepted if expr returns true. This function can be used to check plausibility of entry.

skipcondition=expr

edit

The entry is skipped if expr returns true.
