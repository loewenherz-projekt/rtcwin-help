The while loop

    

The statement while creates a loop where the condition for continuation is checked prior to the execution of the loop itself. The structure is as follows:

 
while (Expression)
  Statement;
 

Statement is executed as long as the result of the Boolean expression is true. If the result of expression is false, the loop is terminated.

·There is no semicolon behind Expression,
·Statement may consist of multiple statements, combined by a pair of curved brackets (compound statement),
·If Expression already results in false at first check, Statement is not executed at all.

 

Example:

string ReadString (int port)

{  

  string ReceiveBuffer[80] = "";

  int i = 0;

  

  while (charreceived (port) && i < 80)

    ReceiveBuffer[i++] = readchar (port);

    

  return (ReceiveBuffer);  

}
