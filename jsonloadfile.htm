jsonloadfile

    

RTC Function

Loads the content of a file into a JSON statement list.

Declaration:

bool jsonloadfile (int id, string fname)

Group:

JSON interface

Parameters:

id
fname

Logical reference number of the JSON statement list
File name

Return value:

Function result

 

The function jsonloadfile reads the content of the file specified by the parameter fname and writes it into the JavaScript statement list specified by the parameter id for subsequent evaluation. The logical reference number id has a range from 0 to 7 and addresses one of 8 possible JSON statement lists. If the content of the file can be loaded successfully into the statement list, jsonloadfile returns true as function result; otherwise the result is false.

 

Example:

/* Read statement list...  */

jsonloadfile(JSON_HANDLE, "shared/functions.js");
/* ... */

 

Note:

The file does not have to be opened before it is read using jsonloadfile.

The statement list is automatically cleared before new content is written to it. Calling jsonclear is not necessary.
