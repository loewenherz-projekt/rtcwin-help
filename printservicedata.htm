printservicedata

    

RTC Function

Prints the Service Mode settings.

Declaration:

int printservicedata (int port)

Group:

System

Parameters:

port

Logical port number

Return value:

Error code

 

The function printservicedata reads all Service Mode settings and outputs them on the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen).

If Service Mode data can be printed successfully, printservicedata returns 0 as function result; otherwise the corresponding error code is returned instead.

 

Beispiel:

/* Print service mode parameter */

int result = printservicedata (port);
