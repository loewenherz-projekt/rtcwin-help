msgargcnt

    

RTC Function

Returns the number of arguments of a message.

Declaration:

int msgargcnt (void)

Group:

Message handler

Parameters:

-

 

Return value:

Number of arguments

 

The function msgargcnt returns the number of arguments contained in the data container of the most recently read message. Beforehand the message has to be read from the message queue using the function msgread. The values of the individual arguments can be evaluated using the function msgargval.

 

Example:

  switch (msgread (0.2))

  {

    case MSG_NONE:            break;

    case MSG_EVENT_APPLICATION_WEB_INTERFACE:

    {

      for (i = 0; i < msgargcnt(); i++) // read arguments from message...

        data[i] = msgargval(i);

      /* ... */

      break;

    } 

    /* ... */

  }
