findnextfield

    

RTC Function

Finds the next record in a database table.

Declaration:

int findnextfield (dbase db, any field)

Group:

Database functions

Parameters:

db
field    

Database table
Search field

Return value:

Function result

 

The function findnextfield searches the database table specified by the parameter db for a record succeeding the current one. As search criteria the given search field specified by the parameter field is used. If more than one record exists matching the specified search field, the index field – i.e. the field that has been declared first – is taken as secondary search criteria. The function result returns whether a successive record was found.

0

Record found

1

Record not found

 

Example:

/* Output first weight records sorted by plate No. */

FirstWeight.No = 0;

while (findnextfield (FirstWeight, PlateNo) == 0)

{

  /* Output record */

}

 

Note:

The function findnextfield is a method of the declaration word dbase.
