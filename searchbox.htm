searchbox

    

RTC Function

Searches in a database table.

Declaration:

void searchbox (dbase db)

Group:

Display and keyboard

Parameters:

db

Database table

Return value:

-

 

The instruction searchbox creates a list of table entries for the database table specified by db sorted by the search field — i.e. the table field created in second place — in ascending order. The instruction setsearchtitle can be used to assign a headline for the searchbox. If no use is made of this option, the list of records is shown without a headline.

 

The operator can select a record with the cursor up and down keys. The selected record is indicated by bold characters. The cursor right and left keys can be used to shift the listed records horizontally to view further fields of the table. By pressing the enter key the selection is made and the record is available for further processing.

Alternatively, a record can be selected by typing in the first character(s) of the search field. As feedback, a blue input box pops up. During entry, automatically the first record is highlighted whose search field matches the entry made so far.

 

 

Note:

The function infobox does not save the original content of the application screen. I.e. the programmer cannot expect that text previously written into the display is still present after the execution of searchbox and must design his program accordingly.
