codebox

    

RTC Function

Dialog for password entry.

Declaration:

char codebox (string prompt, string pw)

Group:

Display and keyboard

Parameters:

prompt
pw

Operator prompt for password entry
Password

Return value:

Code of terminating function key

 

The instruction codebox provides a complete dialog for the entry of a password. codebox creates an embossed box in the center of the application screen that shows the upper part the text specified by the parameter prompt to prompt the operator for password entry. The text is shown in black proportional characters on light gray background and may extend over up to three lines. Line wraps can be specified by the sequence '\n'. Below the text, an input field allows the entry of the password. The entered characters are replaced by asterisks.

For the prompt as well as for the password entry, the currently selected font, size and colors are used. The background color of the input field changes to blue.

If the entry is completed with pressing the enter key the entered password is compared with the provided parameter pw. If entry and parameter pw match, the dialog is terminated. Otherwise a corresponding message is shown and the enter key discarded. If the password entry is completed with another function key, the dialog is terminated immediately without checking the given entry. The finally pressed function key is returned as function result.

Besides the provided password pw, also the Service Mode password is accepted.

 

Example:

void Configuration (void)

{

  if (codebox ("Please enter password for access\nof configuration menu:",
       MasterPassword) == ok)
  {

  /* ... */

  }

}

 

 

Note:

The instruction codebox does not save the original content of the application screen. I.e. the programmer cannot expect that text previously written into the display is still present after the execution of codebox and must design his program accordingly.
