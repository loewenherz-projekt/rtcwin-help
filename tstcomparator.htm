tstcomparator

    

RTC Function

Checks the state of a real-time comparator.

Declaration:

bool tstcomparator (int sno, int n)

Group:

External inputs/outputs

Parameters:

sno
n

Scale number
Number of comparator

Return value:

State of comparator

 

The function tstcomparator checks the state of a real-time comparator. The parameter n specifies the number that was given to the comparator when it was activated by the function setcomparator; sno is the number of the scale the comparator is assigned to. If the switching condition of the comparator is fulfilled, tstcomparator returns true as function result; otherwise the result is false.

 

Example:

#define ANALOG_OUTPUT 0

#define OUT0_FAST 0

 

/* Start filling loop */

void InitFastFilling (void) // ProcessState: PS_INIT_FAST_FILL

{

  ProcessRepeat = PS_INIT_FAST_FILL;

  setcomparator (1, 1, OUT0_FAST, Target - Preact, "<", 1);

  ProcessState = PS_FAST_FILL;

}

 

void FastFilling (void) // ProcessState: PS_FAST_FILL

{

  if (tstcomparator (1, 1))

  {

    aout (ANALOG_OUTPUT,  FastAnalogValue);

  }

  else  /* Filling complete */

  {

    clrcomparator (1, 1);

    aout (ANALOG_OUTPUT, 0);

    ProcessState = PS_INIT_TOLERANCE_CHECK;

  }

}
