islocked

    

RTC Function

Indicates whether a record of a database table is locked.

Declaration:

bool islocked (dbase db)

Group:

Database functions

Parameters:

db

Database table

Return value:

Function result

 

The instruction islocked searches the database table specified by the parameter db for a record matching the current content of the index field and checks whether it is locked. The record is stored against the index field, i.e. the field that has been declared first. If the record is locked, islocked returns true as function result; otherwise the result is false.

A record can be locked against modification/deletion using the instruction lock. A locked record can be released using the instruction unlock.

 

Note:

The function islocked is a method of the declaration word dbase.
