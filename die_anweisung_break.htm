The break Statement

    

The statement break can be used to (prematurely) exit the innermost while, do-while or for loop or the switch statement.

 

Example:

for (;;)
{

  if (keypressed()) ch = readkey();

  if (ch == fct6) break;

  /*... */

}

/* ... */

 

 

·The statement break immediately terminates execution of a loop. Program execution continues with the next instruction following the loop.
·In nested loops, program execution returns to the next higher loop.
·The instruction break does not serve to exit a function. Use the statement return instead.
