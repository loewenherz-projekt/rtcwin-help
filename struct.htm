Name of the struct
Data type of the struct item
Item name

The instruction struct may be used for the declaration of a struct. The declaration word struct is followed by the name of the struct and – embraced in curved brackets – the individual struct items. Each item consists of a declaration word for the data type and the item name. The data types bool, char, int, float, decimal and string are allowed. Each struct item can be addressed with its unique identifier consisting of the struct name, a full stop and the item name, e.g. struct.item.

Data structures defined by struct allow to make functions available as parameters.

Example:

struct st_message

{

  string text[64];

  int color;

}

 

 

st_message Message;

 

void ShowOperatorMessage (st_message msg)

{

  fontsize (18);

  font (24); 

  background (msg.color);

  posto (0, 105); boxto (320, 130, 1);  

  showtext (5, 108, mid (msg.text, 32));

}

 

 

//...

Message.text = "Weigh hopper not empty!";

Message.color = CL_RED;

ShowOperatorMessage (Message);
