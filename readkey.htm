readkey

    

RTC Function

Reads a character from the keyboard.

Declaration:

char readkey (void)

Group:

Display and keyboard

Parameters:

-

Return value:

Key code

 

The function readkey reads a character from the terminal keyboard or the connected PC keyboard. If the keyboard buffer contains any characters not yet read, readkey returns the corresponding key code without delay. Otherwise program execution waits until the operator presses a key. The function keypressed can be used to check the availability of a key prior to the execution of readkey.

 

Example:

/* If key pressed: read key and process... */

if (keypressed()) ch = readkey(); else ch = 0;

if (ch != 0)

{

  /* ... */
