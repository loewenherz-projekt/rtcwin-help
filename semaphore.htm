semaphore

    

RTC Declaration Word

Declaration of a semaphore variable.

Declaration:

semaphore name

Group:

Declaration words

Parameters:

name     Name of semaphore variable

 

The instruction semaphore declares a semaphore variable with given name. Semaphore variables are used for synchronization of the access of resources and memory areas that are commonly used from various tasks. Semaphore variables may be locked and unlocked using the instructions lock, trylock and unlock.

 

Example:

semaphore Sema1;

 

/* ... */

while (!trylock(Sema1))

{

  if (keypressed())

    if (readkey()) return;

  delay (0.1);

} 

/* Access to shared resource */

unlock (Sema1); // Release semaphore variable
