flash

    

RTC Declaration Word

Declaration of a non-volatile variable.

Declaration:

flash  decl  name

Group:

Declaration words

Parameters:

decl       Declaration word name
name     Variable name

 

The instruction flash declares a variable with given name name as non-volatile. flash precedes the literal declaration word decl. The variables declared using flash are stored in the battery-backed memory as are nov variables. flash variables, however, can also be stored in the flash memory using the instruction flashsave. Thus, they are saved permanently and independently from the backup battery. The instruction flashload copies the flash memory variables to the battery-backed RAM.

 

Example:

flash  int HostChannel;
