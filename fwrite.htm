fwrite

    

RTC Function

Writes a string into a file.

Declaration:

int fwrite (int fno, string s, int len)

Group:

File handling

Parameters:

fno
s
len

Logical file number
String to be sent
Number of characters to be send

Return value:

Number of written characters

 

The function fwrite writes the provided string s into the file specified by the logical file number fno. The file has to be opened first using the instruction fopen. If the string can be written successfully, fwrite returns the number of written characters as function result; otherwise the result is 0. After execution of fwrite, the internal read/write pointer is moved behind the last inserted character.

The parameter len specifies the number of characters to be written. If the string is longer than len, only the corresponding part of the string is transmitted. If the string is shorter than len, it is filled up with a corresponding number of zero characters #0.

 

Example:

bool WriteWeighingDataToLogFile (string directory, string data)

{

int result = 0;

  if (fopen (1, concat (directory, "WeighLog.csv")) == 0)    // directory = "shared/"

  {

    result = fwrite (1, data, length (data));                // data = "date;gross;net"

    if (fclose (1) > 0)

    {

      ShowErrorMsg ("Error log file!");

      return (false);

    }    

 

    if (result == 0)                                         // Evaluate variable result

    {

      ShowErrorMsg ("Error log file!");

      return (false);

    }

    else if (result != length (data))

    {

      ShowErrorMsg ("Storage full1!");

      return (false);

    } 

    else

      return (true);

  }

  else  

  {

    ShowErrorMsg ("Error log file!");

    return (false);

  }

}

 

Note:

There is no end character defined for the function fwrite, i.e. zero characters and other control characters can also be read from the file.
