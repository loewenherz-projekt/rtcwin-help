The Programming Language RTC     Preprocessor

Conditional Programming

    

Constants defined via the #define statement may be evaluated by the preprocessor in an if-clause to generate different programs out of the same source file. The following structures are available:

#ifdef  name

  RTC statements

#endif

The preprocessor includes the RTC statements between the #ifdef and the #endif in the generated program only if "name" does exist.

#ifndef  name

  RTC statements

#endif

The preprocessor includes the RTC statements between the #ifndef and the #endif in the generated program only if "name" does not exist.

Both instructions #ifdef and #ifndef may be extended with an optional #else portion:

#ifdef  name

  RTC statements-1

#else

  RTC statements-2

#endif

If "name" exists, the preprocessor includes the RTC statements-1 in the program; otherwise the RTC statements-2 are included.

 

Note:

Predefined preprocessor constants are available to customize application programs for different hardware platforms or interpreter versions.
