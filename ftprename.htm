ftprename

    

RTC Function

Renames a file on the FTP server.

Declaration:

bool ftprename (int fno, string path, string oldname, string newname)

Group:

File handling

Parameters:

fno          path
oldname
newname

Logical connection number                                     Directory path
Original file name
New file name

Return value:

Function result

 

The function ftprename searches the file specified by the parameter oldname on the FTP server and assigns it the name specified by the parameter newname. For this purpose, a connection has to be established beforehand via ftpopen and – if applicable – a log-in performed using ftpauthenticate. The parameter fno is the logical connection number assigned with the execution of the function ftpopen.

If the file can be renamed successfully, sftprename returns true as function result; otherwise the result is false.

 

Example:

/* ... */

if (ftpconnected (0))

{

  if (ftplist (0, "/"))  // open directory ...

  {

    while (ftplistread (0, filename)) // read files...

    {

    // process files contained in filename......

          

    // Rename processed file

      if (!ftprename (0, "/", filename, concat (getstddate(), "_", filename)))

        ShowErrorMsg ("FTP access denied!");

    }

  }

}

else

  ShowErrorMsg ("Connection to FTP server lost!");
