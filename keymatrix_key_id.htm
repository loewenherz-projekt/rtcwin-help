keymatrix_key_id

    

RTC Function

Determines the object ID of a button within a matrix.

Declaration:

int keymatrix_key_id (int key_no)

Group:

Touch screen programming

Parameters:

key_no

Number of the button within the matrix

Return value:

Object ID of the button

 

The instruction keymatrix_key_id returns the object ID of a given button in a key matrix. The page with a matrix of buttons has to be generated beforehand using the instruction keymatrix_generate. The button is defined using the parameter key_no. Counting of key_no starts from 0 for the leftmost button in the top row and continues row by row from left to right.

 

Example:

/* Start button 'green'... */

widget_foreground (keymatrix_key_id (3), CL_GREEN);
