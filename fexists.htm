fexists

    

RTC Function

Checks the existence of a file.

Declaration:

int fexists (string fname)

Group:

File handling

Parameters:

fname

File name

Return value:

Function result

 

The function fexists checks the existence of the file specified by the parameter fname. The parameter fname contains the full file name as follows:

internal/name.ext

The file name with the extension ext is contained in the battery-backed memory of the terminal and cannot be accessed externally.

shared/name.ext

The file name with the extension ext is contained in the directory "shared". This folder can be accessed externally via FTP access (login: "shared," password as set in Service Mode).

usbstick/.../name.ext

The file name with the extension ext is contained on a USB memory stick. Further subdirectories may be specified as applicable.

Files on the USB memory stick can be accessed externally via FTP (login: "usbshared," password as set in Service Mode).

sharename/.../name.ext

The file name with the extension ext is contained on a network drive. sharename is the symbolic path name as it was returned by the instruction smbconnect. Further subdirectories may be specified as applicable.

If the file name does not specify a directory, the folder 'internal' is used as default. As function result, fexists returns the status of the file as follows:

0File fname does not exist
1fname exists and is a file
2fname exists and is a directory
3fname exists and is a link (e.g. "usbstick")
-1fname exists but has an unknown type

 

Example:

// New orders in file?

bool CopyDataToLogFile (string source_name, string dest_name)

{

string source_file[1023] = concat ("shared/", source_name, ".csv");

string destination_file[1023] = concat ("usbstick/", dest_name, ".csv");

 

  infobox ("Kopiere Log-Datei...");

  if (fexists (destination_file) == 1)

    if (fremove (destination_file) != 0)

    {

      ShowErrorMsg ("Zugriff verweigert");

      return (false);

    }

 

  if (! fcopy (source_file, destination_file))

  {

    ShowErrorMsg ("Datei konnte nicht kopiert werden!");

    return (false);

  }

  else

    return (true);

}
