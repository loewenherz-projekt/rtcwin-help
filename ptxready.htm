ptxready

    

RTC Function

Checks the CTS input of a serial communication port.

Declaration:

bool ptxready (int port)

Group:

Communication

Parameters:

port

Logical port number

Return value:

Function result

 

The function ptxready checks the CTS input of the serial communication port specified by the logical port number port whether the connected device is ready to receive data. For this, RTS/CTS handshaking must be enabled in Service Mode of the weighing terminal; otherwise ptxready always returns true as function result.

 

Example:

/* Print weighing ticket */

bool OpenPrtPort (int printer_port, string printer_port_parameter)

{

  if (printer_port > 7) // For USB and network printer

  {

    if (popen (printer_port, printer_port_parameter) != 0)

    {

      ShowErrorMsg ("Printer off-line!");

      pclose (printer_port);

      return (false);

    }

  }

  else if (!ptxready (printer_port))

  {

    ShowErrorMsg ("Printer off-line!");

    return (false);

  }

  

  print (printer_port, Ticket);  // Print ticket/label

  

  if (printer_port > 7)          // Close channel if USB and network printer

    pclose (printer_port);

  

  return (true);

}
