The Programming Language RTC     XML Interface

Attributes

    

Each node or text in an XML document may be assigned one or more attributes as shown in the following example:

<staff>
  <employee id="1" Name="John Brown">
    <Department Name="shipping" />
  </employee>
  <employee id="2" Name="Jack Williams">
    <Department Name="purchasing" />
  </employee>
</staff>

 

The node "employee" in the example above has two attributes: id and Name. Attributes can be assigned to a node using xmlsetattribute:

xmlinsertchild (HANDLE, "staff", "employee", 1, 1);
xmlsetattribute (HANDLE, "staff/employee", "id", "1");
xmlsetattribute (HANDLE, "staff/employee", "Name", "John Brown");

 

In this example, first a (new) node employee is created and then the attribute id is set to 1. If, as in the example above, multiple nodes exist with identical name, the attribute of the first node in the list matching the name is set. Alternatively, the target node can be selected using the instructions xmlfirstchild, xmllastchild or xmlnextchild, and xmlprevchild.

 

But there still is an easier way:

xmlsetattribute (HANDLE, "staff/employee<id=1>", "Name", "John Brown");

 

This above statement explicitly searches for the node employee with id equal 1. Again, the programmer can utilize the fact that the node is created in case that such a node not yet exists.
