string

    

RTC Function

Converts an expression into data type string.

Declaration:

int string (any v)

Group:

Type conversion

Parameters:

v

Variable or function with result of any type

 

The function string converts the variable of any type specified by the parameter v into the data type string. Numeric variable of type int, float, double or decimal are converted into a string containing the value as text, if applicable with minus sign and decimal point or comma. Variables of type bool are converted into a string of length 1, containing e.g. a 'Y' for true or an 'N' for false, depending on the chosen language.

Instead of a variable v, also a function or an algebraic expression may be provided as parameter. The numeric or Boolean result is then converted into a string as described above.

 

Example:

msg = concat ("Deviation = ", string (deviation), " %");

 

Note:

For a formatted conversion of a number into a string, the function adj can be used.
