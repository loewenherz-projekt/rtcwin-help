The switch Statement

    

The switch statement consists of one expression (selector) and a list (of any length) of branches. The structure is as follows:

 

switch (Expression)

{

  case Constant_1 : Statement_1;

  case Constant_2 : Statement_2;

    

  default : Statement_d;

}

 

If the case constant Constant_i matches the value of Expression, the subsequent program part Statement_i (including all following instructions) is carried out. If none of the constants matches the result of Expression, Statement_d following the word default is executed instead. If the default portion is missing, the complete switch structure remains ineffective in case that Expression does not match a constant.

·There is no semicolon behind Expression,
·The constants must be of type int or char.
·The statements may consist of multiple instructions. They do not have to be embraced by a pair of curved brackets.
·There may be any number of case branches.
·The default part is optional and may be omitted.
·The execution of the case branches is from top downwards. As soon as a match is found, the complete rest of the switch construct is executed – that also includes the subsequent case branches. (Use the break statement where applicable!)

 

Example:

ch = readkey();

switch (ch)

{

  case fct1:

  case ok:    Registration();

              break;

  case fct3:  ShowSubTotals();

              break;       

  case fct6:  return();       

}
