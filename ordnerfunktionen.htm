The Programming Language RTC     Communication and File Access

Directory Handling

    

RTC also provides functions to manage the accessible directories. Application-accessible directories are:

"internal" or ""

Internal battery-backed memory, externally not accessible. If no directory name is specified, 'internal' is default.

"shared"

Folder "shared" in the internal battery-backed memory that is accessible from the external via FTP.

"usbstick"

Memory on the external USB memory stick. Further subdirectories may be specified as applicable.

sharename

External network drive. sharename is the symbolic path name as it was returned by the function smbconnect.

For directory management the following instructions are available:

bool dopen (int refno, string path)

Opens a directory for subsequent access.

void dclose (int refno)

Closes a directory.

bool dsort (int refno, int crit, int dir)

Sorts the files in the directory for subsequent read cycles.

bool dread (int refno, string *name)

Reads the next file name from the directory.

void dseekfirst (int refno)

Reads the first file name from the directory.
