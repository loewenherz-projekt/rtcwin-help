md5finish

    

RTC Function

Calculates the MD5 checksum over a pool of data.

Declaration:

string md5finish (int handle)

Group:

String functions

Parameters:

handle
 

Reference number

Return value:

Result string

 

The functioning md5finish calculates the MD5 checksum (hash value) over a pool of data. Checksum calculation has to be initiated beforehand using the function md5init. The data has to be assembled via one or multiple calls of md5append. md5finish returns as function result a 32-character string containing the hexadecimal representation of the MD5 checksum.

 

Example:

string ChkSum[32];

 

  md5init (0);

  md5append (0, Data, length (Data));

  ChkSum = md5finish (0);
