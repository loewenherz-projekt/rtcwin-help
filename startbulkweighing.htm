startbulkweighing

    

RTC Function

Starts approved totalizing of weight values.

Declaration:

int startbulkweighing (int sno)

Group:

Scale functions

Parameters:

sno

Scale number

Return value:

Error code

 

The function startbulkweighing initializes the variable totalbulkweight to zero and starts totalizing for the scale specified by the optional parameter sno. If the parameter sno is missing, totalizing is started for the first scale instead. With cyclically calling the functions getfilledbulkweight and getdischargedbulkweight, the weight of the filled and discharged weigh hopper is then captured and the difference added to the content of totalbulkweight. A concluding call of storetotalbulkweight terminates totalizing of weight values and stores the totals in the approved weight storage (alibi memory).

If the weighing can be initialized successfully, startbulkweighing returns 0 as function result; otherwise the corresponding error code is returned instead.
