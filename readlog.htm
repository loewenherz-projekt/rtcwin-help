readlog

    

RTC Function

Reads the ring buffer of a real-time weight log.

Declaration:

int readlog (int sno, int *tarray, int *warray)

Group:

Scale functions

Parameters:

sno
tarray
warray

Scale number
Destination array for time values
Destination array for weight values

Return value:

Number of read records

 

The function readlog reads the content of the weight-log ring buffer and writes the data into the provided array variables. Logging of weights has to be started and ended beforehand using the instruction startlog and stoplog. After the execution of the function readlog, the weights are contained in the integer array warray, whereas the array tarray contains the corresponding system times. For better performance both, weight and time arrays are of type integer and herein contained values must be converted as applicable. The weight values comply with the high-resolution net weight nethires with the decimal point omitted, whereas the time values show the time in milliseconds since start of logging. readlog returns the number of read weight and time pairs as function result.

 

Example:

/* Determine flow rate */

i = readlog (1, tlog[0], wlog[0]);

Flowrate = (wlog[i] - wlog[0]) / (tlog[i] - tlog[0]) * Factor;
