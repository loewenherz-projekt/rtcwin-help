settime

    

RTC Function

Set the internal real-time clock.

Declaration:

void settime (int y, int m, int d, int h, int n, int s)

Group:

System

Parameters:

y
m
d
h
n
s

Year
Month
Day
Hour
Minute
Second

Return value:

-

 

The function settime sets the internal real-time clock according to the provided parameters. The year may be specified in short or in long writing (e.g. 8 or 2008). For the hour, the 24-hour system is presumed.

 

Example:

bool SetDateTime (int yy, int mm, int dd, int hh, int nn, int ss)

{

  if ((dd >= 1 && dd <= 31) && 

      (mm >= 1 && mm <= 12) && 

      (yy >= 0 && yy <= 99) &&

      (hh >= 0 && hh <= 23) && 

      (nn >= 0 && nn <= 59) && 

      (ss >= 0 && ss <= 59))

  {

    settime (yy, mm, dd, hh, nn, ss);

    infobox ("Set date and time");

    return (true);

  }

  else

  {

    infobox ("Invalid parameter !");

    return (false);  

  }

}
