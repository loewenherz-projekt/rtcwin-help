setzero

    

RTC Function

Sets the scale to zero.

Declaration:

int setzero ([int sno], [float timeout])

Group:

Scale functions

Parameters:

sno

Optional scale number

Return value:

Error code

 

The function setzero sets the scale specified by the optional parameter sno to gross = 0. If the parameter sno is missing, the first scale is set to zero instead. If the scale can be set to zero successfully, setzero returns 0 as function result. If the scale is not in zero setting range or in case of an error, the corresponding error code is returned instead.

 

Example:

err = getweight (indicatedscale());

if (gross < (5 * division))
  err = setzero (indicatedscale()) ; // If gross < 5 increments: set scale to zero

else

  err = autotare (indicatedscale()); // ... else taring

 

Note:

For zero setting, the scale has to be settled. If the scale does not provide a stable weight within 6 seconds, a timeout error is returned.
