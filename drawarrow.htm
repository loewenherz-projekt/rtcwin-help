drawarrow

    

RTC Function

Draws a direction triangle.

Declaration:

void drawarrow (int x, int y, int dir)

Group:

Display and keyboard

Parameters:

x
y
dir

Horizontal position of the triangle
Vertical position of the triangle
Orientation

Return value:

-

 

The instruction drawarrow draws a direction triangle on the default display at the position specified by the two parameters x and y. The two parameters x and y specify the position of the upper left corner of a square encircling the triangle. The size of the triangle is taken from the currently selected font size. The triangle is filled with the current drawing color.

The parameter dir specifies the orientation of the triangle as follows:

dir =0

dir =1

dir =2

dir =3

Pointing right

Pointing down

Pointing left

Pointing up

 

Example:

drawarrow (20, 10, 1); /* Draw triangle pointing down */
