writeparameter

    

RTC Function

Sets a system or Service Mode parameter.

Declaration:

int writeparameter (string group, string par, string val)

Group:

System

Parameters:

group
par
val

Parameter group
Parameter name
Value

Return value:

Error code

 

The function writeparameter sets the parameter specified by par in the group group to the value specified by the parameter val. This may concern system settings as well as Service Mode parameters. If the parameter can be set successfully, writeparameter returns 0 as function result; otherwise an error code is returned instead.
