triangle

    

RTC Function

Draws a triangle.

Declaration:

void triangle (int x1, int y1, int x2, int y2)

Group:

Display and keyboard

Parameters:

x1
y1
x2
y2

Horizontal coordinate x of first corner point
Vertical coordinate y of first corner point
Horizontal coordinate x of second corner point
Vertical coordinate y of second corner point

Return value:

-

 

The instruction triangle draws a triangle between the current cursor position and the two corner points specified by the coordinates x1/y1 and x2/y2. The triangle is filled with the current drawing color.

 

Example:

posto (40, 10);

triangle (32, 20, 48, 20); /* Draw an upright triangle */
