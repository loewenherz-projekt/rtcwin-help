setunit

    

RTC Function

Assigns a unit text to a variable or database field.

Declaration:

void setunit (any var, string unit)

Group:

Display and keyboard

Parameters:

var
unit

Variable or database field
Unit sign

Return value:

-

 

The instruction setunit assigns the given variable or database field (dbname.fieldname) the unit sign specified by the parameter unit. During the execution of the functions inputbox or edit, this unit text is appended to the respective variable.

 

Example:

dbase "Article File" Article [250]

{

  "Article No."   decimal  No%3.0; selectmode = input;

  "Name"          string   Name[24];

  "Density"       decimal  Density%6.4;

  "Price/kg"      decimal  Price%5.2;

}

 

setunit (Article.Density, " g/ml");

setunit (Article.Price, " €");
