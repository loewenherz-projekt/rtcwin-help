crc16

    

RTC Function

Determines the CRC16 checksum of a string.

Declaration:

int crc16 (string s, int len)

Group:

String functions

Parameters:

s
len

String
Length of the string

Return value:

Checksum

 

The function crc16 returns as function result the CRC16 checksum (cyclic redundancy check) of the string specified by s.The checksum is calculated over the provided length len and returned as function result. If s is shorter than len, the result is unspecified.

 

Example:

SendBuffer = concat (Data, itoh (crc16 (Data, length (Data))));

writestring (Host, SendBuffer);
