#Binary

    

RTC Function

Embeds an encrypted object file into the program.

Declaration:

#binary  fname

Group:

Preprocessor

Parameters:

fname

Name of encrypted object file

 

With the instruction #binary, an external encrypted object file can be included in the program design. The external file is specified by the given name fname (in quotation marks or angle brackets) and has to be contained in the current project.

 

Example:

#binary  "ISM_Driver.obj"   /* Driver for pulse counter module */

 

Note:

For non-encrypted files such as c- or h-files the statement#include has to be used.
