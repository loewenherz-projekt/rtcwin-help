pos

    

RTC Function

Returns the position of a character in a string.

Declaration:

int pos (string s, char c)

Group:

String functions

Parameters:

s
c

String
Character to search

Return value:

Position of character

 

The function pos returns the position of the character specified by c in the string s.Counting starts from 0 (first position). If c is not contained in the string, pos returns -1 as function result. If c is contained in the string more than once, the position of the first appearance of c is returned.

 

Example:

i = pos (ReceiveBuffer, ';');
