md5init

    

RTC Function

Starts the calculation of an MD5 checksum.

Declaration:

void md5init (int handle)

Group:

String functions

Parameters:

handle    

Reference number

Return value:

-

 

The function md5init starts the calculation of a 128-bit checksum (hash value) using the MD5 (message-digest) algorithm. The data over which the check sum is built is assembled by repetitive calls of the function md5append. The parameter handle serves as a reference. When all data is put together, the 32-character hexadecimal checksum is calculated by a conclusive call of the function md5finish.

 

Example:

string ChkSum[32];

 

  md5init (0);

  md5append (0, Data, length (Data));

  ChkSum = md5finish (0);
