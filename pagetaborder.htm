pagetaborder

    

RTC Function

Determines the order of objects on a page.

Declaration:

void pagetaborder (int id, string list)

Group:

Touch screen programming

Parameters:

id
list

Logical reference number of the page
List of objects

Return value:

-

 

The instruction pagetaborder determines the order of objects on a page to facilitate access with the tab key of a connected PC keyboard. The page is defined using the parameter id. The parameter list contains the identification numbers of the individual objects in the respective order. The object IDs in list are separated by semicolons.

 

Example:

pagetaborder (PAGE_MY_INPUT_BOX, concat (ID_INPUT_DATE, ";", 

                                         ID_INPUT_TIME, ";", 

                                         ID_INPUT_TICKET_NO, ";", 

                                         ID_INPUT_PASSOWRD));
