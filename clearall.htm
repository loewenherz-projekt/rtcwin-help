clearall

    

RTC Function

Clears the data memory.

Declaration:

void clearall (void)

Group:

Miscellaneous functions

Parameters:

-

 

Return value:

-

 

The function clearall clears the complete battery-backed memory. The internal directories intern and shared are emptied. All database tables, nov variables and print formats will be lost.

In contrast to a successive call of the function clear for all existing database tables, clearall also rebuilds the table structures. Thus, a subsequent call of clear is not necessary.

 

Example:

void SetFactoryDefaults (void)

{

  clearall();

  

  UserPassword = "";

  WithPrinter = false; 

  WithTransmission = false; 

  /* ... */

}
