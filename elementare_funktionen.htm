The Programming Language RTC     Output to the Display

Basic Functions

    

Following table shows the basic display functions contained in RTC:

void background (int color)

Specification of background color.

void foreground (int color)

Specification of writing/drawing color.

void posto (int x, int y)

Specification of start position for the next drawing function.

void lineto (int x, int y)

Draw a line from current to specified position.

void boxto (int x, int y, int type)

Draw a box of given type from current to specified position.

void triangle (int x1, int y1, int x2, int y2)

Draw a triangle between current position and the two specified points.

void rectangle (int w, int h)

Draw a square of given width and height starting at current position.

void polygon (int mode)

Draw a filled polygon in combination with subsequent lineto statements.

void arc (int w, int h, int w0, int w1)

Draw an elliptical arc of given width and height. The two other parameters specify start and end angle of the arc.

void pie (int w, int h, int w0, int w1)

Draw a filled elliptical pie of given width and height. The two other parameters specify start and end angle of the pie.

void clearscreen (void)

Clear screen with current background color.

void clearto (int x, int y)

Clear a square screen area from current to specified position and fill with current background color.

void font (int f)

Specify font and attribute (if applicable) for subsequent write operations.

void fontsize (int fs)

Specify font size for subsequent write operations.

void showtext (int x, int y, string s)

Output a text at the specified position. Current font style, size and attributes are used.

char input (int x, int v, any v)

Entry of a variable at the specified position.

char tinput (int x, int v, any v, float t)

Entry of a variable at the specified position with timeout.

void drawarrow (int x, int y, int dir)

Draw a direction triangle at the specified position.

void begindraw (void)

Starts buffering of subsequent display functions.

void enddraw (void)

Stops buffering of display functions and outputs all buffered instructions.
