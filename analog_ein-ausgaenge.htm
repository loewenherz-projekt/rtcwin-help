Analog Inputs/Outputs

    

The following two functions are available for the handling of the analog inputs and outputs:

void aout (int no, float v)

Output of value to the specified analog output port. The output value has to be specified as percentage (0...100 %) of the maximum value.

float ain (int no)

Read the specified analog input port. The return value is understood as percentage (0...100 %) of the maximum value.

The configuration of the analog input or output is contained in the Service Mode of the weighing terminal. The analog value is always understood as percentage of the maximum value and thus is independent of the configuration (0–20 mA, 4–20 mA, 0–10 V, 2–10 V) and the resolution (12 bit/15 bit) of the installed module.
