The Programming Language RTC     Communication and File Access

Serial Communication

    

For the communication via serial interface or TCP/IP port, the following functions are available:

int popen (int port, string s)

Open a communication channel (ports 0–7 are generally open)

void pclose (int port)

Close a communication channel

bool pconnected (int port)

Check the connection status of a communication channel

int preceived (int port)

Check communication channel for available characters not yet read

bool ptxready (int port)

Check if communication channel is ready for transmission

char preadchar (int port)

Read a character from a communication channel

int pwritechar (int port, char ch)

Output a character to a communication channel

int preadstring (int port, string s, int len,
char eoc, float timeout)

Read string from a communication channel under consideration of an end character

int pread (int port, string s, int len,
float timeout)

Read string from a communication channel without consideration of an end character

int pwritestring (int port, string s, int len)

Output a string to a communication channel

int readstring (int port, string s)

Read a string from a communication channel using the communication protocol set up in Service Mode

int writestring (int port, string s)

Output a string to a communication channel using the communication protocol set up in Service Mode
