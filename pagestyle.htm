pagestyle

    

RTC Function

Sets the criteria for the appearance of a page.

Declaration:

void pagestyle (int id, string style)

Group:

Touch screen programming

Parameters:

id
style

Logical reference number of the page
Format string

Return value:

-

 

The instruction pagestyle sets the criteria for the look of the page specified by the parameter id. The string parameter style contains the formatting instructions in style sheet syntax.

 

Example:

void CreatePage (void) 

{

  begindraw();

  createpage (PAGE_1);

  pagestyle (SELECT_BOX_ID, "background: #d0e0ff;");  // SysTec Default

    beginlayout (LAYOUT_TYPE_VBOX, 0, 2);

    /* ... */

    endlayout();

  enddraw();

}

 

Note:

The format attributes set by the instruction pagestyle are processed each time the page is called. This may have a negative impact on the system performance.
