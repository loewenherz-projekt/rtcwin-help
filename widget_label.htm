widget_label

    

RTC Function

Sets the operator prompt of an input field.

Declaration:

void widget_label (int id, string prompt)

Group:

Touch screen programming

Parameters:

id
prompt

Logical reference number of the object
Operator prompt

Return value:

-

 

The instruction widget_label sets the operator prompt of an input field. The operator prompt is shown left of the input field of the on-screen keypad.

 

Example:

// Create input field...
void CreateInputWidget (int id, string label_text)

{
  createwidget (id, WIDGET_TYPE_TEXT_INPUT);

    widget_fontsize   (id, 24);  

    widget_fontface   (id, FT_RTC_BOLD)

    widget_dimensions (id, 50, 50); 

    widget_label      (id, label_text); // Operator prompt

    widget_format     (id, INPUT_FORMAT_DECIMAL, 6, 2);

}
