double

    

RTC Declaration Word

Declaration of a double-precision floating-point variable.

Declaration:

double name [%len.dec]

Group:

Declaration words

Parameters:

name
len
dec    

Variable or function name
Optional specification of length
Optional number of decimal digits

 

The instruction double declares a variable with given name or the return value of a function with given name as floating-point number with double precision. The type double covers the real numbers in the range of

± 1.79 * 10–308 to ± 1.79 * 10308

with a precision of 15–16 digits. The data type float can be used for single precision. During declaration, the variable name may be assigned an initial value. The types double, decimal and float may be assigned each other.

During declaration, the variable can optionally be formatted for input and output in respect of length and number of decimals. The length len is understood as overall length including decimals and the decimal point itself, as applicable, whereas dec only defines the number of trailing decimals.

Example:

double CorrFactor%8.6;

double pi = 3.14159265359;

 

Note:

When the optional length is specified, the comma also has to be taken into account. The format %8.6 thus defines a variable with one digit before and six digits after the decimal point.
