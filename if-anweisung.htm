The Programming Language RTC   Conditions

The if Statement

    

The if statement is used to put program execution under certain conditions. The structure is as follows:

 
if ( Expression )
  Statement_1;
else
  Statement_2;
 

If the result of the Boolean Expression is true, the following Statement_1 is executed, otherwise the Statement_2 following the word else is carried out. If the else portion is missing, the if statement has no effect in case that expression results in false.

·There is no semicolon behind Expression,
·The else part is optional and may be omitted,
·Statement_1 and Statement_2 may consist of multiple statements combined by a pair of curved brackets (compound statement).

Statement_1 and Statement_2 again may be if statements, i.e. if statements may be nested. By definition it is agreed upon that the else always refers to the last if that has not yet been assigned an else part. Thus, the construction

 
if (Expression_1) if (Expression_2) Statement_1; else Statement_2;
 

is interpreted as:

 

if (Expression_1)
{

  if (Expression_2)

    Statement_1;

  else

    Statement_2;

}
