writechar

    

RTC Function

Outputs a character on a communication channel.

Declaration:

int writechar (int port, char c)

Group:

Communication

Parameters:

port
c

Logical port number
Character to be send

Return value:

Error code

 

The function writechar outputs the character specified by the parameter c on the communication channel specified by the logical port number port. The port has to be opened first (e.g. using popen).

 

Example:

if (writechar (port, STX) == 0)

{

/* Transmit complete string */

}

else

{

/* Evaluate error */

}
