Expressions

    

Expressions consist of operands and operators. Most of the operators link two operands and are thus called "binary." The top three operators of the table below, however, are "unary" and only work upon one operand.

Operators:

! ~ -

Not, Complement, Minus.

* / %

Multiplication, Division, Modulo.

+ -

Addition, Subtraction.

<< >>

Bitwise shift to the left and to the right.

<= < > >= == !=

Logical comparison on less-equal, less, greater, greater-equal, equal and unequal.

&

Bitwise And

^

Bitwise Exclusive-Or

|

Bitwise Or

&&

Logical And

||

Logical Or

 

Priority of the operators listed above is from top downwards. Operators within a line have identical priority. Where multiple operands are to be linked up, the operation with highest priority is executed first. Operations with identical priority are executed from left to right.

The following table shows the availability of operators in respect of the data type of the operands.

! && ||

Data type bool only.

~ % << >> & ^ |

Data type int only.

+

Data types char, int, float, decimal and string.

- * /

Data types char, int, float and decimal.

<= < > >= == !=

Data types char, int, float, decimal and string.

 

·The result of the addition of types char and string is a concatenated string.
·The comparison of types char and string is based on the ASCII code of the individual characters. Comparison of strings is carried out from left to right.

Incrementing or decrementing of a variable may also be included in an expression. In the expression

 

j = i++;

 

the first variable j is assigned the value of i and i incremented afterward; whereas in the case

 

j = ++i;

 

first the variable i is incremented and then the new value assigned to the variable j.
